거리(km)와 시간(시간)을 입력받아 평균 속도(km/h)를 소수점 첫째 자리로 출력하세요.
(입력값: 150, 2.5)

Q41.

원금과 연이율(%)을 입력받아 1년 후 원금 + 이자를 출력하세요.
(입력값: 1000000, 3.5)

출력:1,035,000.0원
💡 힌트
원금 * (1 + 이율/100)

Q48.

시험 점수 5개를 입력받아 리스트에 담고, 정렬된 점수와 평균을 출력하세요.
(입력값: 78, 92, 65, 88, 74)

출력:정렬: [65, 74, 78, 88, 92]
평균: 79.4
💡 힌트
점수를 scores 리스트에 append()로 추가
sorted(scores) 로 정렬된 새 리스트 생성
✅ 정답
scores = []
for i in range(1, 6):
    s = int(input(f"점수 {i}: "))
    scores.append(s)
print(f"정렬: {sorted(scores)}")
print(f"평균: {sum(scores)/len(scores):.1f}")

🏆 도전 문제 — 이름, 국어, 영어, 수학 점수를 입력받아 아래 형식의 성적표를 출력하세요.
(입력값: 이수빈, 85, 92, 78)

출력:===== 이수빈 성적표 =====
국어: 85점
영어: 92점
수학: 78점
평균: 85.0점
최고: 92점 | 최저: 78점
💡 힌트
scores = [kor, eng, math]
sum(scores)/len(scores) 로 평균
max(scores), min(scores) 로 최고·최저
✅ 정답
name = input("이름: ")
kor  = int(input("국어: "))
eng  = int(input("영어: "))
math = int(input("수학: "))
scores = [kor, eng, math]
avg = sum(scores) / len(scores)
print(f"===== {name} 성적표 =====")
print(f"국어: {kor}점")
print(f"영어: {eng}점")
print(f"수학: {math}점")
print(f"평균: {avg:.1f}점")
print(f"최고: {max(scores)}점 | 최저: {min(scores)}점")

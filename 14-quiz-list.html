Q50.

🏆 도전 문제 — nums = [3, 7, 1, 8, 4, 6, 2, 9, 5]에서 중앙값(정렬 후 가운데 값)을 출력하세요.

출력:5
💡 힌트
1. sorted()로 정렬한 새 리스트를 만든다.
2. 가운데 인덱스 = len(lst) // 2
3. 정렬된 리스트의 가운데 인덱스 요소를 출력한다.
✅ 정답
nums = [3, 7, 1, 8, 4, 6, 2, 9, 5]
s = sorted(nums)          # 정렬
mid = len(s) // 2         # 가운데 인덱스 = 4
print(s[mid])              # s[4] = 5

세 변이 모두 같으면 → "정삼각형"
두 변이 같으면 → "이등변삼각형"
모두 다르면 → "일반삼각형"
(삼각형 성립 조건 검사는 제외해도 됩니다)

💡 힌트

정삼각형: a == b == c / 이등변: a == b or b == c or a == c. 정삼각형을 먼저 검사하세요.

✅ 정답
a = int(input("변 a: "))
b = int(input("변 b: "))
c = int(input("변 c: "))
if a == b == c:
    print("정삼각형")
elif a == b or b == c or a == c:
    print("이등변삼각형")
else:
    print("일반삼각형")
Q29.

🔥 도전 문제 — BMI 계산기를 만드세요.
몸무게(kg)와 키(m)를 입력받아 BMI = 몸무게 / (키 ** 2)를 계산하고,
BMI < 18.5 → "저체중"
18.5 이상 23 미만 → "정상"
23 이상 25 미만 → "과체중"
25 이상 → "비만"
결과를 "BMI: [값] → [구분]" 형식으로 출력하세요. (소수점 1자리)

💡 힌트

BMI 먼저 계산 후 변수에 저장 → elif 연쇄로 구분 결정 → f-string으로 출력. 소수점 1자리는 f"{bmi:.1f}".

✅ 정답
weight = float(input("몸무게(kg): "))
height = float(input("키(m): "))
bmi    = weight / (height ** 2)
if bmi < 18.5:
    label = "저체중"
elif bmi < 23:
    label = "정상"
elif bmi < 25:
    label = "과체중"
else:
    label = "비만"
print(f"BMI: {bmi:.1f} → {label}")
Q30.

🔥 도전 문제 — 간단한 계산기를 만드세요.
두 수와 연산자(+, -, *, /)를 입력받아 계산 결과를 출력하세요.
나누기(/)는 b == 0이면 "0으로 나눌 수 없습니다"를 출력하세요.
잘못된 연산자면 "지원하지 않는 연산자입니다"를 출력하세요.

💡 힌트

연산자 분기를 먼저 처리하고, /일 때는 중첩 if로 b == 0을 따로 검사하세요.

✅ 정답
a  = float(input("첫 번째 수: "))
op = input("연산자(+ - * /): ")
b  = float(input("두 번째 수: "))
if op == "+":
    print(f"결과: {a + b}")
elif op == "-":
    print(f"결과: {a - b}")
elif op == "*":
    print(f"결과: {a * b}")
elif op == "/":
    if b == 0:
        print("0으로 나눌 수 없습니다")
    else:
        print(f"결과: {a / b}")
else:
    print("지원하지 않는 연산자입니다")

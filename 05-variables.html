변수와 데이터 타입

파이썬이 정보를 기억하는 방법 — 데이터에 이름을 붙이고 분류하기

▸ 이 챕터를 마치면
  • 변수가 무엇인지, 왜 필요한지 이해한다
  • int, float, str, bool 네 가지 기본 타입을 구분하고 사용할 수 있다
  • type()으로 데이터의 타입을 확인할 수 있다
  • int(), float(), str()로 타입을 변환할 수 있다
  • 변수 이름 짓는 규칙을 이해하고 좋은 이름을 고를 수 있다

1.변수란 무엇인가요?

여러분이 마트에서 물건을 살 때, 총 금액을 계산기로 두드렸다가 잊어버리면 곤란하죠. 그 금액을 어딘가에 메모해 두면 나중에 다시 쓸 수 있습니다.

파이썬에서 변수(Variable)는 바로 그 메모장입니다. 어떤 값을 이름을 붙여서 컴퓨터 메모리에 저장하고, 나중에 그 이름으로 꺼내 쓸 수 있습니다.

price = 3500
변수 이름
price
대입 연산자
= (같다는 뜻이 아님!)
저장할 값
3500
⚠ 중요 — = 는 "같다"가 아닙니다

수학에서 =는 "좌변과 우변이 같다"는 의미지만, 파이썬에서 =는 "우변의 값을 좌변 변수에 저장하라"는 명령입니다. 방향이 있습니다. 오른쪽 값 → 왼쪽 변수로 흘러들어간다고 생각하세요.

변수를 만들고 나면 그 이름으로 값을 꺼내 쓸 수 있습니다.

python
123456
# 변수에 값을 저장한다
price = 3500
name  = "아메리카노"
count = 2

# 변수를 꺼내 쓴다
print(name, price * count)
실행 결과
아메리카노 7000

변수 덕분에 3500이라는 숫자를 두 번 쓰지 않아도 됩니다. 나중에 가격이 바뀌면 price = 4000 한 줄만 고치면 됩니다. 이게 변수가 존재하는 이유입니다.

2.변수 이름 짓는 규칙

변수 이름은 아무렇게나 지을 수 없습니다. 파이썬이 정한 규칙이 있습니다.

규칙 올바른 예 잘못된 예
영문자, 숫자, 밑줄(_)만 사용 가능 user_name, score1 user-name, score!
숫자로 시작할 수 없음 score1 1score
파이썬 예약어는 사용 불가 my_if, for_count if, for, True
대소문자 구분 name과 Name은 다른 변수 —
💡 좋은 변수 이름 짓기 — snake_case

파이썬에서는 여러 단어를 연결할 때 밑줄(_)을 씁니다. 이를 snake_case라고 합니다. userName보다 user_name, totalScore보다 total_score가 파이썬다운 이름입니다. 또한 변수 이름은 항상 그 값이 무엇인지 알 수 있게 지어야 합니다. x = 3500보다 coffee_price = 3500이 훨씬 읽기 좋은 코드입니다.

3.데이터 타입이란?

세상의 정보는 종류가 다양합니다. 나이는 숫자고, 이름은 글자고, "켜짐/꺼짐"은 참/거짓입니다. 파이썬은 데이터를 종류별로 분류해서 저장합니다. 이 분류를 데이터 타입(Data Type)이라고 합니다.

브론즈에서 가장 먼저 배워야 할 기본 타입 네 가지입니다.

타입의미예시특징
int 정수 42, -7, 0 소수점 없는 수
float 실수 (소수) 3.14, -0.5, 2.0 소수점 있는 수
str 문자열 "안녕", 'Python' 따옴표로 감싼 텍스트
bool 불리언 (참/거짓) True, False 딱 두 가지 값만 존재
python
12345678
# int — 정수
age = 17

# float — 소수점이 있는 수
height = 172.5

# str — 문자열 (텍스트)
school = "슈퍼고등학교"

# bool — 참(True) 또는 거짓(False)
is_student = True
ℹ bool의 True / False

True와 False는 반드시 첫 글자를 대문자로 씁니다. true나 false로 쓰면 오류가 납니다. 파이썬이 정한 특별한 값이기 때문입니다.

4.type() — 타입 확인하기

변수나 값의 타입이 무엇인지 궁금할 때 type()을 씁니다. 파이썬이 그 값을 어떤 종류로 보고 있는지 알려줍니다.

python
12345678
print(type(42))          # 정수
print(type(3.14))        # 실수
print(type("hello"))     # 문자열
print(type(True))        # 불리언

# 변수에 담아서도 확인 가능
name = "파이썬"
print(type(name))
실행 결과
<class 'int'>
<class 'float'>
<class 'str'>
<class 'bool'>
<class 'str'>

<class 'int'>처럼 출력됩니다. 지금은 class라는 단어는 무시하고, 작은따옴표 안의 타입 이름만 읽으면 됩니다.

5.타입 변환 — 종류를 바꾸기

때로는 타입을 직접 바꿔야 할 상황이 생깁니다. 예를 들어 사용자가 입력한 "42"(문자열)를 숫자로 더하려면 먼저 정수로 바꿔야 합니다.

int() — 정수로 변환

python
12345
print(int("42"))       # 문자열 → 정수
print(int(3.9))        # 실수 → 정수 (소수점 버림)

# 이렇게 쓰면 숫자처럼 더할 수 있음
num = int("10")
print(num + 5)
실행 결과
42
3
15

float() — 실수로 변환

python
12
print(float("3.14"))   # 문자열 → 실수
print(float(5))         # 정수 → 실수 (5.0)
실행 결과
3.14
5.0

str() — 문자열로 변환

python
1234
score = 98
# 숫자와 문자열을 + 로 합치려면 str()로 변환 필요
print("점수: " + str(score))
print(str(3.14))         # 실수 → 문자열
실행 결과
점수: 98
3.14

6.자주 만나는 오류

① 문자열과 숫자를 그냥 더하기

타입이 다른 값을 +로 합치면 오류가 납니다. 파이썬은 문자열과 숫자를 자동으로 바꿔주지 않습니다.

python — 잘못된 코드
12
age = 20
print("나이는 " + age)   # 오류! str + int 불가
오류 메시지
TypeError: can only concatenate str (not "int") to str
✅ 수정 방법

str()로 숫자를 문자열로 바꿔서 더하거나, 다음 챕터에서 배울 f-string을 쓰면 됩니다.
print("나이는 " + str(age))

② 변수를 만들기 전에 사용하기

python — 잘못된 코드
1
print(score)   # 오류! score를 아직 만들지 않았음
오류 메시지
NameError: name 'score' is not defined
✅ 수정 방법

변수는 반드시 사용하기 전에 먼저 만들어야 합니다. score = 100을 먼저 쓰고 나서 print(score)를 써야 합니다.

③ int() 변환 불가능한 문자열

python — 잘못된 코드
1
print(int("hello"))   # 오류! "hello"는 숫자가 아님
오류 메시지
ValueError: invalid literal for int() with base 10: 'hello'
✅ 수정 방법

int()는 숫자처럼 생긴 문자열만 변환할 수 있습니다. "42", "100"처럼 실제로 숫자로 된 문자열만 가능합니다.

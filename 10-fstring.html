f-string 완전정리

문자열 포맷팅의 끝판왕 — 변수를 문자열 안에 자연스럽게 끼워 넣는 가장 세련된 방법

▸ 이 챕터를 마치면
  • f-string의 기본 문법을 이해하고 변수를 문자열 안에 삽입할 수 있다
  • f-string 안에서 연산, 메서드 호출, 조건 표현식을 쓸 수 있다
  • 숫자 서식 지정으로 소수점, 천 단위 구분자, 정렬을 제어할 수 있다
  • f-string 이전 방식들과 비교해 왜 f-string이 더 나은지 설명할 수 있다

1.왜 f-string인가?

변수 값을 문장 속에 넣어야 하는 상황은 코딩에서 매우 자주 발생합니다. 예를 들어 name = "Alice", score = 95일 때 "Alice님의 점수는 95점입니다."를 출력하려면 어떻게 해야 할까요?

파이썬에는 이 문제를 해결하는 방법이 여러 가지 있었습니다. 시대순으로 살펴보면:

python — 세 가지 방식 비교
12345678
name  = "Alice"
score = 95

# ① + 연결 방식 (불편함)
print(name + "님의 점수는 " + str(score) + "점입니다.")

# ② f-string 방식 (권장 ✅)
print(f"{name}님의 점수는 {score}점입니다.")
실행 결과
Alice님의 점수는 95점입니다.
Alice님의 점수는 95점입니다.

결과는 동일하지만 f-string 방식이 훨씬 읽기 쉽습니다. str()으로 타입 변환도 불필요하고, 따옴표와 +가 넘쳐나지도 않습니다. 파이썬 3.6부터 도입된 f-string은 현재 가장 권장되는 문자열 포맷팅 방법입니다.

💡 f-string이란?

formatted string literal의 줄임말입니다. 문자열 앞에 f 또는 F를 붙이고, 중괄호 { } 안에 변수나 표현식을 넣으면 파이썬이 자동으로 값을 채워줍니다.

2.f-string의 기본 구조

f"안녕하세요, {name}님!"
f 접두사
f 또는 F
따옴표
" 또는 '
일반 텍스트
그대로 출력
중괄호
{ } — 표현식 영역
표현식
변수 또는 계산식

기본 변수 삽입

python
1234567
name  = "Bob"
age   = 28
city  = "서울"

print(f"이름: {name}")            # 문자열 변수
print(f"나이: {age}살")           # 정수 변수
print(f"{name}은 {city}에 삽니다.")  # 여러 변수
실행 결과
이름: Bob
나이: 28살
Bob은 서울에 삽니다.
💡 타입 변환이 자동

f-string은 { } 안의 값을 자동으로 문자열로 변환합니다. str(age)처럼 직접 변환할 필요가 없습니다.

3.중괄호 안에 표현식 넣기

f-string의 강력함은 { } 안에 단순 변수뿐 아니라 어떤 파이썬 표현식도 넣을 수 있다는 데 있습니다. 계산식, 메서드 호출, 조건 표현식 모두 가능합니다.

계산식

python
123456
price = 12000
qty   = 3

print(f"단가: {price}원")
print(f"수량: {qty}개")
print(f"합계: {price * qty}원")  # 중괄호 안에서 바로 계산
실행 결과
단가: 12000원
수량: 3개
합계: 36000원

메서드 호출

python
1234
name = "python"

print(f"원본: {name}")
print(f"대문자: {name.upper()}")  # 중괄호 안에서 메서드 호출
실행 결과
원본: python
대문자: PYTHON

조건 표현식 (삼항 연산자)

값A if 조건 else 값B 형태의 조건 표현식을 { } 안에 넣을 수 있습니다. 조건문을 아직 배우지 않았더라도, 이 패턴은 매우 자주 쓰이므로 형태를 기억해두세요.

python
1234
score = 85

# score가 60 이상이면 "합격", 아니면 "불합격"
print(f"결과: {'합격' if score >= 60 else '불합격'}")
실행 결과
결과: 합격
⚠ 주의 — 따옴표 충돌

f-string 자체에 큰따옴표를 쓰고 있다면, { } 안에서는 작은따옴표를 써야 합니다. 반대도 마찬가지입니다. 위 예제처럼 f"...{'합격'} ..." 형태로 작성하세요.

4.숫자 서식 지정 — 콜론(:) 활용

f-string의 진짜 파워는 서식 지정자(format specifier)입니다. {변수:서식} 형태로 콜론 뒤에 서식을 지정하면 숫자를 원하는 형태로 표시할 수 있습니다.

FORMAT SPECIFIER 구조
{변수:채울문자정렬너비.소수점자리타입}
변수 — 출력할 값
채울문자 — 빈 자리를 채울 문자 (기본: 공백)
정렬 — < 왼쪽 / > 오른쪽 / ^ 가운데
너비 — 전체 출력 칸 수
소수점 — 소수점 이하 자릿수
타입 — f(소수) d(정수) s(문자열) %(퍼센트)
💡 전부 외울 필요 없어요

서식 지정자의 모든 옵션을 외울 필요는 없습니다. 소수점, 천 단위, 너비 정렬 이 세 가지만 익혀도 실무에서 충분합니다.

소수점 자릿수 제어

python
123456
pi    = 3.141592653589793
ratio = 0.8765

print(f"원주율: {pi:.2f}")       # 소수점 2자리
print(f"원주율: {pi:.5f}")       # 소수점 5자리
print(f"비율: {ratio:.1%}")      # 퍼센트로 변환 (소수점 1자리)
실행 결과
원주율: 3.14
원주율: 3.14159
비율: 87.7%

천 단위 구분자

python
1234
salary   = 52000000
product  = 1299000

print(f"연봉: {salary:,}원")      # 천 단위 쉼표
print(f"가격: {product:,}원")     # 천 단위 쉼표
실행 결과
연봉: 52,000,000원
가격: 1,299,000원

너비 지정과 정렬

출력 결과를 표처럼 정렬해야 할 때 유용합니다. <는 왼쪽, >는 오른쪽, ^는 가운데 정렬입니다.

python
12345678
# 10칸 너비로 각각 정렬
print(f"{'왼쪽':<10}|")    # 왼쪽 정렬 (기본값)
print(f"{'오른쪽':>10}|")   # 오른쪽 정렬
print(f"{'가운데':^10}|")   # 가운데 정렬

# 실용 예 — 성적표
print(f"{'이름':<8} {'점수':>5} {'등급':^4}")
print(f"{'Alice':<8} {95:>5} {'A':^4}")
print(f"{'Bob':<8} {82:>5} {'B':^4}")
실행 결과
왼쪽        |
      오른쪽|
    가운데  |
이름        점수 등급
Alice         95  A 
Bob           82  B

숫자 앞에 0 채우기

python
1234
hour   = 9
minute = 5
second = 3

print(f"현재 시각: {hour:02d}:{minute:02d}:{second:02d}")
실행 결과
현재 시각: 09:05:03

5.여러 줄 f-string과 중첩

여러 줄 f-string

삼중 따옴표(""")와 f-string을 함께 쓰면 여러 줄에 걸친 포맷팅이 가능합니다.

python
123456789
name    = "Alice"
product = "파이썬 입문서"
price   = 25000

receipt = f"""
주문자: {name}
상품명: {product}
결제금액: {price:,}원
"""
print(receipt)
실행 결과
주문자: Alice
상품명: 파이썬 입문서
결제금액: 25,000원

6.이전 방식과의 비교

파이썬에는 f-string 외에도 두 가지 포맷팅 방식이 있습니다. 오래된 코드에서 마주칠 수 있으니 읽을 줄은 알아야 합니다.

방식예시평가
+ 연결 "이름: " + name + ", 나이: " + str(age) ❌ 불편, str() 변환 필요
% 포맷 "이름: %s, 나이: %d" % (name, age) △ 구식, C 언어 스타일
.format() "이름: {}, 나이: {}".format(name, age) △ 가독성 보통, 파이썬 2 호환
f-string f"이름: {name}, 나이: {age}" ✅ 권장, 가독성 최고, 빠름
ℹ 언제 .format()을 쓰나요?

파이썬 3.5 이하를 지원해야 하는 레거시 프로젝트가 아니라면 f-string을 쓰세요. 새로 작성하는 코드에서는 f-string이 표준입니다.

7.자주 하는 실수

① f를 빠뜨린 경우

python — 잘못된 코드
12
name = "Alice"
print("{name}님 환영합니다")  # f 없음 → 변수 대입 안 됨!
실행 결과 (의도와 다름)
{name}님 환영합니다
✅ 수정 방법

문자열 앞에 f를 붙이세요. f"{name}님 환영합니다"

② 중괄호를 출력하고 싶을 때

f-string 안에서 중괄호 자체를 출력하려면 {{와 }}처럼 두 번 씁니다.

python
123
var = "name"
print(f"파이썬 변수: {{{var}}}")
# {{ → {  /  {var} → "name"  /  }} → }
실행 결과
파이썬 변수: {name}

③ 서식 지정자에서 콜론 위치 오류

python — 잘못된 코드
12
pi = 3.14159
print(f"{pi .2f}")  # 오류! 콜론이 빠짐
오류 메시지
SyntaxError: f-string: invalid syntax
✅ 수정 방법

서식 지정자는 반드시 콜론 뒤에 써야 합니다. f"{pi:.2f}"

▸ 핵심 요약
기본 문법 f"텍스트 {변수} 텍스트" — 문자열 앞에 f, 변수는 { }로 감싼다 표현식 삽입 f"{a + b}" / f"{name.upper()}" — { } 안에 계산식·메서드 모두 가능 소수점 f"{pi:.2f}" — 소수점 2자리까지 표시 천 단위 f"{n:,}" — 1,000,000처럼 쉼표 삽입 너비·정렬 f"{s:<10}" / f"{s:>10}" / f"{s:^10}" — 왼쪽·오른쪽·가운데 0 채우기 f"{n:02d}" — 2자리, 앞에 0 채움 (예: 09) 퍼센트 f"{0.875:.1%}" → 87.5% 중괄호 출력 f"{{중괄호}}" — {{ }}처럼 두 번 써서 중괄호 자체를 출력

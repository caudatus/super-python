Q50.

🏆 종합 도전 문제: 아래 변수들을 활용해 정해진 형식으로 출력하세요.

title    = "  super python  "   # 앞뒤 공백 제거 후 각 단어 첫 글자 대문자
version  = "3.12.0"             # . 기준으로 나눠서 첫 번째 숫자만
slogan   = "read write build"   # 단어 사이를 | 로 연결
출력:Super Python v3 | read|write|build
💡 힌트
① title.strip().title() → "Super Python"
② "v" + version.split(".")[0] → "v3"
③ "|".join(slogan.split()) → "read|write|build"
세 부분을 + 나 print() 인수로 조합하세요.
✅ 정답
title   = "  super python  "
version = "3.12.0"
slogan  = "read write build"

t = title.strip().title()         # "Super Python"
v = "v" + version.split(".")[0]  # "v3"
s = "|".join(slogan.split())     # "read|write|build"

print(t + " " + v + " | " + s)  # "Super Python v3 | read|write|build"

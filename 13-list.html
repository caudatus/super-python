리스트 — 여러 값을 하나로 묶기

변수 하나에 값 하나만 담을 수 있었죠. 리스트는 값 여러 개를 순서대로 담는 상자입니다. 파이썬에서 가장 많이 쓰이는 자료형이라 해도 과언이 아닙니다.

▸ 이 챕터를 마치면
  • 리스트를 만들고 특정 요소를 인덱스로 꺼낼 수 있다
  • 리스트를 수정·추가·삭제하는 핵심 메서드를 쓸 수 있다
  • 슬라이싱으로 리스트의 일부를 잘라낼 수 있다
  • len(), in, sorted() 등으로 리스트를 다룰 수 있다
  • 리스트 안에 리스트(중첩 리스트)가 어떻게 동작하는지 이해한다

1.리스트란 무엇인가?

지금까지 변수 하나에 값 하나씩 담았습니다. 그런데 반 학생 30명의 이름을 저장해야 한다면 어떻게 할까요?

python — 변수 하나에 값 하나 (비효율적)
123
name1 = "김민준"
name2 = "이서연"
name3 = "박지호"  # ... 30번까지 계속?

이렇게 하면 변수 30개가 필요합니다. 리스트(list)를 쓰면 한 줄로 해결됩니다.

python — 리스트로 해결
12
names = ["김민준", "이서연", "박지호"]
print(names)
OUTPUT
['김민준', '이서연', '박지호']

리스트는 대괄호 [ ]로 만들고, 각 값은 쉼표로 구분합니다. 리스트 안의 각 값을 요소(element)라고 부릅니다.

💡 리스트의 4가지 특징
①순서가 있다 — 요소들이 입력한 순서를 기억합니다.
②변경 가능하다 — 나중에 값을 추가, 수정, 삭제할 수 있습니다.
③중복을 허용한다 — 같은 값이 여러 번 들어와도 됩니다.
④타입이 섞여도 된다 — 문자열, 숫자, 불리언을 한 리스트에 담을 수 있습니다.
python — 다양한 리스트
1234567
numbers  = [10, 20, 30, 40, 50]      # 숫자 리스트
fruits   = ["사과", "바나나", "포도"]  # 문자열 리스트
mixed    = ["파이썬", 3, True]         # 타입 혼합 리스트
empty    = []                        # 빈 리스트
dupes    = [1, 1, 2, 3, 3]           # 중복 허용

print(type(numbers))
print(len(fruits))                   # 요소 개수
OUTPUT
<class 'list'>
3

2.인덱싱 — 요소 꺼내기

리스트의 각 요소에는 위치 번호가 붙어 있습니다. 이것을 인덱스(index)라고 합니다. 파이썬 인덱스는 0부터 시작합니다.

fruits =
[
"사과"
0 / -3
"바나나"
1 / -2
"포도"
2 / -1
]
■ 양수 인덱스 (앞에서) | ■ 음수 인덱스 (뒤에서)
python — 인덱싱
12345678
fruits = ["사과", "바나나", "포도"]

print(fruits[0])   # 첫 번째 요소
print(fruits[1])   # 두 번째 요소
print(fruits[2])   # 세 번째 요소 (마지막)

# 음수 인덱스 — 뒤에서 셉니다
print(fruits[-1])  # 마지막 요소
print(fruits[-2])  # 뒤에서 두 번째
OUTPUT
사과
바나나
포도
포도
바나나
⚠ IndexError — 범위를 벗어난 인덱스

요소가 3개인 리스트에서 fruits[3]을 쓰면 오류가 납니다. 인덱스는 0부터 len(리스트) - 1까지만 유효합니다.

IndexError
fruits[3]
IndexError: list index out of range

3.요소 수정하기

인덱스를 사용해 특정 위치의 값을 바꿀 수 있습니다.

python — 요소 수정
12345
scores = [80, 75, 90]
print(scores)

scores[1] = 85   # 두 번째 요소를 85로 변경
print(scores)
OUTPUT
[80, 75, 90]
[80, 85, 90]

4.슬라이싱 — 리스트 자르기

인덱스 하나로 요소 하나를 꺼냈다면, 슬라이싱은 범위를 지정해 여러 요소를 한 번에 꺼냅니다. 문자열 슬라이싱과 문법이 똑같습니다.

python — 슬라이싱 기본
12345678910
nums = [10, 20, 30, 40, 50]

print(nums[1:4])   # 인덱스 1 이상 4 미만 → [20, 30, 40]
print(nums[:3])    # 처음부터 3 미만 → [10, 20, 30]
print(nums[2:])    # 2 이상 끝까지 → [30, 40, 50]
print(nums[:])     # 전체 복사 → [10, 20, 30, 40, 50]
print(nums[::2])   # 2 간격으로 → [10, 30, 50]

# 역순 슬라이싱
print(nums[::-1])  # 전체 뒤집기 → [50, 40, 30, 20, 10]
OUTPUT
[20, 30, 40]
[10, 20, 30]
[30, 40, 50]
[10, 20, 30, 40, 50]
[10, 30, 50]
[50, 40, 30, 20, 10]
💡 슬라이싱 공식

리스트[start:stop:step] — start는 포함, stop은 미포함입니다. step을 생략하면 1씩 이동합니다. 슬라이싱은 항상 새 리스트를 반환하므로 원본이 바뀌지 않습니다.

5.핵심 메서드 — 추가·삭제·정렬

리스트를 다루는 메서드들을 한눈에 봅니다.

.append(value)
맨 끝에 요소 추가
fruits.append("딸기")
.insert(i, value)
인덱스 i 위치에 삽입
fruits.insert(1, "키위")
.remove(value)
해당 값의 첫 번째 항목 삭제
fruits.remove("바나나")
.pop(i)
i번째 요소 꺼내고 삭제 (인덱스 생략 시 마지막)
last = fruits.pop()
.sort()
리스트를 오름차순 정렬 (원본 변경)
nums.sort()
.reverse()
리스트 순서를 뒤집음 (원본 변경)
nums.reverse()
.index(value)
해당 값의 첫 번째 인덱스 반환
fruits.index("사과")
.count(value)
해당 값이 몇 번 등장하는지 반환
nums.count(3)
python — 메서드 실습
123456789101112131415161718
fruits = ["사과", "바나나", "포도"]

# 추가
fruits.append("딸기")
print(fruits)   # ['사과', '바나나', '포도', '딸기']

fruits.insert(1, "키위")
print(fruits)   # ['사과', '키위', '바나나', '포도', '딸기']

# 삭제
fruits.remove("바나나")
print(fruits)   # ['사과', '키위', '포도', '딸기']

popped = fruits.pop()
print(popped)   # 딸기 (꺼낸 값)
print(fruits)   # ['사과', '키위', '포도']

# 정렬
nums = [3, 1, 4, 1, 5]
nums.sort()
print(nums)     # [1, 1, 3, 4, 5]
OUTPUT
['사과', '바나나', '포도', '딸기']
['사과', '키위', '바나나', '포도', '딸기']
['사과', '키위', '포도', '딸기']
딸기
['사과', '키위', '포도']
[1, 1, 3, 4, 5]

6.리스트와 함께 쓰는 내장 함수

메서드 외에도 파이썬 기본 함수들이 리스트와 잘 어울립니다.

함수설명예시
len(lst)요소 개수 반환len([1,2,3]) → 3
sum(lst)숫자 리스트의 합계sum([10,20,30]) → 60
min(lst)최솟값 반환min([3,1,4]) → 1
max(lst)최댓값 반환max([3,1,4]) → 4
sorted(lst)정렬된 새 리스트 반환 (원본 유지)sorted([3,1,2]) → [1,2,3]
list()다른 타입을 리스트로 변환list("abc") → ['a','b','c']
💡 .sort() vs sorted() — 어떻게 다를까?

.sort()는 원본 리스트 자체를 바꾸고 None을 반환합니다. sorted()는 원본을 그대로 두고 정렬된 새 리스트를 반환합니다. 원본을 보존해야 한다면 sorted()를 쓰세요.

python — .sort() vs sorted()
12345678910
original = [3, 1, 4, 1, 5]

# sorted() — 원본 보존
new_list = sorted(original)
print("원본:", original)   # [3, 1, 4, 1, 5] 그대로
print("정렬:", new_list)   # [1, 1, 3, 4, 5]

# .sort() — 원본 변경
original.sort()
print("원본:", original)   # [1, 1, 3, 4, 5] 바뀜!
OUTPUT
원본: [3, 1, 4, 1, 5]
정렬: [1, 1, 3, 4, 5]
원본: [1, 1, 3, 4, 5]

7.in 연산자 — 포함 여부 확인

in 연산자로 특정 값이 리스트에 있는지 True / False로 확인할 수 있습니다.

python — in 연산자
123456
fruits = ["사과", "바나나", "포도"]

print("사과" in fruits)    # True
print("딸기" in fruits)    # False
print("딸기" not in fruits) # True
OUTPUT
True
False
True

8.리스트 연산 — + 와 *

문자열처럼 리스트도 +로 합치고 *로 반복할 수 있습니다.

python — 리스트 연산
1234567
a = [1, 2, 3]
b = [4, 5, 6]

print(a + b)     # 두 리스트 연결
print(a * 3)     # a를 3번 반복

# 초기값이 0인 리스트 5개 만들기 (자주 쓰는 패턴)
zeros = [0] * 5
print(zeros)
OUTPUT
[1, 2, 3, 4, 5, 6]
[1, 2, 3, 1, 2, 3, 1, 2, 3]
[0, 0, 0, 0, 0]

9.중첩 리스트 — 리스트 안에 리스트

리스트의 요소로 또 다른 리스트를 넣을 수 있습니다. 표 모양의 데이터(행렬, 성적표 등)를 표현할 때 자주 씁니다.

python — 중첩 리스트
123456789101112
# 학생별 [이름, 국어, 수학] 성적 표
grade = [
    ["김민준", 90, 85],
    ["이서연", 78, 92],
    ["박지호", 95, 88],
]

# 첫 번째 학생의 이름
print(grade[0][0])   # "김민준"

# 두 번째 학생의 수학 점수
print(grade[1][2])   # 92
OUTPUT
김민준
92
ℹ 읽는 법 — grade[행][열]

grade[1][2]는 "grade의 1번째 리스트에서 2번째 요소"를 의미합니다. 바깥 인덱스가 행(학생), 안쪽 인덱스가 열(항목)이라고 생각하면 쉽습니다.

③ .remove()는 값 기준, 같은 값이 여러 개면 첫 번째만 삭제

python
123

lst.remove(2)
print(lst)   # [1, 2, 3] — 첫 번째 2만 삭제됨
▸ 핵심 요약
리스트 생성 [값1, 값2, ...] — 대괄호 안에 쉼표로 구분 인덱싱 lst[0] 첫 번째, lst[-1] 마지막. 인덱스는 0부터 시작 슬라이싱 lst[start:stop:step] — stop은 미포함. 원본 변경 없음 추가 .append() 끝에 추가, .insert(i, v) i 위치에 삽입 삭제 .remove(v) 값으로 삭제, .pop(i) 인덱스로 꺼내고 삭제 정렬 .sort() 원본 변경, sorted() 새 리스트 반환 포함 확인 값 in 리스트 → True / False 유용한 함수 len() 개수, sum() 합, min() max() 최솟값·최댓값
📦
리스트를 손에 넣었습니다!

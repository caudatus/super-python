Q1.

open()의 세 가지 필수 인자를 빈칸에 채워 파일을 올바르게 열어보세요.
파일 이름은 "hello.txt", 모드는 읽기, 인코딩은 UTF-8로 설정하세요.

f = open(___, ___, encoding=___) f.close()
답 (3가지):"hello.txt", "r", "utf-8"
HINT

open(파일이름, 모드, encoding=인코딩) — 파일 이름과 인코딩은 문자열, 읽기 모드는 "r"입니다.

Q14.

fruits = ["apple", "banana", "cherry"]
파일 내용:apple banana cherry
HINT

writelines()는 줄바꿈을 자동으로 넣지 않습니다. 각 항목에 "\n"을 더하거나, 리스트 컴프리헨션으로 처리하세요.

Q16.

파일을 for 루프로 순회해 줄 번호를 함께 출력하는 코드를 작성하세요. numbers.txt는 3줄짜리 파일입니다.

출력 형태:1: 첫째줄내용 2: 둘째줄내용 3: 셋째줄내용

Q22.

아래 CSV 형식의 파일을 읽어 각 행을 딕셔너리 리스트로 만들어 출력하세요.

# students.txt 내용 Alice,90 Bob,85 Carol,92
출력:[{'name': 'Alice', 'score': 90}, {'name': 'Bob', 'score': 85}, {'name': 'Carol', 'score': 92}]
HINT

각 줄을 .strip().split(",")으로 분리하면 ["Alice", "90"] 형태가 됩니다. 이를 딕셔너리로 만드세요.

Q30.

🏆 종합 도전 문제 — 성적 관리 프로그램을 완성하세요.

  • 아래 형식의 scores.txt 파일을 읽습니다
  • 전체 평균, 최고점 학생, 최저점 학생을 출력합니다
  • 평균 이상인 학생만 골라 pass.txt에 저장합니다
# scores.txt 내용 민준,88 서연,95 지호,72 하은,81 도윤,90
출력:평균: 85.2 최고점: 서연 (95점) 최저점: 지호 (72점) pass.txt 저장 완료 (3명)
HINT

① 파일 읽어 리스트 of 튜플 만들기 → ② 평균 계산 → ③ max/min으로 최고/최저점 찾기 → ④ 평균 이상만 골라 파일 쓰기

ANSWER
# 1단계: 파일 읽기 students = [] with open("scores.txt", "r", encoding="utf-8") as f: for line in f: name, score = line.strip().split(",") students.append((name, int(score))) # 2단계: 통계 계산 avg = sum(s for _, s in students) / len(students) best = max(students, key=lambda x: x[1]) worst= min(students, key=lambda x: x[1]) print(f"평균: {avg}") print(f"최고점: {best[0]} ({best[1]}점)") print(f"최저점: {worst[0]} ({worst[1]}점)") # 3단계: 평균 이상 학생 파일 저장 passed = [(n, s) for n, s in students if s >= avg] with open("pass.txt", "w", encoding="utf-8") as f: for name, score in passed: f.write(f"{name},{score}\n") print(f"pass.txt 저장 완료 ({len(passed)}명)")

조건문 — if, elif, else

프로그램이 스스로 판단하게 만드는 방법. 조건이 참이면 이걸 하고, 거짓이면 저걸 하라 — 이것이 모든 프로그램의 핵심입니다.

▸ 이 챕터를 마치면
  • if 문으로 조건이 참일 때만 코드를 실행할 수 있다
  • elif와 else로 여러 경우를 순서대로 분기할 수 있다
  • 비교 연산자(==, !=, >, <, >=, <=)를 정확히 사용할 수 있다
  • and, or, not으로 조건을 조합할 수 있다
  • 들여쓰기가 왜 중요한지 이해하고 올바르게 쓸 수 있다

1.조건문이란?

지금까지 작성한 코드는 위에서 아래로 순서대로 실행됐습니다. 하지만 현실의 프로그램은 항상 같은 행동만 하지 않습니다.

카페 앱을 생각해봅시다. 나이가 19세 이상이면 아메리카노를 주문할 수 있고, 미만이면 "성인만 가능합니다"라는 메시지를 보여줍니다. 이렇게 조건에 따라 다른 동작을 실행하는 것이 조건문입니다.

나이 입력
│
▼
나이 >= 19 ?
YES ✓
│
▼
주문 완료!
NO ✗
│
▼
성인만 가능합니다

파이썬에서는 이 로직을 if문 하나면 표현할 수 있습니다.

2.if 문 기본 구조

if는 "만약 ~라면"이라는 뜻입니다. 조건이 True일 때만 들여쓰기된 블록이 실행됩니다.

if 조건식:← 콜론(:) 필수!
실행할 코드← 반드시 들여쓰기(스페이스 4칸)
실행할 코드 2← 같은 들여쓰기 = 같은 블록
다음 코드← 들여쓰기 없음 = if 블록 밖
python
12345
age = int(input("나이를 입력하세요: "))

if age >= 19:
    print("어서오세요! 주문할 수 있습니다.")

print("프로그램 종료")  # 항상 실행됨
실행 결과 (입력: 20)
어서오세요! 주문할 수 있습니다.
프로그램 종료
실행 결과 (입력: 16)
프로그램 종료

나이가 16이면 if 블록 안의 print는 실행되지 않고, 블록 밖의 "프로그램 종료"는 항상 실행됩니다.

⚠ 들여쓰기 — 파이썬에서 가장 중요한 규칙

파이썬은 들여쓰기로 코드 블록을 구분합니다. 스페이스 4칸이 표준입니다. 탭과 스페이스를 섞으면 오류가 납니다.

콜론(:) 뒤에 반드시 들여쓰기된 블록이 있어야 합니다. 없으면 IndentationError가 발생합니다.

3.else — 그렇지 않다면

else는 if 조건이 False일 때 실행되는 블록입니다. "아니면"이라고 생각하면 됩니다.

python
123456
age = int(input("나이를 입력하세요: "))

if age >= 19:
    print("어서오세요! 주문할 수 있습니다.")
else:
    print("죄송합니다. 성인만 이용 가능합니다.")
실행 결과 (입력: 17)
죄송합니다. 성인만 이용 가능합니다.

if와 else는 반드시 둘 중 하나만 실행됩니다. 둘 다 실행되거나 둘 다 건너뛰는 일은 없습니다.

4.elif — 아니면, 이 조건은?

경우가 셋 이상일 때는 elif(else if의 줄임말)를 사용합니다. elif는 앞의 조건이 모두 False일 때만 검사됩니다.

python
12345678910
score = int(input("점수를 입력하세요: "))

if score >= 90:
    print("A등급")
elif score >= 80:
    print("B등급")
elif score >= 70:
    print("C등급")
else:
    print("재시험")
실행 결과 (입력: 85)
B등급

점수 85는 90 이상이 아니므로 첫 번째 if를 통과하고, 80 이상이므로 elif에서 잡힙니다. 한 번 조건에 걸리면 나머지 elif와 else는 모두 건너뜁니다.

💡 elif는 몇 개든 쓸 수 있습니다

elif는 필요한 만큼 이어 붙일 수 있습니다. else는 선택 사항이며, 쓴다면 반드시 맨 마지막에 옵니다. 순서는 항상 if → elif → elif → ... → else입니다.

5.비교 연산자

조건식에는 비교 연산자를 사용합니다. 결과는 항상 True 또는 False입니다.

연산자의미예시결과
==같다5 == 5True
!=다르다5 != 3True
>크다7 > 3True
<작다2 < 1False
>=크거나 같다5 >= 5True
<=작거나 같다4 <= 3False
⚠ = 과 == 는 완전히 다릅니다

=는 대입(x = 5 → x에 5를 넣는다), ==는 비교(x == 5 → x가 5인가?)입니다. 조건식에서 =를 쓰면 SyntaxError가 납니다.

python
123456
name = input("이름: ")

if name == "관리자":        # 문자열도 == 로 비교
    print("관리자 페이지로 이동합니다.")
else:
    print(f"안녕하세요, {name}님!")
실행 결과 (입력: 관리자)
관리자 페이지로 이동합니다.

6.논리 연산자 — and, or, not

두 조건을 함께 검사하거나 반전시킬 때 논리 연산자를 씁니다.

연산자의미예시결과
and둘 다 참일 때만 Trueage >= 18 and age <= 6518~65세면 True
or하나라도 참이면 Truescore >= 90 or bonus == True90점 이상이거나 보너스면 True
not참/거짓 반전not name == "손님"손님이 아니면 True
python
12345678910
age   = int(input("나이: "))
score = int(input("점수: "))

# and — 나이가 18세 이상이고 점수가 60점 이상이어야 합격
if age >= 18 and score >= 60:
    print("합격입니다!")
else:
    print("조건을 충족하지 못했습니다.")

# or — 나이가 65세 이상이거나 장애인이면 우선 탑승
실행 결과 (나이: 20, 점수: 75)
합격입니다!
💡 파이썬 범위 조건 — 수학처럼 쓸 수 있습니다

다른 언어와 달리 파이썬은 0 <= score <= 100처럼 연속 비교를 지원합니다. score >= 0 and score <= 100과 완전히 같습니다.

8.중첩 if — if 안에 if

if 블록 안에 또 다른 if를 넣을 수 있습니다. 이를 중첩 if라고 합니다.

python
1234567891011
score = int(input("점수: "))
extra = input("보너스 점수 있음? (y/n): ")

if score >= 60:
    print("합격!")
    if extra == "y":           # 중첩 if — 합격인 경우에만 검사
        print("보너스 점수로 장학금 대상입니다!")
else:
    print("불합격. 재시험 기회가 있습니다.")
실행 결과 (점수: 80, 보너스: y)
합격!
보너스 점수로 장학금 대상입니다!
if score >= 60: ← 바깥 if (들여쓰기 0)
    print("합격!") ← 4칸 들여쓰기
    if extra == "y": ← 중첩 if (4칸)
   print("장학금!") ← 8칸 들여쓰기
else: ← 바깥 if의 else
print("불합격")
💡 중첩보다 and가 더 간결할 때도 있습니다

조건 두 개를 모두 만족해야 한다면 중첩 대신 and로 합치는 게 더 읽기 좋을 수 있습니다.
if score >= 60 and extra == "y": → 중첩 없이 같은 결과.

9.자주 하는 실수

실수 1 — 콜론(:) 빠뜨리기

SyntaxError

    print("A등급")
SyntaxError: expected ':'

실수 2 — = 와 == 혼동

SyntaxError
if name = "admin":    ← = 는 대입, 조건식에는 == 를 써야 합니다
SyntaxError: invalid syntax. Maybe you meant '==' ?

IndentationError
if score >= 60:
  print("합격")    ← 스페이스 2칸

SyntaxError
else:
    print("기타")
elif score >= 80:   ← else 뒤에 elif가 올 수 없습니다

Q1.

좌표를 나타내는 튜플 point = (3, 7)을 만들고, 첫 번째 값(x 좌표)을 출력하세요.

출력:3
HINT

튜플도 리스트처럼 인덱스로 접근합니다: 튜플[0]이 첫 번째 값이에요.

ANSWER
point = (3, 7) print(point[0]) # 인덱스 0 → x 좌표 3
Q2.

튜플 colors = ("red", "green", "blue")의 길이를 출력하세요.

출력:3
HINT

len() 함수는 튜플에도 사용할 수 있어요.

ANSWER
colors = ("red", "green", "blue") print(len(colors)) # 튜플 요소 개수 → 3

튜플 언패킹: a, b, c = (10, 20, 30) — 좌변의 변수 수와 튜플 원소 수가 일치해야 해요.

ANSWER

Q5.

셋(set) fruits = {"apple", "banana", "cherry"}을 만들고, "banana"가 포함되어 있는지 확인해 출력하세요.

출력:True
HINT

"값" in 셋 — 셋에서의 in 연산은 리스트보다 훨씬 빠릅니다.

ANSWER
fruits = {"apple", "banana", "cherry"} print("banana" in fruits) # 포함 여부 → True
Q6.

아래 리스트에서 중복을 제거한 셋을 만들고, 셋의 길이를 출력하세요.

nums = [1, 2, 2, 3, 3, 3, 4]
출력:4
HINT

set(리스트)로 리스트를 셋으로 변환하면 중복이 자동 제거됩니다.

ANSWER
nums = [1, 2, 2, 3, 3, 3, 4] unique = set(nums) # 중복 제거 print(len(unique)) # 고유값 4개 → 4
Q7.

셋 s = {1, 2, 3}에 add()로 4를 추가하고, 셋을 출력하세요.

출력 (순서 무관):{1, 2, 3, 4}
HINT

셋에 요소를 추가할 때는 셋.add(값)을 사용합니다. 리스트의 append()와 비슷하지만 이름이 달라요.

ANSWER
s = {1, 2, 3} s.add(4) # 요소 추가 print(s) # {1, 2, 3, 4} — 셋은 순서 없음
Q8.

셋 s = {10, 20, 30, 40}에서 discard()로 20을 제거하고, 셋을 출력하세요.

출력 (순서 무관):{10, 30, 40}
HINT

셋.discard(값) — 값이 없어도 오류가 나지 않는 안전한 제거 방법입니다. remove()는 없으면 KeyError가 발생해요.

ANSWER
s = {10, 20, 30, 40} s.discard(20) # 20 제거 (없어도 에러 없음) print(s)

아래 두 셋의 합집합(union)을 구해 출력하세요.

a = {1, 2, 3} b = {3, 4, 5}
출력 (순서 무관):{1, 2, 3, 4, 5}
HINT

a | b 또는 a.union(b) — 두 방법 모두 합집합을 구합니다.

ANSWER
a = {1, 2, 3} b = {3, 4, 5} print(a | b) # 합집합: 두 셋의 모든 원소 (중복 없이)

Q10.

아래 두 셋의 교집합(intersection)을 구해 출력하세요.

a = {1, 2, 3, 4} b = {3, 4, 5, 6}
출력 (순서 무관):{3, 4}
HINT

a & b 또는 a.intersection(b) — 두 셋 모두에 있는 원소만 반환합니다.

ANSWER
a = {1, 2, 3, 4} b = {3, 4, 5, 6} print(a & b) # 교집합: 공통 원소만
Q11.

아래 두 셋의 차집합(a에만 있는 것)을 구해 출력하세요.

a = {1, 2, 3, 4} b = {3, 4, 5, 6}
출력 (순서 무관):{1, 2}
HINT

a - b 또는 a.difference(b) — a에는 있지만 b에는 없는 원소를 반환합니다.

ANSWER
a = {1, 2, 3, 4} b = {3, 4, 5, 6} print(a - b) # 차집합: a에만 있는 원소
Q12.

아래 두 셋의 대칭차집합(어느 한쪽에만 있는 것)을 구해 출력하세요.

a = {1, 2, 3} b = {2, 3, 4}
출력 (순서 무관):{1, 4}
HINT

a ^ b 또는 a.symmetric_difference(b) — 한쪽에만 있고 양쪽 모두에는 없는 원소를 반환합니다.

ANSWER
a = {1, 2, 3} b = {2, 3, 4} print(a ^ b) # 대칭차집합: 한쪽에만 있는 원소
Q13.

튜플 scores = (88, 95, 72, 95, 81, 72, 90)에서 95가 몇 번 등장하는지 출력하세요.

출력:2
HINT

튜플도 리스트처럼 .count(값) 메서드를 지원합니다.

ANSWER
scores = (88, 95, 72, 95, 81, 72, 90) print(scores.count(95)) # 95 등장 횟수 → 2
Q14.

튜플 t = (5, 3, 8, 1, 9, 2)를 정렬된 리스트로 변환해 출력하세요.

출력:[1, 2, 3, 5, 8, 9]
HINT

튜플 자체는 정렬할 수 없지만 sorted(튜플)은 정렬된 리스트를 반환합니다.

ANSWER
t = (5, 3, 8, 1, 9, 2) print(sorted(t)) # sorted()는 리스트를 반환
Q15.

아래 리스트에서 중복을 제거한 뒤 다시 정렬된 리스트로 반환해 출력하세요.

data = [5, 3, 5, 1, 3, 2, 4, 2]
출력:[1, 2, 3, 4, 5]
HINT

sorted(set(리스트)) — set으로 중복 제거 후, sorted로 정렬하면 한 줄로 해결됩니다.

data = [5, 3, 5, 1, 3, 2, 4, 2] print(sorted(set(data))) # set으로 중복 제거 → sorted로 정렬
Q16.

아래 코드의 출력 결과를 예상하고 직접 작성해 확인하세요.

t = (1, [2, 3], 4) t[1].append(99) print(t)
출력:(1, [2, 3, 99], 4)
HINT

튜플 자체는 불변이지만, 그 안에 담긴 리스트는 가변입니다. 튜플의 두 번째 원소가 가리키는 리스트 객체 자체를 바꾸는 것은 가능합니다.

ANSWER
# 튜플 불변 ≠ 튜플 안 리스트 불변 t = (1, [2, 3], 4) t[1].append(99) # 리스트 원소는 변경 가능 print(t) # (1, [2, 3, 99], 4)
Q17.

small = {2, 4} big = {1, 2, 3, 4, 5}
출력:True
HINT

small.issubset(big) 또는 small <= big — "small의 모든 원소가 big 안에 있나요?"

Q23.

아래 튜플 목록에서 점수(score)를 기준으로 최고점 학생의 이름을 출력하세요.

records = [ ("민준", 88), ("서연", 95), ("지호", 72), ("하은", 91) ]

max(records, key=lambda x: x[1]) — key에 람다를 사용해 두 번째 원소(점수) 기준으로 최댓값을 찾을 수 있어요.

Q24.

HINT

Q26.

아래 함수가 여러 값을 반환하고 있습니다. 반환값을 받아 각 변수에 언패킹하고 출력하세요.

def get_stats(nums): return min(nums), max(nums), sum(nums) / len(nums) data = [4, 7, 2, 9, 1, 6]
출력:최솟값: 1 최댓값: 9 평균: 4.833333333333333
HINT

파이썬에서 함수가 쉼표로 여러 값을 반환하면 실제로는 튜플로 반환됩니다. low, high, avg = get_stats(data)로 언패킹하세요.

ANSWER
def get_stats(nums): return min(nums), max(nums), sum(nums) / len(nums) data = [4, 7, 2, 9, 1, 6] low, high, avg = get_stats(data) # 튜플 언패킹 print(f"최솟값: {low}") print(f"최댓값: {high}") print(f"평균: {avg}")
Q27.

아래 문장에서 단어를 분리해 중복 없는 고유 단어 셋을 만들고, 고유 단어 수를 출력하세요.

sentence = "the quick brown fox jumps over the lazy dog the fox"
출력:8
HINT

sentence.split()로 단어 리스트를 만들고, set()으로 중복을 제거한 뒤 len()으로 세어보세요.

ANSWER
sentence = "the quick brown fox jumps over the lazy dog the fox" words = sentence.split() # 공백 기준 분리 unique_words = set(words) # 중복 제거 print(len(unique_words)) # 고유 단어 수 → 8

출력:{'지호'}

ANSWER
a = {"민준", "서연", "지호", "하은"} b = {"서연", "지호", "도윤", "소민"} c = {"지호", "하은", "소민", "유나"} print(a & b & c) # 세 셋 모두의 교집합
Q29.

아래 리스트에서 한 번만 등장하는 숫자들만 모아 정렬된 리스트로 출력하세요.

nums = [1, 2, 3, 2, 4, 3, 5]
출력:[1, 4, 5]
HINT

리스트 컴프리헨션 + count(): sorted([x for x in nums if nums.count(x) == 1])
또는 set을 활용: 한 번만 등장 = 전체 셋 - 두 번 이상 등장하는 원소 셋

🏆 종합 도전 문제 — 쇼핑몰 장바구니 분석기를 작성하세요.

  • 고객 A, B가 각각 장바구니에 담은 상품이 아래 두 셋에 있습니다
  • 두 고객이 모두 담은 상품 수를 공통: N개 형태로 출력하세요
  • 고객 A만 담은 상품을 정렬해 A만: ['상품', ...] 형태로 출력하세요
  • 두 고객 중 어느 한 명이라도 담은 전체 상품 수를 전체: N개 형태로 출력하세요
cart_a = {"노트북", "마우스", "키보드", "모니터"} cart_b = {"마우스", "키보드", "헤드셋", "웹캠"}
출력:공통: 2개 A만: ['노트북', '모니터'] 전체: 6개
HINT

공통: len(cart_a & cart_b), A만: sorted(cart_a - cart_b), 전체: len(cart_a | cart_b)

ANSWER
cart_a = {"노트북", "마우스", "키보드", "모니터"} cart_b = {"마우스", "키보드", "헤드셋", "웹캠"} # 교집합: 두 고객 모두 담은 상품 print(f"공통: {len(cart_a & cart_b)}개") # 차집합: A만 담은 상품 (정렬) print(f"A만: {sorted(cart_a - cart_b)}") # 합집합: 전체 고유 상품 수 print(f"전체: {len(cart_a | cart_b)}개")

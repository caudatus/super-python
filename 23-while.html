while문, break, continue

조건이 참인 동안 계속 반복하는 while문과, 반복을 원하는 시점에 멈추거나 건너뛰는 break·continue를 배웁니다.

이 챕터를 마치면
  • while문의 동작 원리를 설명할 수 있다
  • for문과 while문을 상황에 맞게 선택할 수 있다
  • break로 반복을 즉시 중단할 수 있다
  • continue로 특정 반복 회차를 건너뛸 수 있다
  • 무한 루프를 이해하고 안전하게 다룰 수 있다

01while문이란?

for문은 "리스트나 range의 모든 항목을 한 번씩 돌아라"라는 명령입니다. 반면 while문은 조금 다릅니다. "이 조건이 참인 동안 계속 반복해라"라는 명령입니다.

일상으로 비유하면 이렇습니다. for문은 "서랍 10칸을 모두 열어봐"처럼 횟수가 정해져 있습니다. while문은 "배가 부를 때까지 계속 먹어"처럼 조건이 충족될 때까지 계속합니다. 언제 끝날지는 조건에 달려 있습니다.

기본 구조

python
12
while 조건식:
    반복할 코드   # 조건이 True인 동안 계속 실행

동작 순서를 단계별로 살펴보겠습니다.

① 조건식을 평가한다
② True → 블록 안의 코드를 실행 → 다시 ①로
③ False → while문 종료, 다음 코드로 이동

가장 단순한 예시

python
12345678
count = 1              # 시작 값

while count <= 5:      # count가 5 이하인 동안 반복
    print(count)
    count += 1           # 매 반복마다 count를 1 증가

# 출력:
# 1  2  3  4  5
⚠ 핵심 주의사항

count += 1 처럼 조건에 영향을 주는 변화가 반드시 있어야 합니다. 이 줄이 없으면 count는 영원히 1이고, 조건은 영원히 True여서 프로그램이 멈추지 않습니다. 이것을 무한 루프라고 합니다.

02for문 vs while문 — 언제 무엇을 쓸까?

둘 다 "반복"이지만 적합한 상황이 다릅니다.

구분for문while문
반복 횟수 미리 정해져 있을 때 조건에 따라 달라질 때
전형적 사용 리스트 순회, range() 반복 사용자 입력 대기, 게임 루프
예시 성적 10개를 모두 출력 정답 맞힐 때까지 계속 질문
💡 선택 기준

"몇 번 반복할지 미리 알 수 있다" → for문을 쓰세요.
"언제 끝날지 실행해봐야 안다" → while문이 자연스럽습니다.

03while문 실전 패턴

패턴 1 — 카운터 기반 반복

for문처럼 특정 횟수를 반복하고 싶을 때도 while로 쓸 수 있습니다.

python
12345
i = 0
while i < 3:
    print(f"반복 {i+1}회")
    i += 1
# 출력: 반복 1회 / 반복 2회 / 반복 3회

패턴 2 — 누적 계산

조건에 따라 합계·곱셈 등을 누적하는 패턴입니다.

python
123456
total = 0
n = 1
while n <= 100:
    total += n
    n += 1
print(total)   # 5050 (1+2+...+100)

패턴 3 — 사용자 입력 검증

이것이 while문이 for문보다 훨씬 편리한 대표적 상황입니다. 사용자가 올바른 값을 입력할 때까지 반복합니다.

python
1234567
password = ""

while password != "1234":
    password = input("비밀번호를 입력하세요: ")

print("로그인 성공!")
# 비밀번호가 "1234"가 될 때까지 계속 묻습니다

위 코드를 변형해서 5의 배수를 입력할 때까지 반복하는 프로그램을 만들어보세요.

힌트: while int(n) % 5 != 0:

04무한 루프와 while True

조건이 항상 참이면 루프는 영원히 돌아갑니다. while True:는 이것을 의도적으로 사용하는 패턴입니다. 언뜻 위험해 보이지만, break와 함께 쓰면 아주 유용합니다.

python
12345678
while True:          # 무한 루프 시작
    answer = input("'quit'을 입력하면 종료: ")
    if answer == "quit":
        break          # ← 여기서 루프 탈출
    print(f"입력: {answer}")

print("프로그램 종료")
# "quit"을 입력하기 전까지 계속 입력을 받습니다
ℹ 실수로 무한 루프에 빠졌다면

터미널에서 Ctrl + C 를 누르면 강제 중단됩니다. VS Code에서는 실행 패널의 ■ 정지 버튼을 클릭하세요.

05break — 반복을 즉시 중단

break는 "지금 당장 반복문을 탈출해라"라는 명령입니다. break를 만나는 순간, 반복 조건과 관계없이 즉시 루프 바깥으로 빠져나갑니다.

python
12345678910
numbers = [3, 7, 2, 9, 4, 6]

for n in numbers:
    if n == 9:
        print("9를 찾았습니다!")
        break          # 9를 찾은 순간 즉시 탈출
    print(n)

# 출력:
# 3  7  2  9를 찾았습니다!   ← 9 이후의 4, 6은 출력되지 않음

break는 for문과 while문 모두에 사용할 수 있습니다. 특히 "찾으면 멈춘다", "틀리면 즉시 종료" 같은 패턴에 자주 씁니다.

while + break 조합 (검색 패턴)

python
123456789
count = 0

while True:
    guess = input("숫자를 맞혀보세요 (정답: 7): ")
    count += 1
    if guess == "7":
        print(f"정답! {count}번 만에 맞혔습니다.")
        break
    print("틀렸습니다. 다시 시도하세요.")

06continue — 이번 회차만 건너뛰기

continue는 break와 헷갈리기 쉽습니다. 차이를 명확히 짚겠습니다.

  • break → 반복문 완전 탈출, 루프 이후 코드로 이동
  • continue → 이번 회차의 나머지 코드만 건너뛰기, 다음 반복은 계속 진행
python
12345678
for i in range(1, 7):
    if i % 2 == 0:
        continue       # 짝수이면 print를 건너뛰고 다음 i로
    print(i)

# 출력:
# 1  3  5   ← 짝수(2,4,6)는 건너뜁니다
💡 continue 동작 원리

continue를 만나면 해당 반복의 나머지 줄을 실행하지 않고, for문이면 다음 항목으로, while문이면 조건 검사로 즉시 돌아갑니다.

while + continue 예시

python
123456789
n = 0
while n < 10:
    n += 1
    if n % 3 == 0:
        continue       # 3의 배수는 출력 건너뜀
    print(n)

# 출력:
# 1  2  4  5  7  8  10   ← 3, 6, 9 제외
⚠ while + continue 무한 루프 주의

while문에서 continue를 쓸 때는 반드시 증감 코드(n += 1)를 continue 앞에 두세요. continue 뒤에 두면 건너뛰어 증감이 안 되고 무한 루프에 빠집니다.

✏ 직접 해보세요

break와 continue의 차이를 직접 체감해보세요. 아래 두 코드를 실행해서 출력 결과를 비교해 보세요.

코드 A (break): for i in range(5): if i == 3: break / print(i)

코드 B (continue): for i in range(5): if i == 3: continue / print(i)

코드 A는 3에서 멈추고, 코드 B는 3만 건너뛰고 4도 출력됩니다.

07중첩 반복문에서 break / continue

중첩 반복문(반복문 안의 반복문)에서 break와 continue는 자신이 속한 가장 안쪽 반복문에만 영향을 미칩니다.

python
123456789
for i in range(1, 4):          # 바깥 루프
    for j in range(1, 4):      # 안쪽 루프
        if j == 2:
            break              # 안쪽 루프만 탈출
        print(f"{i}-{j}")

# 출력:
# 1-1  2-1  3-1   ← 각 i마다 j=1만 출력되고 j=2에서 안쪽 break
# 바깥 루프(i)는 계속 진행됩니다
ℹ 바깥 루프까지 탈출하려면

파이썬에는 중첩 루프 전체를 한 번에 탈출하는 전용 문법이 없습니다. 실무에서는 플래그 변수(found = True)를 활용하거나, 반복 로직을 함수로 분리하고 return을 쓰는 방법을 많이 씁니다. 함수는 29챕터에서 배웁니다.

08종합 예시 — 숫자 맞히기 게임

지금까지 배운 while, break, continue를 모두 활용한 간단한 게임입니다. 아직 배우지 않은 random 모듈은 "무작위 숫자 뽑기"로만 이해하면 됩니다.

python
1234567891011121314151617181920
import random

answer = random.randint(1, 10)  # 1~10 사이의 랜덤 정수
tries = 0

print("1~10 사이의 숫자를 맞혀보세요!")

while True:
    guess = input("숫자 입력: ")

    if not guess.isdigit():
        print("숫자만 입력해주세요!")
        continue              # 숫자가 아니면 tries 증가 없이 건너뜀

    tries += 1
    guess = int(guess)

    if guess == answer:
        print(f"정답! {tries}번 만에 맞혔습니다 🎉")
        break                 # 정답이면 게임 종료
    elif guess < answer:
        print("더 큰 숫자입니다!")
    else:
        print("더 작은 숫자입니다!")
💡 코드 포인트

while True로 무한 루프를 만들고, 숫자가 아닌 입력은 continue로 건너뛰며, 정답을 맞히면 break로 탈출합니다. 세 가지 개념이 자연스럽게 협력하는 구조입니다.

CHAPTER SUMMARY
while 조건:
조건이 True인 동안 반복. 조건이 False가 되면 종료.
while True:
의도적 무한 루프. 반드시 break와 함께 사용.
break
반복문 즉시 탈출. 가장 안쪽 루프에만 적용.
continue
이번 회차 나머지 코드 건너뜀. 다음 반복은 계속.
for vs while
횟수 고정 → for / 조건 기반, 횟수 불명 → while
무한 루프 탈출
실수 시 Ctrl+C 로 강제 중단 가능.

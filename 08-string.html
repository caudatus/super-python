파이썬의 문자열

텍스트를 저장하고, 자르고, 변형하는 모든 것

▸ 이 챕터를 마치면
  • 문자열을 만드는 네 가지 방법과 이스케이프 시퀀스를 이해한다
  • 인덱싱으로 특정 문자를 꺼내고, 슬라이싱으로 부분 문자열을 추출할 수 있다
  • len() / upper() / lower() / strip() / replace() / split() / find() / count() / startswith() / endswith() / join() 를 활용한다
  • 문자열이 불변(immutable)임을 이해하고 올바른 수정 방법을 안다

1.문자열 만들기 — 네 가지 방법

문자열(String)은 문자들의 순서 있는 나열입니다. 파이썬에서는 따옴표로 감싸면 무엇이든 문자열이 됩니다.

python
123456789101112
# 1. 작은따옴표
a = 'Hello'

# 2. 큰따옴표
b = "파이썬"

# 3. 작은따옴표 세 개 — 여러 줄 문자열
c = '''안녕하세요
반갑습니다'''

# 4. 큰따옴표 세 개 — 여러 줄 문자열
d = """첫 번째 줄
두 번째 줄"""
💡 언제 어떤 따옴표?

평소엔 "큰따옴표"를 기본으로 쓰세요. 문자열 안에 큰따옴표가 포함될 땐 '작은따옴표'로 감싸거나 이스케이프(\")를 씁니다. 여러 줄 문자열은 """삼중 큰따옴표"""가 편리합니다.

이스케이프 시퀀스 — 특수 문자 넣기

문자열 안에 탭, 줄바꿈, 따옴표 등 특수 문자를 넣으려면 \(백슬래시)를 앞에 붙입니다.

python
123456
print("줄바꿈:\n두 번째 줄")    # \n = 줄바꿈
print("탭:\t오른쪽")             # \t = 탭
print("그가 말했다. \"안녕!\"")   # \" = 큰따옴표
print('It\'s Python')           # \' = 작은따옴표
print("경로: C:\\Users")         # \\ = 백슬래시 자체
print(r"C:\Users\new")           # r"..." = raw string, \를 그대로 출력
실행 결과
줄바꿈:
두 번째 줄
탭:	오른쪽
그가 말했다. "안녕!"
It's Python
경로: C:\Users
C:\Users\new

2.인덱싱 — 특정 문자 꺼내기

문자열은 문자 하나하나에 번호(인덱스)가 붙어 있습니다. 파이썬은 0번부터 시작하고, 뒤에서부터 세는 음수 인덱스도 지원합니다.

word = "PYTHON"

print(word[0])   # P  — 첫 번째 문자
print(word[2])   # T
print(word[5])   # N  — 마지막 문자

print(word[-1])  # N  — 뒤에서 첫 번째
print(word[-2])  # O  — 뒤에서 두 번째
print(word[-6])  # P  — 첫 번째와 같음

⚠ IndexError — 범위 초과

"PYTHON"은 인덱스 0~5까지만 유효합니다. word[6]처럼 범위를 벗어나면 IndexError가 발생합니다. 마지막 유효 인덱스는 언제나 len(문자열) - 1입니다.

3.슬라이싱 — 부분 문자열 잘라내기

인덱싱이 문자 하나를 꺼낸다면, 슬라이싱은 원하는 범위를 한꺼번에 잘라냅니다.

문자열[ start : stop : step ]
start — 시작 인덱스 (포함, 생략하면 처음부터)
stop — 끝 인덱스 (미포함, 생략하면 끝까지)
step — 간격 (생략하면 1, 음수면 역방향)
python
123456789101112
s = "PYTHON"

print(s[0:3])   # PYT    — 인덱스 0, 1, 2 (3 미포함)
print(s[2:5])   # THO    — 인덱스 2, 3, 4
print(s[2:])    # THON   — 2번부터 끝까지
print(s[:4])    # PYTH   — 처음부터 3번까지
print(s[:])     # PYTHON — 전체 복사

# step 활용
print(s[::2])   # PTO    — 한 칸씩 건너뛰기
print(s[::-1])  # NOHTYP — 뒤집기!

# 음수 인덱스 슬라이싱
print(s[-3:])   # HON    — 뒤에서 3글자
print(s[:-2])   # PYTH   — 끝 2글자 제외

💡 [::-1] 은 문자열 뒤집기의 관용구

s[::-1]은 파이썬에서 문자열을 뒤집는 가장 간결한 방법입니다. 알고리즘 문제에서 회문(palindrome) 판별에 자주 쓰입니다. 기억해 두면 두고두고 유용합니다.

4.len() — 문자열 길이

len()은 문자열의 글자 수를 반환합니다. 공백도 문자 1개로 셉니다.

python
12345
print(len("PYTHON"))     # 6
print(len("슈퍼파이썬"))   # 6  — 한글도 1글자 = 1
print(len("hello world")) # 11 — 공백 포함
print(len(""))           # 0  — 빈 문자열
word = "PYTHON"
print(word[len(word) - 1])  # N — 마지막 문자 (word[-1]과 동일)

5.핵심 메서드 — 변형과 검색

문자열 메서드는 문자열.메서드이름() 형식으로 씁니다. 메서드는 원래 문자열을 바꾸지 않고 새 문자열을 반환합니다.

대소문자 변환

메서드역할예시결과
.upper()전체 대문자"hello".upper()"HELLO"
.lower()전체 소문자"HELLO".lower()"hello"
.capitalize()첫 글자만 대문자"hello world".capitalize()"Hello world"
.title()각 단어 첫 글자 대문자"hello world".title()"Hello World"
python
12345
s = "Hello World"
print(s.upper())   # HELLO WORLD
print(s.lower())   # hello world
# 실용 예: 대소문자 무시하고 비교
print("YES".lower() == "yes")  # True

공백 제거 — strip()

사용자 입력을 처리할 때 앞뒤 공백이 끼어있는 경우가 많습니다.

python
1234
s = "  파이썬  "
print(s.strip())   # "파이썬"   — 앞뒤 공백 모두
print(s.lstrip())  # "파이썬  " — 왼쪽만
print(s.rstrip())  # "  파이썬" — 오른쪽만

교체 — replace()

python
1234567
sentence = "나는 사과를 좋아한다. 사과는 맛있다."

# replace(찾을 문자열, 바꿀 문자열)
print(sentence.replace("사과", "수박"))
# 나는 수박를 좋아한다. 수박는 맛있다.

# 세 번째 인자: 최대 교체 횟수
print(sentence.replace("사과", "수박", 1))
# 나는 수박를 좋아한다. 사과는 맛있다. ← 첫 번째만

분리 — split()

python
1234567
sentence = "사과 바나나 딸기"
print(sentence.split())
# ['사과', '바나나', '딸기']

csv = "홍길동,20,서울"
print(csv.split(","))
# ['홍길동', '20', '서울']

검색 — find(), count()

python
123456
s = "banana"
# find() — 처음 발견된 인덱스. 없으면 -1
print(s.find("a"))    # 1
print(s.find("z"))    # -1 (없음)
# count() — 등장 횟수
print(s.count("a"))   # 3
print(s.count("na"))  # 2

시작/끝 확인 — startswith(), endswith()

python
1234
filename = "report_2024.pdf"
print(filename.startswith("report"))  # True
print(filename.endswith(".pdf"))       # True
print(filename.endswith(".xlsx"))      # False

합치기 — join()

split()의 반대. 리스트의 문자열들을 구분자로 이어 붙입니다.

python
12345
words = ["사과", "바나나", "딸기"]
# "구분자".join(리스트)
print(", ".join(words))   # 사과, 바나나, 딸기
print(" - ".join(words))  # 사과 - 바나나 - 딸기
print("".join(words))     # 사과바나나딸기

6.in 연산자 — 포함 여부

in은 특정 문자열이 포함되어 있는지 확인합니다. 결과는 항상 True 또는 False입니다.

python
12345
email = "user@example.com"
print("@" in email)      # True  — 이메일 형식 검사에 활용
print(".com" in email)   # True
print("gmail" in email)  # False
print("@" not in email)  # False — not in 도 가능

7.문자열은 불변(immutable)이다

파이썬 문자열은 한 번 만들어지면 특정 위치의 문자를 직접 바꿀 수 없습니다. 이것을 불변(immutable)이라고 합니다.

python — 오류 나는 코드
12
word = "hello"
word[0] = "H"   # TypeError!
오류 메시지
TypeError: 'str' object does not support item assignment
✅ 올바른 방법 — 새 문자열 만들기

문자열을 수정하려면 replace()를 쓰거나, 슬라이싱으로 새 문자열을 조합해 변수에 다시 대입하세요.

python — 올바른 방법
12345
word = "hello"
# 방법 1: replace()
word = word.replace("h", "H", 1)
# 방법 2: 슬라이싱 조합
word = "H" + word[1:]   # "H" + "ello" = "Hello"
print(word)  # Hello

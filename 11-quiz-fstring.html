Q1.

name = "Alice"일 때, f-string을 사용해 안녕하세요, Alice!를 출력하세요.

출력:안녕하세요, Alice!
💡 힌트
f"문자열 {변수} 문자열" 형태로 작성하세요.
✅ 정답
name = "Alice"
print(f"안녕하세요, {name}!")
# f 접두사와 { } 안에 변수명을 넣으면 자동 삽입됩니다.
Q2.

age = 25일 때, f-string으로 나는 25살입니다.를 출력하세요.

출력:나는 25살입니다.
💡 힌트
정수도 f-string 안에서 자동으로 문자열로 변환됩니다.
✅ 정답
age = 25
print(f"나는 {age}살입니다.")
# str(age)로 변환할 필요가 없습니다.
Q3.

city = "서울", pop = 9700000일 때, 서울의 인구는 9700000명입니다.를 출력하세요.

출력:서울의 인구는 9700000명입니다.
💡 힌트
하나의 f-string 안에 { } 두 개를 쓸 수 있습니다.
✅ 정답
city = "서울"
pop  = 9700000
print(f"{city}의 인구는 {pop}명입니다.")

{ } 안에 a + b처럼 계산식을 직접 쓸 수 있습니다.

Q11.

Q50.

🏆 종합 도전! 아래 변수들을 사용해 쇼핑 영수증을 출력하세요.

store    = "슈퍼파이썬 마켓"
item1    = "아메리카노"; p1 = 4500; q1 = 2
item2    = "크루아상";    p2 = 3200; q2 = 3
discount = 0.1
출력: ===슈퍼파이썬 마켓===
아메리카노 x2 9,000원
크루아상 x3 9,600원
소계: 18,600원
할인(10%): -1,860원
최종: 16,740원
💡 힌트
subtotal = p1*q1 + p2*q2
disc_amt = int(subtotal * discount)
total = subtotal - disc_amt
각 줄을 f-string과 :, 서식으로 출력하세요.
✅ 정답
store    = "슈퍼파이썬 마켓"
item1 = "아메리카노"; p1 = 4500; q1 = 2
item2 = "크루아상";    p2 = 3200; q2 = 3
discount = 0.1
subtotal = p1*q1 + p2*q2
disc_amt = int(subtotal * discount)
total    = subtotal - disc_amt
print(f"==={store}===")
print(f"{item1:<6}  x{q1}  {p1*q1:,}원")
print(f"{item2:<6}  x{q2}  {p2*q2:,}원")
print(f"소계: {subtotal:,}원")
print(f"할인({discount:.0%}): -{disc_amt:,}원")
print(f"최종: {total:,}원")

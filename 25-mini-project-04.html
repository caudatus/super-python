2.게임 규칙 & 실행 예시

규칙

컴퓨터는 1~9 사이의 서로 다른 숫자 3개를 무작위로 선택합니다. 플레이어는 숫자 3개를 입력하고, 힌트를 받아 정답을 좁혀나갑니다.

판정의미예시 — 정답: 1 5 9
스트라이크 (S) 숫자와 자리가 모두 맞음 입력 1 5 3 → 2S 0B
볼 (B) 숫자는 맞지만 자리가 다름 입력 9 1 5 → 0S 3B
아웃 숫자가 하나도 포함되지 않음 입력 2 3 4 → 아웃!
3스트라이크 모든 자리 정확히 일치 → 게임 종료 입력 1 5 9 → 정답!
💡 입력 형식

숫자 3개를 공백으로 구분해서 입력합니다: 1 5 9
중복 숫자(예: 1 1 5)나 3개가 아닌 입력은 다시 요청합니다.

4.단계별 구현 가이드

한 단계씩 완성하고 직접 실행해서 결과를 확인한 뒤 다음으로 넘어가세요. 처음부터 완성본을 만들려다 막히면 아무것도 못 합니다.

STEP 1 정답 생성 & 시작 배너 출력 random.sample · print

컴퓨터가 사용할 정답 숫자 3개를 만들고 게임 시작 배너를 출력합니다. random.sample()은 주어진 범위에서 중복 없이 N개를 골라줍니다.

STEP 1 구현
import random

answer = random.sample(range(1, 10), 3)   # [3, 7, 2] 같은 무작위 리스트
tries = 0                                   # 시도 횟수

print("⚾ 숫자 야구 게임")
print("1~9에서 서로 다른 숫자 3개를 맞혀보세요!")
print("시도 횟수가 적을수록 높은 점수를 받습니다.")
print()

# 개발 중에는 이 줄의 주석을 해제해서 정답을 확인하며 테스트하세요
# print(f"[개발용] 정답: {answer}")
💡 random.sample vs random.randint

random.randint(1, 9)는 중복을 허용하므로 야구 게임에 맞지 않습니다. random.sample(범위, 개수)는 중복 없이 뽑아주므로 이 게임에 딱 맞습니다.

▶ 실행하면 배너가 출력됩니다. 정답 확인 주석을 해제하면 테스트할 때 편합니다.

STEP 2 입력 받기 & 유효성 검사 input().split() · isdigit() · continue

잘못된 입력이면 continue로 건너뛰어 다시 입력받습니다. 유효성 검사는 세 단계입니다 — 개수 → 숫자 여부 → 중복.

STEP 2 구현 — while 루프 안에 작성
while True:
    raw   = input("숫자 3개를 공백으로 입력하세요: ")
    parts = raw.split()        # "1 5 9"  →  ["1", "5", "9"]

    # 검증 1: 정확히 3개인지
    if len(parts) != 3:
        print("⚠ 숫자 3개를 공백으로 구분해서 입력해야 합니다. 다시 입력하세요.")
        continue

    # 검증 2: 1~9 사이 숫자인지
    valid = True
    for p in parts:
        if not p.isdigit() or not (1 <= int(p) <= 9):
            valid = False
            break
    if not valid:
        print("⚠ 1~9 사이의 숫자만 입력 가능합니다. 다시 입력하세요.")
        continue

    # 문자열 리스트 → 정수 리스트로 변환
    guess = [int(p) for p in parts]

    # 검증 3: 중복 없는지
    if len(set(guess)) != 3:
        print("⚠ 서로 다른 숫자 3개를 입력해야 합니다. 다시 입력하세요.")
        continue

    # 여기까지 왔다면 유효한 입력 → 시도 횟수 증가
    tries += 1
    # STEP 3 코드가 여기에 이어집니다
💡 set()으로 중복 검사하기

set()은 중복을 제거한 집합입니다. [1, 1, 5]를 set으로 바꾸면 {1, 5}가 되어 길이가 2 — 원래 리스트 길이(3)와 다르므로 중복이 있습니다.

⚠ 리스트 컴프리헨션 미리보기

[int(p) for p in parts]는 "parts의 각 요소를 int로 바꾼 리스트"라는 뜻입니다. 이 문법은 실버 과정에서 자세히 배웁니다. 지금은 문자열 리스트를 숫자 리스트로 바꾸는 방법으로만 이해하고 그대로 사용하세요.

▶ abc, 1 1 5, 5 9 등을 입력해서 오류 메시지가 제대로 나오는지 확인하세요.

STEP 3 스트라이크 & 볼 판정 로직 for i in range(3) · in 연산자 · 카운터

게임의 핵심입니다. 스트라이크는 자리도 숫자도 같은 경우, 볼은 숫자는 같지만 자리가 다른 경우입니다.

스트라이크를 먼저 셉니다. 그다음, 정답에 포함되어 있지만 자리가 다른 숫자의 수가 볼입니다.

STEP 3 구현 — tries += 1 아래에 이어서
    # 스트라이크: 자리와 숫자 모두 일치
    strike = 0
    for i in range(3):
        if guess[i] == answer[i]:
            strike += 1

    # 볼: 정답에 있지만 자리가 다름
    ball = 0
    for i in range(3):
        if guess[i] in answer and guess[i] != answer[i]:
            ball += 1

    # 결과 출력
    if strike == 3:
        print(f"🎉 정답! {tries}번 만에 맞혔습니다!")
        break                           # 게임 종료!
    elif strike == 0 and ball == 0:
        print("→ 아웃!")
    else:
        print(f"→ {strike}S {ball}B")

▶ 정답을 미리 출력해두고, 다양한 입력을 넣어 S/B 수가 올바른지 확인하세요.

STEP 4 점수 계산 & 최종 출력 if / elif / else · f-string

게임 종료(break) 후, while 루프 바깥에 점수 계산 코드를 작성합니다. 들여쓰기에 주의하세요.

STEP 4 구현 — while 루프 완전히 바깥
# while True: 블록이 끝난 뒤 (들여쓰기 없음)

if tries == 1:
    score = 10
elif tries <= 3:
    score = 8
elif tries <= 6:
    score = 6
elif tries <= 10:
    score = 4
else:
    score = 2

print(f"⭐ 점수: {score}점  (10점 만점)")

▶ 여러 번 실행해서 시도 횟수에 따라 점수가 달라지는지 확인하세요.

5.막혔을 때 — 단계별 힌트

충분히 혼자 시도해본 뒤에 열어보세요.

💡 힌트 1 — input().split()이 뭔가요? ▼

split()은 공백을 기준으로 문자열을 나눠서 리스트로 반환합니다.

예시
raw   = "1 5 9"
parts = raw.split()
print(parts)     # ["1", "5", "9"]  ← 문자열 리스트!
print(len(parts)) # 3

결과가 문자열 리스트임에 주의하세요. 숫자 비교를 하려면 int(p)로 변환해야 합니다.

💡 힌트 2 — 스트라이크와 볼의 차이가 헷갈려요 ▼

예시로 직접 추적해보겠습니다. 정답이 [1, 5, 9], 입력이 [9, 1, 5]라면:

추적 예시
# 스트라이크: 같은 인덱스, 같은 값
# i=0: guess[0]=9, answer[0]=1 → 다름
# i=1: guess[1]=1, answer[1]=5 → 다름
# i=2: guess[2]=5, answer[2]=9 → 다름  → strike = 0

# 볼: answer에 있지만 같은 자리가 아님
# i=0: 9 in [1,5,9]=True, 9≠answer[0](1)=True → 볼!
# i=1: 1 in [1,5,9]=True, 1≠answer[1](5)=True → 볼!
# i=2: 5 in [1,5,9]=True, 5≠answer[2](9)=True → 볼!
# → ball = 3  →  결과: 0S 3B ✅

💡 힌트 4 — isdigit()이 뭔가요? ▼

문자열이 숫자로만 이루어져 있으면 True, 아니면 False를 반환합니다.

예시
print("5".isdigit())    # True
print("abc".isdigit())  # False
print("1.5".isdigit())  # False  (소수점 포함 → False)

isdigit()으로 먼저 확인한 뒤 int()로 변환해야 안전합니다. 순서를 바꾸면 ValueError가 발생합니다.

6.전체 정답 코드

먼저 스스로 완성해보고, 아래 정답과 비교해보세요. 결과가 같다면 코드가 달라도 모두 정답입니다.

import random

# ── 초기화 ─────────────────────────────────────
answer = random.sample(range(1, 10), 3)
tries = 0

print("⚾ 숫자 야구 게임")
print("1~9에서 서로 다른 숫자 3개를 맞혀보세요!")
print("시도 횟수가 적을수록 높은 점수를 받습니다.")
print()

# ── 게임 루프 ──────────────────────────────────
while True:
    raw   = input("숫자 3개를 공백으로 입력하세요: ")
    parts = raw.split()

    # 검증 1: 개수
    if len(parts) != 3:
        print("⚠ 숫자 3개를 공백으로 구분해서 입력해야 합니다. 다시 입력하세요.")
        continue

    # 검증 2: 1~9 사이 숫자
    valid = True
    for p in parts:
        if not p.isdigit() or not (1 <= int(p) <= 9):
            valid = False
            break
    if not valid:
        print("⚠ 1~9 사이의 숫자만 입력 가능합니다. 다시 입력하세요.")
        continue

    # 검증 3: 중복 없는지
    guess = [int(p) for p in parts]
    if len(set(guess)) != 3:
        print("⚠ 서로 다른 숫자 3개를 입력해야 합니다. 다시 입력하세요.")
        continue

    tries += 1

    # ── 판정 ───────────────────────────────────
    strike = 0
    for i in range(3):
        if guess[i] == answer[i]:
            strike += 1

    ball = 0
    for i in range(3):
        if guess[i] in answer and guess[i] != answer[i]:
            ball += 1

    # ── 결과 출력 ──────────────────────────────
    if strike == 3:
        print(f"🎉 정답! {tries}번 만에 맞혔습니다!")
        break
    elif strike == 0 and ball == 0:
        print("→ 아웃!")
    else:
        print(f"→ {strike}S {ball}B")

# ── 점수 계산 ──────────────────────────────────
if tries == 1:
    score = 10
elif tries <= 3:
    score = 8
elif tries <= 6:
    score = 6
elif tries <= 10:
    score = 4
else:
    score = 2

print(f"⭐ 점수: {score}점  (10점 만점)")

  • 추측 기록 — 매 시도마다 지금까지의 모든 입력과 결과를 리스트에 저장해서 출력

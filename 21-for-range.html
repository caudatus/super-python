for문과 range()

같은 코드를 반복하는 가장 파이썬다운 방법. 리스트의 모든 항목을 순서대로 처리하거나, 숫자를 세면서 원하는 횟수만큼 반복하는 법을 배웁니다.

▸ 이 챕터를 마치면
  • for 문의 구조와 동작 원리를 이해할 수 있다
  • range()로 원하는 숫자 범위를 만들어 반복할 수 있다
  • 리스트를 for 문으로 순회(iterate)할 수 있다
  • for 문 안에서 if를 조합해 조건부 반복을 처리할 수 있다
  • 반복문을 중첩해 2차원 패턴을 출력할 수 있다
  • enumerate()로 인덱스와 값을 동시에 다룰 수 있다

1.반복이란?

지금까지 작성한 코드는 위에서 아래로 한 번 실행되고 끝났습니다. 그런데 실제 프로그램에서는 같은 작업을 여러 번 반복해야 하는 상황이 자주 옵니다.

학생 5명의 이름을 출력한다고 가정해봅시다. 반복문 없이 짜면 이렇게 됩니다:

python
12345
print("김민준")
print("이서연")
print("박지후")
print("최수아")
print("정하윤")

학생이 5명이면 괜찮지만, 100명이라면? 1000명이라면? 반복문을 쓰면 몇 줄로 해결됩니다.

python
123
students = ["김민준", "이서연", "박지후", "최수아", "정하윤"]
for name in students:
    print(name)
실행 결과
김민준
이서연
박지후
최수아
정하윤

리스트 크기가 1000개로 늘어도 코드는 똑같이 3줄입니다. 이것이 반복문의 힘입니다.

2.for 문 기본 구조

for는 "~의 각 항목에 대해"라는 뜻입니다. 리스트, 문자열 등 순서 있는 데이터를 하나씩 꺼내 블록을 반복 실행합니다.

for 변수 in 순회 대상 : ← 콜론(:) 필수!
실행할 코드 ← 반드시 들여쓰기(스페이스 4칸)
실행할 코드 2 ← 같은 들여쓰기 = 같은 블록
다음 코드 ← 들여쓰기 없음 = for 블록 밖, 반복 끝난 뒤 실행

변수에는 원하는 이름을 쓰면 됩니다. 관습적으로 리스트 이름의 단수형을 씁니다 — students라면 student, scores라면 score처럼요.

매 반복마다 변수에 순회 대상의 다음 항목이 자동으로 들어오고, 블록이 실행됩니다. 마지막 항목까지 처리하면 반복이 끝납니다.

반복이 일어나는 과정 — 단계별 추적

반복 횟수name 에 들어오는 값print(name) 출력
1번째"김민준"김민준
2번째"이서연"이서연
3번째"박지후"박지후
4번째"최수아"최수아
5번째"정하윤"정하윤
→ 더 이상 항목 없음 → 반복 종료 → for 블록 다음 코드 실행

3.range() — 숫자 범위 만들기

리스트 없이 숫자를 세면서 반복하고 싶을 때는 range()를 씁니다. range()는 숫자 범위를 만들어주는 함수입니다.

형태 1 — range(끝)

0부터 시작해서 끝 바로 전 숫자까지 생성합니다. "끝"은 포함되지 않습니다.

range(5) → 0 1 2 3 4 5 ← 5는 포함되지 않음 (0 이상 5 미만)
python
12
for i in range(5):
    print(i)
실행 결과
0
1
2
3
4

형태 2 — range(시작, 끝)

시작부터 끝 바로 전까지 생성합니다.

range(1, 6) → 0 1 2 3 4 5 6 ← 1 이상 6 미만 (0과 6은 포함 안 됨)
python
123
# 1부터 5까지 더하기
for i in range(1, 6):
    print(f"{i}번째 반복")
실행 결과
1번째 반복
2번째 반복
3번째 반복
4번째 반복
5번째 반복

형태 3 — range(시작, 끝, 증감)

세 번째 인수는 증감값(step)입니다. 생략하면 기본값은 1입니다.

python
12345678
# 짝수만 출력 (0, 2, 4, 6, 8)
for i in range(0, 10, 2):
    print(i, end=" ")

print()  # 줄바꿈

# 카운트다운 (5, 4, 3, 2, 1)
for i in range(5, 0, -1):
    print(i, end=" ")
실행 결과
0 2 4 6 8
5 4 3 2 1
💡 range() 세 가지 형태 한눈에 보기
형태생성되는 숫자예시
range(끝)0 이상 끝 미만range(5) → 0 1 2 3 4
range(시작, 끝)시작 이상 끝 미만range(2, 6) → 2 3 4 5
range(시작, 끝, 증감)시작부터 증감씩range(1, 10, 3) → 1 4 7
⚠ range()의 끝 숫자는 포함되지 않습니다

이 점이 초보자가 가장 많이 실수하는 부분입니다. "1부터 10까지"를 원한다면 range(1, 11)처럼 끝에 +1을 해야 합니다.

4.리스트 순회하기

리스트를 for 문에 직접 넣으면 항목을 하나씩 꺼내 처리할 수 있습니다. 이를 순회(iteration)라고 합니다.

예제 1 — 리스트 항목 전부 출력

python
123
fruits = ["사과", "바나나", "체리", "망고"]
for fruit in fruits:
    print(f"🍎 {fruit}")
실행 결과
🍎 사과
🍎 바나나
🍎 체리
🍎 망고

예제 2 — 합계 누적 계산

반복문 밖에서 합계 변수를 초기화하고, 반복할 때마다 더해나갑니다. total += score는 total = total + score와 같습니다.

python
123456
scores = [85, 92, 78, 90, 88]
total = 0             # 합계 초기화

for score in scores:
    total += score    # 반복마다 더하기

print(f"합계: {total}, 평균: {total / len(scores):.1f}")
실행 결과
합계: 433, 평균: 86.6

예제 3 — 문자열 순회

문자열도 for로 순회할 수 있습니다. 각 글자가 하나씩 나옵니다.

python
12
for char in "Python":
    print(char, end="-")
실행 결과
P-y-t-h-o-n-

5.for + if — 조건부 반복

for 블록 안에 if를 넣으면 "반복하면서 특정 조건을 만족하는 항목만 처리"할 수 있습니다. 반복문과 조건문이 결합되는 순간 코드의 표현력이 크게 늘어납니다.

예제 1 — 짝수만 출력

python
123
for n in range(1, 11):
    if n % 2 == 0:
        print(n, end=" ")
실행 결과
2 4 6 8 10

예제 2 — 점수 등급 일괄 처리

python
12345678910
scores = [95, 72, 88, 61, 100, 55]

for score in scores:
    if score >= 90:
        grade = "A"
    elif score >= 80:
        grade = "B"
    elif score >= 70:
        grade = "C"
    else:
        grade = "F"
    print(f"{score}점 → {grade}등급")
실행 결과
95점 → A등급
72점 → C등급
88점 → B등급
61점 → F등급
100점 → A등급
55점 → F등급
💡 들여쓰기 레벨에 주의하세요

for 블록 안의 if는 4칸 더 들여쓰고, if 블록 안의 코드는 또 4칸 더 들여씁니다.

for score in scores: ← 들여쓰기 0 (for)
    if score >= 90: ← 들여쓰기 4 (for 블록 안의 if)
        grade = "A" ← 들여쓰기 8 (if 블록)
    print(...) ← 들여쓰기 4 (for 블록, if 블록 밖)

6.enumerate() — 인덱스와 값을 동시에

리스트를 순회할 때 값만이 아니라 몇 번째 항목인지(인덱스)도 함께 필요한 경우가 있습니다. enumerate()를 쓰면 인덱스와 값을 동시에 받을 수 있습니다.

python
1234
fruits = ["사과", "바나나", "체리"]

for i, fruit in enumerate(fruits):
    print(f"{i}번: {fruit}")
실행 결과
0번: 사과
1번: 바나나
2번: 체리

1번부터 시작하고 싶다면 enumerate(fruits, 1)처럼 시작값을 두 번째 인수로 넘깁니다.

python
123
for i, fruit in enumerate(fruits, 1):   # 1부터 시작
    print(f"{i}번: {fruit}")
실행 결과
1번: 사과
2번: 바나나
3번: 체리
💡 enumerate() 없이 같은 결과를 내는 방법

for i in range(len(fruits)):로도 인덱스를 얻을 수 있습니다. 하지만 enumerate()가 더 간결하고 파이썬답습니다.

7.중첩 for — for 안에 for

for 블록 안에 또 다른 for를 넣을 수 있습니다. 바깥 반복이 한 번 돌 때마다 안쪽 반복이 처음부터 끝까지 전부 실행됩니다. 구구단, 좌표 탐색, 2차원 데이터 처리 등에 씁니다.

예제 1 — 별 삼각형 출력

python
123
for row in range(1, 6):         # row: 1~5
    for col in range(row):        # col: 0 ~ row-1
        print("★", end="")
    print()                         # 한 줄 끝나면 줄바꿈
실행 결과
★
★★
★★★
★★★★
★★★★★

예제 2 — 구구단 (3단, 4단)

python
123
for dan in range(3, 5):          # 3단, 4단
    for i in range(1, 10):
        print(f"{dan} × {i} = {dan*i}")
    print()                          # 단과 단 사이 빈 줄
실행 결과 (일부)
3 × 1 = 3
3 × 2 = 6
...
3 × 9 = 27

4 × 1 = 4
...
💡 반복 횟수 계산

바깥 반복이 N번, 안쪽 반복이 M번이면 안쪽 블록은 총 N × M번 실행됩니다. 중첩 단계가 깊어질수록 실행 횟수가 기하급수적으로 늘어나므로 필요한 만큼만 중첩하세요.

8.자주 하는 실수

실수 1 — range() 끝 숫자를 포함하려다 하나 부족

논리 오류 (에러는 안 나지만 결과가 틀림)
# "1부터 10까지" 출력하려 했는데
for i in range(1, 10):     ← 1~9 만 출력됨
    print(i)

# 올바른 코드
for i in range(1, 11):     ← 1~10 출력

실수 2 — 콜론(:) 빠뜨리기

SyntaxError
for i in range(5)          ← 콜론이 없으면
    print(i)
SyntaxError: expected ':'

실수 3 — 반복 변수를 루프 밖에서 수정

논리 오류
for i in range(5):
    i = i + 10          ← 이 변경은 다음 반복에서 무시됨
    print(i)            # 10, 11, 12, 13, 14 출력 — 헷갈릴 수 있음

# range()의 i 값은 매 반복마다 자동으로 덮어써집니다
# 반복 변수 자체를 수정하는 것은 권장하지 않습니다

실수 4 — 누적 변수를 for 안에서 초기화

논리 오류
for score in scores:
    total = 0           ← 매 반복마다 0으로 리셋됨!
    total += score

# 올바른 코드: 누적 변수는 for 밖에서 초기화
total = 0               ← for 바깥에서 한 번만
for score in scores:
    total += score

9.실전 예제 모음

예제 1 — 리스트에서 최댓값 직접 찾기

123456
numbers = [34, 17, 92, 45, 63, 8]
max_val = numbers[0]     # 첫 번째 값으로 초기화

for n in numbers:

print(f"최댓값: {max_val}")

123456
products = ["노트북", "이어폰", "마우스"]
ratings  = [4, 5, 3]

for i, product in enumerate(products):

    print(f"{product:{6}} {stars}")

예제 3 — FizzBuzz

프로그래밍 면접에서 단골로 등장하는 문제입니다. 1부터 20까지 출력하되, 3의 배수면 "Fizz", 5의 배수면 "Buzz", 15의 배수면 "FizzBuzz"를 출력합니다.

python
12345678
for i in range(1, 21):
    if i % 15 == 0:         # 15의 배수 (3과 5 모두 해당) → 먼저 검사!
        print("FizzBuzz")
    elif i % 3 == 0:
        print("Fizz")
    elif i % 5 == 0:
        print("Buzz")
    else:
        print(i)
실행 결과 (일부)
1
2
Fizz
4
Buzz
Fizz
7
...
FizzBuzz
...
📌 왜 15의 배수를 먼저 검사하나요?

15는 3과 5의 공배수입니다. elif는 앞 조건이 False일 때만 실행되므로, 15의 배수 조건을 나중에 두면 이미 3의 배수 조건에 걸려 "Fizz"만 출력됩니다. 더 구체적인(또는 좁은) 조건을 항상 앞에 두세요.

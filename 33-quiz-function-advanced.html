아래 함수 정의에서 기본값 매개변수를 올바르게 사용한 것은 무엇인가요? 올바른 코드를 작성하세요.

# 함수 이름: power
# 매개변수: base (기본값 없음), exp (기본값 2)
# 동작: base ** exp 를 반환
💡 힌트

Q14.

기본값 매개변수와 *args를 함께 쓰는 올바른 함수 정의는 무엇인가요? 아래 함수를 완성하세요. make_tag("안녕", "반가워")는 [안녕] [반가워]를 출력하고, make_tag("안녕", "반가워", sep=" | ")는 안녕 | 반가워를 출력해야 합니다.

💡 힌트
*args 뒤에는 키워드 전용 인수를 놓을 수 있습니다.
str.join()으로 단어들을 구분자로 연결해보세요.
기본 sep을 "[단어]" 형식이 되도록 설정해야 합니다.
✅ 정답
def make_tag(*words, sep=None):
    if sep is None:
        print(" ".join(f"[{w}]" for w in words))
    else:
        print(sep.join(words))

# *args 뒤에 오는 매개변수는 키워드 전용이 됩니다.
# make_tag("안녕", "반가워")       → [안녕] [반가워]
# make_tag("안녕", "반가워", sep=" | ") → 안녕 | 반가워

Q23.

**kwargs를 활용해 상품 주문 정보를 처리하는 함수 order()를 작성하세요. 필수 키는 item과 qty이며, 나머지는 선택 사항입니다. 아래 조건을 만족해야 합니다.

# order(item="사과", qty=3)         → "사과 3개 주문 완료"
# order(item="바나나", qty=2, memo="빨리")  → "바나나 2개 주문 완료 [메모: 빨리]"
💡 힌트
kwargs.get("memo")로 선택 키가 있는지 확인합니다.
None이 아닐 때만 메모 부분을 출력합니다.
✅ 정답
def order(**kwargs):
    item = kwargs["item"]
    qty  = kwargs["qty"]
    memo = kwargs.get("memo")
    msg  = f"{item} {qty}개 주문 완료"
    if memo:
        msg += f" [메모: {memo}]"
    print(msg)

# .get()은 키가 없으면 None을 반환합니다.
# if memo: 는 memo가 None이거나 빈 문자열이면 False가 됩니다.

아래 함수를 이용해 점수 목록에서 평균을 구하고, 90점 이상이면 "우수", 70점 이상이면 "통과", 그 미만이면 "미달"을 함께 반환하는 함수 evaluate()를 완성하세요.

# evaluate(95, 88, 92) → (91.67, "우수") 형태로 반환
# 평균은 소수점 2자리까지 반올림
💡 힌트
avg = round(sum(scores) / len(scores), 2)
그런 다음 if/elif/else로 등급을 결정합니다.
✅ 정답
def evaluate(*scores):
    avg = round(sum(scores) / len(scores), 2)
    if avg >= 90:
        grade = "우수"
    elif avg >= 70:
        grade = "통과"
    else:
        grade = "미달"
    return avg, grade

# evaluate(95, 88, 92) → (91.67, "우수")

쇼핑몰 결제 함수를 설계하세요. checkout() 함수는 상품 이름(*items), 할인율(discount=0), 포인트 사용 여부(use_point=False)를 받습니다. 총 금액을 계산하되 각 상품은 1000원으로 가정하고, 할인 후 포인트 사용 시 500원을 추가 차감한 최종 금액을 반환해야 합니다.

# checkout("사과", "빵")           → 2000
# checkout("사과", "빵", discount=0.1)     → 1800.0
# checkout("사과", "빵", discount=0.1, use_point=True) → 1300.0
💡 힌트
total = len(items) * 1000
after_discount = total * (1 - discount)
if use_point: after_discount -= 500
✅ 정답
def checkout(*items, discount=0, use_point=False):
    total = len(items) * 1000
    final = total * (1 - discount)
    if use_point:
        final -= 500
    return final

# *items 뒤의 매개변수는 모두 키워드 전용으로 동작합니다.
# checkout("사과","빵")              → 2000 * (1-0)     = 2000
# checkout("사과","빵", discount=0.1) → 2000 * 0.9      = 1800.0
# use_point=True 추가               → 1800.0 - 500    = 1300.0
Q30.

지금까지 배운 함수 심화 개념을 모두 활용하는 종합 문제입니다. report() 함수를 작성하세요. 이 함수는 제목(title, 기본값 "리포트"), 항목들(*entries), 메타데이터(**meta)를 받아 아래 형식으로 출력합니다.

# report("월간 보고서", "항목A", "항목B", author="김철수", date="2026-03")
# 출력:
# ===== 월간 보고서 =====
# 1. 항목A
# 2. 항목B
# --- 메타 ---
# author: 김철수
# date: 2026-03
💡 힌트
print(f"===== {title} =====") 로 제목을 출력하고
enumerate(entries, 1)로 번호와 함께 항목을 출력합니다.
meta.items()로 메타데이터를 출력합니다.
✅ 정답
def report(title="리포트", *entries, **meta):
    print(f"===== {title} =====")
    for i, entry in enumerate(entries, 1):
        print(f"{i}. {entry}")
    if meta:
        print("--- 메타 ---")
        for k, v in meta.items():
            print(f"{k}: {v}")

# 일반 매개변수 기본값(title)은 *args 앞에 올 수 있습니다.
# enumerate(entries, 1)은 1번부터 번호를 매겨줍니다.
# if meta: 로 메타데이터가 있을 때만 출력합니다.

텍스트 RPG 전투 시뮬레이터

함수 기초부터 심화, 그리고 모듈까지 — 지금까지 배운 함수 섹션의 모든 개념을 한 프로그램에 녹여냅니다. 랜덤 전투가 실제로 동작하는 나만의 RPG를 만들어보세요.

▸ 이 프로젝트를 마치면
  • 함수를 역할별로 분리하여 프로그램을 구조화할 수 있다
  • 기본값 매개변수와 *args를 실전 코드에 적용할 수 있다
  • random 모듈로 불확실성(크리티컬, 회피)을 구현할 수 있다
  • math 모듈로 수치 계산(데미지 공식)을 처리할 수 있다
  • while과 함수가 어떻게 협력하는지 이해할 수 있다
  • 완성된 게임 한 판을 처음부터 끝까지 직접 실행할 수 있다

📋 구현할 함수 목록

① create_character(name, hp, attack, defense) — 캐릭터 정보를 담은 리스트 반환
② print_status(*characters) — *args로 여러 캐릭터 상태를 한 번에 출력

⚠️ 게임 규칙

• 기본 데미지 = attack - defense, 최소 1 이상
• 크리티컬(20% 확률): 데미지 × 2, "✨ 크리티컬!" 출력
• 회피(10% 확률): 데미지 0, "회피했다!" 출력
• 캐릭터는 [이름, HP, 공격력, 방어력] 형태의 리스트로 관리
• HP가 0 이하가 되면 해당 캐릭터 패배

STEP 3 데미지 계산 함수 (기본값 매개변수) ▶

calculate_damage(attack, defense, crit_rate=0.2) 함수를 만드세요.

반환값은 (최종 데미지, 크리티컬 여부, 회피 여부) 세 가지를 담은 튜플입니다.

로직: ① 회피 판정 (10% 확률, random.random() < 0.1) → ② 크리티컬 판정 (random.random() < crit_rate) → ③ 기본 데미지 = attack - defense, 최소 1 → ④ 크리티컬이면 × 2, math.floor()로 정수화

import random
import math

def calculate_damage(attack, defense, crit_rate=0.2):
    # 회피 먼저 판정
    if random.random() < 0.1:
        return (0, False, True)   # (데미지, 크리티컬, 회피)
    
    base = max(1, attack - defense)
    is_crit = random.random() < crit_rate
    damage = math.floor(base * 2) if is_crit else base
    return (damage, is_crit, False)
import random
import math

def calculate_damage(attack, defense, crit_rate=0.2):
    # ① 회피 판정 (10%)
    if random.random() < 0.1:
        return (0, False, True)

    # ② 기본 데미지 (최소 1)
    base = max(1, attack - defense)

    # ③ 크리티컬 판정
    is_crit = random.random() < crit_rate
    damage  = math.floor(base * 2) if is_crit else base

    return (damage, is_crit, False)

# 세 값을 한 번에 받는 언패킹
dmg, crit, evade = calculate_damage(20, 5)
print(dmg, crit, evade)

STEP 5 전투 루프 함수 ▶

battle(hero, monster) 함수를 만드세요. while True 루프 안에서 라운드를 진행하고, 어느 한 쪽 HP가 0 이하가 되면 승패를 출력하고 break합니다.

순서: ① 라운드 번호 출력 → ② 영웅 공격 → ③ 몬스터 HP 확인(0 이하면 영웅 승리) → ④ 몬스터 공격 → ⑤ 영웅 HP 확인(0 이하면 몬스터 승리) → ⑥ 현재 HP 상태 출력

def battle(hero, monster):
    round_num = 1
    while True:
        print(f"\n--- {round_num} 라운드 ---")
        attack_action(hero, monster)
        if monster[1] <= 0:
            print(f"\n{monster[0]} 이 쓰러졌다!")
            print(f"🏆  {hero[0]} 승리!  ({round_num} 라운드)")
            break
        attack_action(monster, hero)
        if hero[1] <= 0:
            print(f"\n{hero[0]} 이 쓰러졌다!")
            print(f"💀  {monster[0]} 승리!  ({round_num} 라운드)")
            break
        print(f"[{hero[0]} HP: {hero[1]}]  [{monster[0]} HP: {monster[1]}]")

  • ③ 몬스터를 random.choice()로 여러 종류 중 하나를 랜덤으로 뽑아 전투를 시작하게 만들어보세요
  • ④ 전투가 끝난 뒤 라운드 수와 승패를 실제 파일에 저장해보세요 (파일 입출력은 42번 챕터에서 배웁니다)

함수 기초

같은 코드를 반복해서 쓰는 대신, 이름을 붙여 언제든 다시 꺼내 쓸 수 있는 코드 묶음 — 함수를 만드는 법을 배웁니다.

이 챕터를 마치면
  • def로 함수를 정의하고 호출할 수 있다
  • 매개변수(parameter)와 인자(argument)의 차이를 설명할 수 있다
  • return으로 결과값을 돌려주는 함수를 만들 수 있다
  • 함수를 쓰면 코드가 왜 좋아지는지 이해한다
  • 지역 변수와 전역 변수의 차이를 안다

01함수란 무엇인가?

여러분이 매일 아침 커피를 만든다고 상상해보세요. "원두 갈기 → 물 끓이기 → 드리퍼에 붓기 → 기다리기"를 매번 처음부터 되풀이하는 것은 번거롭습니다. 그래서 커피 머신을 사죠. 버튼 하나로 전 과정이 실행됩니다.

프로그래밍의 함수(function)가 바로 그 커피 머신입니다. 여러 줄의 코드를 하나의 이름으로 묶어두면, 그 이름만 부르면 됩니다. 다시 짤 필요가 없습니다.

사실 우리는 지금까지 함수를 계속 써왔습니다. print(), input(), len(), int() — 이것들이 전부 파이썬이 미리 만들어둔 함수입니다. 이번 챕터에서는 직접 함수를 만드는 법을 배웁니다.

💡 함수가 없다면?

함수 없이 "세 명의 이름을 환영 메시지로 출력"하려면 같은 코드를 세 번 써야 합니다. 이름이 100명이면 100번입니다. 함수를 쓰면 딱 한 번만 정의하고 100번 호출합니다.

들여쓰기가 함수의 몸통을 결정합니다

함수 정의 줄 다음에 오는 들여쓰기된 코드가 함수의 몸통(body)입니다. 들여쓰기가 끝나면 함수도 끝납니다.

python — 들여쓰기 범위
123456
def my_func():
    print("함수 안")      # ← 들여쓰기 O → 함수 몸통
    print("여기도 안")    # ← 들여쓰기 O → 함수 몸통

print("함수 밖")          # ← 들여쓰기 X → 함수 바깥 코드
my_func()                # ← 호출

03매개변수 — 함수에 값 전달하기

위의 say_hello()는 항상 같은 문장을 출력합니다. 매개변수(parameter)를 쓰면 함수를 호출할 때마다 다른 값을 전달할 수 있습니다.

python
123456
def greet(name):             # name이 매개변수
    print(f"안녕하세요, {name}님!")

greet("철수")                 # "철수"가 인자
greet("영희")
greet("민준")
실행 결과
안녕하세요, 철수님!
안녕하세요, 영희님!
안녕하세요, 민준님!

매개변수 vs 인자 — 헷갈리는 용어 정리

용어영어위치설명
매개변수 parameter 함수 정의할 때 함수가 받을 값의 자리 이름. def greet(name)의 name
인자 argument 함수 호출할 때 실제로 전달하는 값. greet("철수")의 "철수"
💡 실무에서는 둘 다 "파라미터"라고 부르기도 합니다

현업 개발자들이 대화할 때는 매개변수와 인자를 엄격히 구분하지 않고 혼용하는 경우가 많습니다. 공식 문서를 읽거나 시험을 볼 때 정확히 구분할 수 있으면 됩니다.

매개변수가 여러 개인 경우

매개변수는 쉼표로 구분해 여러 개 받을 수 있습니다.

python
123456
def introduce(name, age, city):
    print(f"이름: {name}")
    print(f"나이: {age}세")
    print(f"사는 곳: {city}")

introduce("김파이썬", 20, "서울")   # 순서대로 매핑됨
실행 결과
이름: 김파이썬
나이: 20세
사는 곳: 서울
⚠ 인자 개수가 반드시 일치해야 합니다

매개변수가 3개인 함수에 인자를 2개만 넘기면 TypeError: missing 1 required positional argument 오류가 납니다. 개수를 정확히 맞춰야 합니다.

04return — 결과값 돌려주기

지금까지의 함수는 결과를 화면에 출력만 했습니다. 하지만 계산 결과를 변수에 저장하거나, 다른 계산에 쓰고 싶다면 return으로 값을 반환해야 합니다.

def add(a, b):
    return a + b    # a + b 값을 호출한 곳으로 돌려줌

result = add(3, 5)   # 반환값 8을 result에 저장
print(result)          # 8

print(add(10, 20))     # 30

print와 return의 차이

초보자가 가장 자주 혼동하는 부분입니다. 차이를 명확히 이해해야 합니다.

python — print vs return 비교
1234567891011121314
# ❌ print만 하는 함수 — 결과를 저장할 수 없음
def bad_add(a, b):
    print(a + b)     # 화면에는 보이지만 값이 사라짐

x = bad_add(3, 5)   # 8이 출력되지만...
print(x)              # None — 반환값이 없으면 None이 저장됨!

# ✅ return하는 함수 — 결과를 저장하고 활용 가능
def good_add(a, b):
    return a + b

x = good_add(3, 5)
print(x)               # 8 ✅
print(x * 2)           # 16 ✅ — 저장된 값을 활용!
실행 결과
8
None
8
16
💡 return은 함수를 즉시 종료합니다

return을 만나는 순간 함수 실행이 멈추고 값을 돌려줍니다. return 뒤에 있는 코드는 실행되지 않습니다.

또한 return만 단독으로 쓰면(값 없이) None을 반환하며 함수를 빠져나옵니다.

💡 docstring — 함수 설명 달기

예제 3 — 함수를 여러 번 조합하기

함수는 다른 함수 안에서 호출할 수 있습니다. 각 기능을 작은 함수로 만들어 조합하면 읽기 쉬운 코드가 됩니다.

python
12345678910111213141516
def get_total(scores):
    """리스트의 합계를 반환합니다."""
    total = 0
    for s in scores:
        total += s
    return total

def get_average(scores):
    """리스트의 평균을 반환합니다."""
    return get_total(scores) / len(scores)   # get_total()을 재활용!

my_scores = [88, 92, 75, 96, 81]
print(f"합계: {get_total(my_scores)}")
print(f"평균: {get_average(my_scores)}")
실행 결과
합계: 432
평균: 86.4

06지역 변수와 전역 변수

함수 안에서 만든 변수는 함수 밖에서 보이지 않습니다. 이런 변수를 지역 변수(local variable)라 합니다. 반대로 함수 바깥에서 만든 변수를 전역 변수(global variable)라 합니다.

python
123456789101112
name = "전역 이름"   # 전역 변수 — 어디서든 읽기 가능

def my_func():
    greeting = "안녕!"   # 지역 변수 — 함수 안에서만 존재
    print(greeting)       # 안녕!
    print(name)            # 전역 변수는 함수 안에서도 읽기 가능

my_func()
print(name)              # 전역 이름
# print(greeting)       # ← NameError! 지역 변수는 밖에서 안 보임
실행 결과
안녕!
전역 이름
전역 이름
ℹ 왜 지역 변수를 쓸까요?

함수마다 독립된 변수 공간을 가지면, 함수 A에서 쓴 변수 이름이 함수 B의 변수와 충돌하지 않습니다. 100개의 함수가 모두 i라는 변수를 써도 서로 영향을 주지 않습니다. 이것이 함수의 큰 장점 중 하나입니다.

⚠ global 키워드는 되도록 쓰지 마세요

global 변수명을 쓰면 함수 안에서 전역 변수를 수정할 수 있습니다. 하지만 이렇게 하면 코드 흐름이 복잡해져서 버그를 찾기 어려워집니다. 브론즈 과정에서는 return으로 값을 돌려주는 방식을 사용하세요.

▸ 핵심 요약
def 함수명(): 함수를 정의합니다. 콜론(:) 뒤 들여쓰기된 코드가 함수 몸통입니다. 함수명() 함수를 호출합니다. 괄호를 붙여야 실행됩니다. 매개변수 함수 정의 시 괄호 안의 변수 이름. 전달받을 값의 자리입니다. 인자 함수 호출 시 실제로 전달하는 값입니다. return 함수를 종료하고 값을 돌려줍니다. 없으면 None을 반환합니다. 지역 변수 함수 안에서만 살아있는 변수. 함수 밖에서는 접근 불가합니다.

단계별 구현 가이드

전체를 한 번에 만들려 하지 마세요. 5단계로 나눠서 한 단계씩 완성하고, 각 단계가 끝날 때마다 실행해서 확인하는 습관을 들이세요. 각 단계마다 힌트와 정답을 숨겨두었습니다 — 정말 막힐 때만 확인하세요.

01
데이터 구조 설계 & load / save 함수
프로그램의 뼈대를 세웁니다. 데이터가 어디에 어떤 형태로 저장되는지 먼저 정의합니다.
딕셔너리 리스트 파일 입출력 예외처리

거래 내역 하나를 딕셔너리로 표현하고, 전체를 리스트로 관리합니다. load_data()와 save_data()를 먼저 완성해야 나머지 모든 기능에서 데이터를 유지할 수 있습니다.

✏ 구현할 것

  • ①transactions = [] — 빈 리스트로 시작
  • ②load_data() — CSV 파일을 읽어 딕셔너리 리스트를 반환. 파일 없으면 빈 리스트 반환
  • ③save_data(transactions) — 리스트를 CSV 형식으로 파일에 저장. 헤더 포함

✔ 이 단계 완성 확인

테스트
transactions = load_data() print(transactions) # 파일 없으면 [] transactions.append({"type":"수입","category":"급여","amount":100000,"memo":"테스트"}) save_data(transactions) # budget.csv 파일이 생성됨을 확인 transactions2 = load_data() print(transactions2) # [{"type":"수입",...}] 출력
HINT — load_data / save_data

load_data 힌트:
try/except FileNotFoundError로 파일이 없는 경우를 처리하세요. CSV를 읽을 때 첫 줄(헤더)은 next(f)로 건너뜁니다. 각 줄을 line.strip().split(",")으로 분리하면 4개 값이 나옵니다.

save_data 힌트:
"w" 모드로 열고 첫 줄에 헤더 "type,category,amount,memo\n"를 씁니다. 이후 리스트를 for로 돌며 각 딕셔너리의 값을 쉼표로 이어 씁니다.

천 단위 콤마 팁:
f"{숫자:,}" — 예: f"{2500000:,}" → "2,500,000"

ANSWER — 1단계
# ─── 1단계: 데이터 불러오기 / 저장 ─────────────── def load_data(): """CSV 파일을 읽어 거래 리스트 반환. 파일 없으면 [].""" transactions = [] try: with open("budget.csv", "r", encoding="utf-8") as f: next(f) # 헤더 줄 건너뜀 for line in f: line = line.strip() if not line: continue try: parts = line.split(",", 3) # 최대 4개로 분리 t_type, category, amount, memo = parts transactions.append({ "type": t_type, "category": category, "amount": int(amount), "memo": memo }) except (ValueError, IndexError): print(f"⚠ '{line}' 파싱 오류, 건너뜀") except FileNotFoundError: pass # 최초 실행 — 빈 리스트로 시작 return transactions def save_data(transactions): """거래 리스트를 CSV 파일로 저장.""" with open("budget.csv", "w", encoding="utf-8") as f: f.write("type,category,amount,memo\n") # 헤더 for t in transactions: f.write(f"{t['type']},{t['category']},{t['amount']},{t['memo']}\n") print(f"💾 budget.csv에 저장 완료 ({len(transactions)}건)")
02
수입 / 지출 추가 함수
사용자 입력을 받아 거래를 리스트에 추가합니다. 잘못된 입력은 예외처리로 안전하게 막습니다.
input() try / except 딕셔너리 리스트 append

add_transaction(transactions, t_type) 하나로 수입과 지출 모두 처리합니다. t_type이 "수입"이냐 "지출"이냐만 다릅니다.

  • ①카테고리, 금액, 메모를 input()으로 입력받기
  • ②금액이 숫자가 아니면 ValueError 처리 → 오류 메시지 출력 후 함수 종료
  • ③금액이 0 이하면 raise ValueError
  • ④딕셔너리 생성 후 transactions.append()
HINT — add_transaction

int(input("금액 > "))를 try 안에 넣으면 문자 입력 시 ValueError가 자동으로 발생합니다.

금액 범위 체크는 int() 변환 성공 후 바로 아래에서 if amount <= 0: raise ValueError("...")로 처리합니다.

성공 메시지에 천 단위 콤마: f"{amount:,}"

ANSWER — 2단계
# ─── 2단계: 수입 / 지출 추가 ───────────────────── def add_transaction(transactions, t_type): """수입 또는 지출 한 건을 입력받아 리스트에 추가.""" print(f"\n── {t_type} 추가 ──") try: category = input("카테고리 (예: 급여/식비/교통비) > ").strip() amount = int(input("금액 > ")) if amount <= 0: raise ValueError("금액은 1원 이상이어야 합니다.") memo = input("메모 (없으면 Enter) > ").strip() except ValueError as e: msg = str(e) print(f"❌ {msg if '이상' in msg else '숫자를 입력하세요.'}") return transactions.append({ "type": t_type, "category": category, "amount": amount, "memo": memo }) sign = "+" if t_type == "수입" else "-" print(f"✅ {t_type} 추가 완료: {category} {sign}{amount:,}원")
03
내역 보기 & 잔액 확인 함수
쌓인 데이터를 보기 좋게 출력합니다. for 루프와 조건문으로 수입/지출을 구분해 표시합니다.
for 반복문 f-string 조건문 숫자 포매팅

show_history()와 show_balance()를 구현합니다. 리스트가 비어있는 경우를 먼저 처리하는 것을 잊지 마세요.

  • ①show_history: 내역 없으면 "내역이 없습니다" 출력. 있으면 순서번호와 함께 한 줄씩 출력
  • ②수입은 +금액, 지출은 -금액으로 구분 표시
  • ③show_balance: type이 "수입"인 것의 amount 합, "지출"인 것의 amount 합, 그 차이(잔액) 출력
HINT — show_history / show_balance

잔액 계산: sum(t["amount"] for t in transactions if t["type"] == "수입") 패턴을 활용하세요.

번호 출력: for i, t in enumerate(transactions, start=1)를 쓰면 번호를 자동으로 매길 수 있습니다.

수입/지출 기호: sign = "+" if t["type"] == "수입" else "-"

ANSWER — 3단계
# ─── 3단계: 내역 보기 & 잔액 확인 ─────────────── def show_history(transactions): """전체 거래 내역을 출력.""" print("\n── 전체 거래 내역 ──") if not transactions: print(" 내역이 없습니다.") return print(f" {'번호':<4} {'구분':<5} {'카테고리':<10} {'금액':>12} {'메모'}") print(" " + "─" * 50) for i, t in enumerate(transactions, start=1): sign = "+" if t["type"] == "수입" else "-" amount = f"{sign}{t['amount']:,}" print(f" {i:<4} {t['type']:<5} {t['category']:<10} {amount:>12} {t['memo']}") def show_balance(transactions): """총 수입, 지출, 잔액 출력.""" income = sum(t["amount"] for t in transactions if t["type"] == "수입") expense = sum(t["amount"] for t in transactions if t["type"] == "지출") balance = income - expense print("\n── 잔액 확인 ──") print(f" 총 수입: {income:>12,} 원") print(f" 총 지출: {expense:>12,} 원") print(f" ─────────────────────") print(f" 현재 잔액:{balance:>12,} 원")

카테고리별 합계 함수
딕셔너리를 이용해 카테고리별로 금액을 집계합니다. 이 단계가 자료구조 활용의 핵심입니다.
딕셔너리 for 반복문 조건문 집계 패턴

수입 카테고리별 합계와 지출 카테고리별 합계를 각각 딕셔너리로 집계해 출력합니다. 예: income_summary = {"급여": 2500000, "용돈": 50000}

카테고리별 합계 출력 예시
── 카테고리별 합계 ── [수입] 급여 5,000,000 원 용돈 50,000 원 [지출] 식비 87,500 원 교통비 25,000 원 문화생활 45,000 원
HINT — show_summary

두 개의 빈 딕셔너리 income_sum = {}, expense_sum = {}를 만들고 for 루프로 순회하세요.

카테고리가 딕셔너리에 없으면 0으로 초기화: income_sum[cat] = income_sum.get(cat, 0) + amount

딕셔너리를 for로 순회할 때: for cat, total in income_sum.items():

# ─── 4단계: 카테고리별 합계 ────────────────────── def show_summary(transactions): """카테고리별 수입·지출 합계 출력.""" if not transactions: print("\n 내역이 없습니다.") return income_sum = {} expense_sum = {} for t in transactions: cat = t["category"] amount = t["amount"] if t["type"] == "수입": income_sum[cat] = income_sum.get(cat, 0) + amount else: expense_sum[cat] = expense_sum.get(cat, 0) + amount print("\n── 카테고리별 합계 ──") print("[수입]") if income_sum: for cat, total in income_sum.items(): print(f" {cat:<12}{total:>12,} 원") else: print(" 없음") print("[지출]") if expense_sum: for cat, total in expense_sum.items(): print(f" {cat:<12}{total:>12,} 원") else: print(" 없음")
05
메인 루프 완성 — main() 함수
모든 함수를 하나로 연결합니다. while 루프로 메뉴를 반복 표시하고 입력에 따라 함수를 호출합니다.
while True if / elif input() 함수 호출

드디어 모든 조각을 연결합니다. main() 안에서 load_data()로 시작하고, while True로 메뉴를 반복하며, 메뉴 6 입력 시 save_data() 후 break합니다.

  • ①프로그램 시작 시 타이틀 출력 + load_data() 호출
  • ②while True: 메뉴 출력 → 입력 → if/elif로 분기
  • ③"6" 입력 → save_data() → break
  • ④파일 맨 아래 if __name__ == "__main__": main() 추가
HINT — main()

choice = input("선택 > ").strip()으로 입력받은 뒤 if choice == "1":처럼 문자열로 비교하세요.

1~6 외 입력은 else:에서 "올바른 번호를 입력하세요 (1~6)."를 출력합니다.

if __name__ == "__main__":은 파이썬 파일을 직접 실행할 때만 main()이 호출되게 하는 관용 패턴입니다.

# ══════════════════════════════════════════════════ # 💰 나의 가계부 v1.0 # 브론즈 최종 프로젝트 — SuperPython # ══════════════════════════════════════════════════ FILE_NAME = "budget.csv" # ── 데이터 불러오기 ──────────────────────────────── def load_data(): transactions = [] try: with open(FILE_NAME, "r", encoding="utf-8") as f: next(f) # 헤더 건너뜀 for line in f: line = line.strip() if not line: continue try: t_type, category, amount, memo = line.split(",", 3) transactions.append({ "type": t_type, "category": category, "amount": int(amount), "memo": memo }) except (ValueError, IndexError): print(f"⚠ '{line}' 파싱 오류, 건너뜀") except FileNotFoundError: pass return transactions # ── 데이터 저장 ──────────────────────────────────── def save_data(transactions): with open(FILE_NAME, "w", encoding="utf-8") as f: f.write("type,category,amount,memo\n") for t in transactions: f.write(f"{t['type']},{t['category']},{t['amount']},{t['memo']}\n") print(f"💾 {FILE_NAME}에 저장 완료 ({len(transactions)}건)") # ── 수입 / 지출 추가 ────────────────────────────── def add_transaction(transactions, t_type): print(f"\n── {t_type} 추가 ──") try: category = input("카테고리 (예: 급여/식비/교통비) > ").strip() amount = int(input("금액 > ")) if amount <= 0: raise ValueError("금액은 1원 이상이어야 합니다.") memo = input("메모 (없으면 Enter) > ").strip() except ValueError as e: msg = str(e) print(f"❌ {msg if '이상' in msg else '숫자를 입력하세요.'}") return transactions.append({"type": t_type, "category": category, "amount": amount, "memo": memo}) sign = "+" if t_type == "수입" else "-" print(f"✅ {t_type} 추가 완료: {category} {sign}{amount:,}원") # ── 전체 내역 보기 ──────────────────────────────── def show_history(transactions): print("\n── 전체 거래 내역 ──") if not transactions: print(" 내역이 없습니다."); return print(f" {'번호':<4} {'구분':<5} {'카테고리':<10} {'금액':>12} {'메모'}") print(" " + "─" * 50) for i, t in enumerate(transactions, start=1): sign = "+" if t["type"] == "수입" else "-" amt = f"{sign}{t['amount']:,}" print(f" {i:<4} {t['type']:<5} {t['category']:<10} {amt:>12} {t['memo']}") # ── 잔액 확인 ───────────────────────────────────── def show_balance(transactions): income = sum(t["amount"] for t in transactions if t["type"] == "수입") expense = sum(t["amount"] for t in transactions if t["type"] == "지출") print(f"\n── 잔액 확인 ──") print(f" 총 수입: {income:>12,} 원") print(f" 총 지출: {expense:>12,} 원") print(f" {'─'*23}") print(f" 현재 잔액:{income - expense:>12,} 원") # ── 카테고리별 합계 ─────────────────────────────── def show_summary(transactions): if not transactions: print("\n 내역이 없습니다."); return income_sum, expense_sum = {}, {} for t in transactions: cat, amt = t["category"], t["amount"] if t["type"] == "수입": income_sum[cat] = income_sum.get(cat, 0) + amt else: expense_sum[cat] = expense_sum.get(cat, 0) + amt print("\n── 카테고리별 합계 ──") print("[수입]") for cat, total in (income_sum or {"없음": 0}).items(): print(f" {cat:<12}{total:>12,} 원" if cat != "없음" else " 없음") print("[지출]") for cat, total in (expense_sum or {"없음": 0}).items(): print(f" {cat:<12}{total:>12,} 원" if cat != "없음" else " 없음") # ── 메뉴 출력 ───────────────────────────────────── def print_menu(): print("\n━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━") print(" 1. 수입 추가 2. 지출 추가") print(" 3. 전체 내역 보기 4. 잔액 확인") print(" 5. 카테고리별 합계 6. 저장 후 종료") print("━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━") # ── 메인 루프 ───────────────────────────────────── def main(): print("╔══════════════════════════════════╗") print("║ 💰 나의 가계부 v1.0 ║") print("╚══════════════════════════════════╝") transactions = load_data() print(f"📂 불러오기 완료 ({len(transactions)}건)" if transactions else "📂 새 가계부를 시작합니다.") while True: print_menu() choice = input("선택 > ").strip() if choice == "1": add_transaction(transactions, "수입") elif choice == "2": add_transaction(transactions, "지출") elif choice == "3": show_history(transactions) elif choice == "4": show_balance(transactions) elif choice == "5": show_summary(transactions) elif choice == "6": save_data(transactions) print("프로그램을 종료합니다. 수고하셨습니다! 👋") break else: print("❌ 올바른 번호를 입력하세요 (1~6).") if __name__ == "__main__": main()

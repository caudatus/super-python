모듈 — import · 표준 라이브러리

파이썬에는 직접 만들 필요 없는 수천 개의 도구가 이미 준비되어 있습니다. 모듈을 불러오는 법을 배우면 그 모든 도구를 바로 쓸 수 있습니다.

▸ 이 챕터를 마치면
  • import로 모듈을 불러와서 사용할 수 있다
  • from ... import, import ... as 구문의 차이를 이해한다
  • random 모듈로 난수와 무작위 선택을 다룰 수 있다
  • math 모듈로 수학 계산을, os 모듈로 파일 경로를 다룰 수 있다

1.모듈이란?

모듈은 미리 만들어 놓은 파이썬 파일입니다. 함수, 변수, 클래스 등이 담겨 있어서 우리 코드에서 불러다 쓸 수 있습니다.

일상적인 비유를 들면, 요리할 때 모든 재료를 직접 농사짓지 않듯이 — 프로그래밍도 모든 기능을 처음부터 만들 필요가 없습니다. 파이썬에는 표준 라이브러리라는 거대한 도구함이 기본으로 탑재되어 있습니다. 설치 없이 바로 import만 하면 됩니다.

random
난수 생성, 무작위 선택
random() randint() choice() shuffle()
math
수학 함수와 상수
sqrt() floor() / ceil() pi / e pow() / log()
os
운영체제 · 파일 경로
getcwd() listdir() path.join() path.exists()

2.import 기본 사용법

모듈을 사용하는 가장 기본적인 방법은 import 모듈이름입니다. 불러온 뒤에는 모듈이름.함수() 형태로 사용합니다.

python
12345
import random          # random 모듈 불러오기

n = random.randint(1, 10)   # 1~10 사이 정수 무작위 선택
print(n)               # 예: 7 (실행할 때마다 달라짐)
실행 결과 (예시 — 매번 달라집니다)
7

여기서 핵심은 점(.)으로 연결한다는 것입니다. random.randint는 "random 모듈 안의 randint 함수"를 의미합니다. 이 점 표기법을 도트 접근이라고 부릅니다.

💡 TIP

import는 파일 맨 위에 모아 쓰는 것이 관례입니다. 파이썬 커뮤니티의 스타일 가이드(PEP 8)는 import를 표준 라이브러리 → 서드파티 → 자체 모듈 순서로 작성하도록 권장합니다.

3.import 구문의 세 가지 방식

모듈을 불러오는 방법은 상황에 따라 세 가지를 선택할 수 있습니다.

방식 1 — import 모듈

모듈 전체를 가져옵니다. 사용할 때 항상 모듈.함수() 형태로 써야 합니다.

python
123
import math

print(math.sqrt(16))   # 4.0

방식 2 — from 모듈 import 함수

모듈에서 특정 함수나 상수만 가져옵니다. 이름 앞에 모듈명을 붙이지 않아도 됩니다.

python
1234
from math import sqrt, pi

print(sqrt(25))   # 5.0   — math. 없이 바로 사용
print(pi)        # 3.141592653589793

방식 3 — import 모듈 as 별명

모듈에 짧은 별명(alias)을 붙입니다. 이름이 긴 모듈을 자주 쓸 때 코드가 간결해집니다.

python
123
import random as r

print(r.randint(1, 100))  # random 대신 r로 접근
✅ 권장 상황

모듈 전체가 필요하거나 어디서 왔는지 명확히 보여주고 싶을 때는 import math를 사용하세요. 코드를 읽는 사람이 math.sqrt를 보고 바로 맥락을 파악합니다.

💡 편의 상황

특정 함수만 반복적으로 쓴다면 from math import sqrt가 편합니다. 단, 같은 이름의 함수를 직접 만들면 충돌이 생길 수 있으니 주의하세요.

⚠ 주의

from math import *처럼 별표(*)로 전부 가져오는 방식은 권장하지 않습니다. 어떤 이름이 어느 모듈에서 왔는지 알 수 없어 코드를 읽기 어려워지고, 이름 충돌이 발생할 수 있습니다.

4.random 모듈 — 난수와 무작위

random 모듈은 게임, 시뮬레이션, 데이터 샘플링 등 무작위가 필요한 거의 모든 곳에서 쓰입니다.

random.random() — 0.0 이상 1.0 미만의 실수

python
123
import random

print(random.random())  # 예: 0.7342891... (0.0 이상, 1.0 미만)

random.randint(a, b) — a 이상 b 이하 정수

python
12345
import random

dice = random.randint(1, 6)   # 1~6 주사위
print(f"주사위: {dice}")        # 예: 주사위: 4
실행 결과 (예시)
주사위: 4
💡 TIP

randint(a, b)는 a와 b 모두 포함합니다. randint(1, 6)이면 1, 2, 3, 4, 5, 6 중 하나가 나옵니다. range(a, b)가 b를 제외하는 것과 다르니 헷갈리지 마세요.

random.choice(시퀀스) — 무작위 원소 하나 선택

python
123456
import random

menu = ["김치찌개", "된장찌개", "비빔밥", "냉면"]
today = random.choice(menu)

print(f"오늘의 점심: {today}")   # 예: 오늘의 점심: 비빔밥
실행 결과 (예시)
오늘의 점심: 비빔밥

random.shuffle(리스트) — 리스트 순서 섞기

python
123456
import random

cards = ["A", "2", "3", "J", "Q", "K"]
random.shuffle(cards)    # 리스트 자체를 섞음 (반환값 없음)
print(cards)            # 예: ['3', 'K', 'A', 'Q', '2', 'J']
실행 결과 (예시)
['3', 'K', 'A', 'Q', '2', 'J']
⚠ 주의

shuffle()은 리스트를 직접 변경하고 None을 반환합니다. result = random.shuffle(cards)처럼 반환값을 받으면 result는 None이 됩니다. 변경된 리스트는 원래 변수 cards에서 확인하세요.

random.sample(시퀀스, k) — 중복 없이 k개 뽑기

python
12345
import random

lotto = random.sample(range(1, 46), 6)  # 1~45에서 6개 추출
print(sorted(lotto))  # 예: [3, 14, 22, 31, 38, 44]
실행 결과 (예시)
[3, 14, 22, 31, 38, 44]

5.math 모듈 — 수학 도구함

math 모듈은 제곱근, 올림/내림, 삼각함수, 로그, 그리고 π(파이) 같은 수학 상수를 제공합니다.

자주 쓰는 함수와 상수

함수/상수설명예시
math.sqrt(x)x의 제곱근 (결과는 float)sqrt(9) → 3.0
math.floor(x)x 이하의 가장 큰 정수 (내림)floor(3.7) → 3
math.ceil(x)x 이상의 가장 작은 정수 (올림)ceil(3.2) → 4
math.pow(x, y)x의 y제곱 (결과는 float)pow(2, 10) → 1024.0
math.abs(x)절댓값 — 내장 abs() 사용 권장abs(-5) → 5
math.pi원주율 π (상수)3.141592653589793
math.e자연상수 e (상수)2.718281828459045
python
1234567891011
import math

# 제곱근
print(math.sqrt(144))     # 12.0

# 올림 / 내림
print(math.ceil(4.1))     # 5
print(math.floor(4.9))    # 4

# 원의 넓이 공식: π × r²
r = 5
area = math.pi * r ** 2
print(f"반지름 {r}인 원의 넓이: {area:.2f}")  # 78.54
실행 결과
12.0
5
4
반지름 5인 원의 넓이: 78.54
💡 TIP

math.pow(2, 10)은 항상 float를 반환합니다. 정수 결과가 필요하면 파이썬 내장 연산자 2 ** 10을 쓰는 것이 더 자연스럽습니다.

6.os 모듈 — 파일과 경로 다루기

os 모듈은 운영체제(Operating System)와 상호작용하는 함수들을 모아 놓은 모듈입니다. 파일 경로를 만들거나, 현재 위치를 확인하거나, 폴더 안의 파일 목록을 가져올 때 사용합니다.

os.getcwd() — 현재 작업 디렉터리

python
123
import os

print(os.getcwd())   # 예: /Users/kim/projects
실행 결과 (환경마다 다름)
/Users/kim/projects

os.listdir(경로) — 폴더 안의 파일/폴더 목록

python
1234
import os

files = os.listdir(".")   # "." = 현재 폴더
print(files)               # 예: ['main.py', 'data.txt', 'images']

os.path.join() — 경로 합치기

경로를 직접 문자열로 이어 붙이면 Windows(\)와 macOS/Linux(/)에서 다르게 동작합니다. os.path.join()을 사용하면 운영체제에 맞게 자동으로 처리됩니다.

python
1234567
import os

folder = "data"
filename = "result.txt"

path = os.path.join(folder, filename)
print(path)  # macOS/Linux: data/result.txt  |  Windows: data\result.txt
실행 결과 (macOS/Linux 기준)
data/result.txt

os.path.exists() — 파일/폴더 존재 여부 확인

python
123456
import os

if os.path.exists("data.txt"):
    print("파일이 있습니다.")
else:
    print("파일이 없습니다.")
ℹ 참고

브론즈에서는 os.getcwd(), os.listdir(), os.path.join(), os.path.exists() 정도만 알면 충분합니다. 더 깊은 os 활용(디렉터리 생성, 삭제, 환경 변수 등)은 플래티넘 과정에서 다룹니다.

7.실전 예제 — 세 모듈 함께 쓰기

배운 세 모듈을 함께 활용해 간단한 프로그램을 만들어 봅니다. 로또 번호 생성기입니다.

python
123456789101112131415
import random
import math
import os

# 로또 번호 생성 (1~45에서 6개, 중복 없이)
numbers = random.sample(range(1, 46), 6)
numbers.sort()

# 합계와 평균 계산
total = sum(numbers)
avg   = math.floor(total / len(numbers))

# 현재 작업 경로 확인
cwd = os.getcwd()

print(f"로또 번호: {numbers}")
print(f"합계: {total}, 평균(내림): {avg}")
print(f"실행 위치: {cwd}")
실행 결과 (예시)
로또 번호: [3, 14, 22, 31, 38, 44]
합계: 152, 평균(내림): 25
실행 위치: /Users/kim/projects

8.자주 나오는 실수

실수 1 — import 없이 사용

python
1
print(random.randint(1, 10))   # import random를 빠뜨림
오류 메시지
NameError: name 'random' is not defined

모듈은 반드시 사용 전에 import해야 합니다. 코드 맨 위에 import random을 추가하세요.

실수 2 — 모듈명 없이 함수 호출

python
12
import math
print(sqrt(9))   # math.sqrt 가 아니라 그냥 sqrt 로 씀
오류 메시지
NameError: name 'sqrt' is not defined

import math를 쓴 경우 반드시 math.sqrt()처럼 모듈 이름을 앞에 붙여야 합니다. 모듈명 없이 쓰려면 from math import sqrt를 사용하세요.

실수 3 — shuffle 반환값 사용

python
1234
import random
items = [1, 2, 3, 4]
result = random.shuffle(items)
print(result)   # None 출력! 원래 의도와 다름
실행 결과
None

shuffle()은 리스트를 제자리에서 변경하고 None을 반환합니다. print(items)로 원래 변수를 출력하세요.

▸ 핵심 요약
import 모듈 모듈 전체를 불러옴. 사용 시 모듈.함수() 형태 필수 from 모듈 import 함수 특정 함수만 가져옴. 모듈명 없이 바로 사용 가능 import 모듈 as 별명 긴 모듈 이름을 짧게 줄여 사용. import numpy as np 패턴 random.randint(a, b) a 이상 b 이하 무작위 정수 (양 끝 포함) random.choice(seq) 시퀀스에서 원소 하나를 무작위로 선택 random.sample(seq, k) 중복 없이 k개를 무작위로 뽑아 새 리스트 반환 random.shuffle(list) 리스트 순서를 제자리에서 섞음. 반환값은 None math.sqrt / ceil / floor 제곱근, 올림, 내림. math.pi / math.e 로 상수 접근 os.getcwd() 현재 작업 디렉터리 경로를 문자열로 반환 os.path.join(a, b) 경로를 OS에 맞게 안전하게 합쳐줌

Q11. 아래 코드에는 오류가 있습니다. 어떤 에러이고, 어떻게 고치나요?

💡 힌트

TypeError — 문자열 + 정수 연산 불가

# 수정 방법 2: f-string 사용 (권장)
print(f"총점: {total}")

Q25. 아래 코드에는 논리적 버그가 있습니다. 에러는 발생하지 않지만 의도와 다릅니다. 버그를 찾아 설명하고 수정하세요. (의도: 양수 점수만 합산)

💡 힌트
s가 음수(-5)일 때 total -= s 는 total += 5 와 같습니다. 의도한 동작인가요?
✅ 정답
else 블록에서 total -= s 를 하면 음수값을 빼므로 오히려 더해집니다.
(-5를 빼면 +5가 됨)

# 수정: else 블록 자체를 제거 (음수는 무시)
for s in scores:
    if s > 0:
        total += s
# 결과: 80 + 90 + 70 = 240

Q26. 아래 코드를 실행하면 몇 번째 반복에서 에러가 발생하나요? 그리고 에러 유형은 무엇인가요?

words = ["파이썬", "자바"]
for i in range(4):
    print(words[i])
💡 힌트
range(4) = 0,1,2,3. words 의 유효 인덱스는 0,1입니다. 언제 범위를 벗어나나요?
✅ 정답
3번째 반복(i=2)에서 IndexError 발생

i=0: words[0] = "파이썬"  ✓
i=1: words[1] = "자바"    ✓
i=2: words[2] → 없음 → IndexError ✗

# 수정: range(len(words)) 사용
for i in range(len(words)):
    print(words[i])

Q27. 아래 코드는 세 가지 에러를 모두 포함하고 있습니다. 각 줄의 에러 유형을 찾아 설명하세요.

x = input("x: ")
y = 10
print(x + y)             # 줄 3
if x > y                  # 줄 4
    print(X)              # 줄 5
💡 힌트
줄 3: input() 반환값 타입 확인
줄 4: if문 문법 확인
줄 5: 변수명 대소문자 확인
✅ 정답
줄 3 → TypeError: x 는 str, y 는 int → str + int 불가
줄 4 → SyntaxError: if x > y 뒤에 콜론(:) 없음
줄 5 → NameError: x(소문자)로 선언했는데 X(대문자) 사용

# 수정된 코드:
x = int(input("x: "))    # int() 변환
y = 10
print(x + y)              # 이제 int + int
if x > y:                 # 콜론 추가
    print(x)              # 소문자로 통일

Q28. 아래 코드는 학생 점수 리스트에서 평균을 구합니다. 오류 없이 실행되지만 결과가 의도와 다릅니다. 버그를 찾아 설명하세요.

scores = [70, 80, 90]
total  = 0
for s in scores:
    total += s
avg = total / 2   # ← 이 줄
print(avg)
💡 힌트
평균은 합계 / 개수입니다. scores의 개수는 몇 개인가요?
✅ 정답
나누는 수를 2로 고정했지만 실제 개수는 3개입니다.

# 현재: 240 / 2 = 120.0 (틀림)
# 수정: len(scores)로 나눠야 함
avg = total / len(scores)   # 240 / 3 = 80.0

Q29. 🔥 도전 문제 — 아래 코드에는 에러가 총 3개 있습니다. 실행 전에 모두 찾아 유형과 위치를 설명하고, 완전히 수정된 코드를 작성하세요.

numbers = input("숫자들 (쉼표 구분): ")
nums_list = numbers.split(",")
total = 0
for n in nums_list:
    total += n               # 줄 5
avg = total / len(nums_list)
print("평균: " + avg)        # 줄 7
💡 힌트
① split()의 결과 타입은? ② float도 str과 합칠 수 없습니다.
③ 줄 5와 줄 7의 타입을 확인하세요.
✅ 정답
① 줄 5 — TypeError: split()은 문자열 리스트를 반환하므로 n은 str.
         total += n 은 int + str 불가.
② 줄 7 — TypeError: "평균: "(str) + avg(float) 불가.
③ (위 ①과 연쇄적으로 avg 도 float이 되어야 함)

# 수정된 코드:
numbers  = input("숫자들 (쉼표 구분): ")
nums_list = numbers.split(",")
total = 0
for n in nums_list:
    total += int(n)          # str → int 변환
avg = total / len(nums_list)
print("평균: " + str(avg))   # float → str 변환
# 또는: print(f"평균: {avg}")

Q30. 🔥 도전 문제 — 아래 코드는 사용자로부터 리스트 인덱스를 입력받아 항목을 출력합니다. 어떤 에러가 발생할 수 있는지 2가지 상황을 설명하고, 두 에러를 모두 예방하는 코드를 작성하세요.

menu = ["김치찌개", "된장찌개", "불고기"]
idx  = input("번호 입력 (0~2): ")
print(menu[idx])
💡 힌트
① input()의 반환 타입을 생각하세요.
② 사용자가 0~2 외의 숫자(예: 5)를 입력하면 어떻게 되나요?
✅ 정답
상황 ① TypeError — idx 가 str이라 menu[idx] 에서 에러 발생
                   (리스트 인덱스는 정수여야 함)

상황 ② IndexError — int 변환 후에도 0~2 범위를 벗어나면 에러 발생
                    (예: 5 입력 시 menu[5]는 없음)

# 두 에러를 모두 예방하는 코드:
menu = ["김치찌개", "된장찌개", "불고기"]
idx  = int(input("번호 입력 (0~2): "))   # ① int() 변환
if 0 <= idx <= len(menu) - 1:           # ② 범위 체크
    print(menu[idx])
else:
    print("유효하지 않은 번호입니다.")

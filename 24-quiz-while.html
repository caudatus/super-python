🔥 도전 문제 — 콜라츠 추측(Collatz Conjecture)을 구현하세요.
규칙: n이 짝수이면 2로 나누고, 홀수이면 3을 곱하고 1을 더합니다. 반복하면 항상 1에 도달합니다.
숫자를 입력받아 1이 될 때까지 과정을 출력하고, 몇 번 만에 1이 됐는지 출력하세요.

💡 힌트

while n != 1:로 루프, n이 짝수이면 n //= 2, 홀수이면 n = 3*n + 1. 단계 수 카운터도 추가하세요.

✅ 정답
n     = int(input("숫자: "))
steps = 0
print(n)
while n != 1:
    if n % 2 == 0:
        n //= 2           # 짝수: 2로 나눔
    else:
        n = 3 * n + 1   # 홀수: 3n+1
    print(n)
    steps += 1
print(f"→ {steps}번 만에 1에 도달")
Q30.

🔥 도전 문제 — 간단한 메뉴 시스템을 구현하세요.
반복적으로 메뉴를 표시하고 사용자가 선택합니다.
1. 안녕하세요 출력 / 2. 현재 횟수 출력 / 3. 종료
메뉴 1을 선택할 때마다 선택 횟수가 올라가며, 메뉴 2에서 "지금까지 [N]번 선택했습니다."를 출력합니다. 3을 선택하면 "프로그램을 종료합니다."를 출력하고 종료합니다. 잘못된 입력은 "올바른 번호를 입력하세요."를 출력합니다.

💡 힌트

while True로 무한루프, 메뉴 출력 → 선택 입력 → if-elif-else로 분기. 횟수는 별도 변수로 관리.

✅ 정답
count = 0
while True:
    print("\n--- 메뉴 ---")
    print("1. 안녕하세요 출력")
    print("2. 현재 횟수 출력")
    print("3. 종료")
    choice = input("선택: ")
    if choice == "1":
        print("안녕하세요!")
        count += 1
    elif choice == "2":
        print(f"지금까지 {count}번 선택했습니다.")
    elif choice == "3":
        print("프로그램을 종료합니다.")
        break
    else:
        print("올바른 번호를 입력하세요.")

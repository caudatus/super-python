세 모듈을 모두 활용하는 BMI 계산기를 만드세요. random으로 키(150~190cm)와 몸무게(50~100kg)를 생성하고, math로 BMI를 소수점 한 자리로 계산하고, os로 저장 경로(현재디렉터리/bmi_result.txt)를 구성해 출력하세요. BMI = 몸무게 / 키(m)²

✅ 정답
import random
import math
import os

height_cm = random.randint(150, 190)
weight    = random.randint(50, 100)
height_m  = height_cm / 100
bmi       = weight / height_m ** 2
save_path = os.path.join(os.getcwd(), "bmi_result.txt")

print(f"키: {height_cm}cm | 몸무게: {weight}kg")
print(f"BMI: {bmi:.1f}")
print(f"저장 경로: {save_path}")

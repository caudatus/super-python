개발환경 설치

파이썬을 설치하고, 코드를 편하게 쓸 수 있는 환경을 만들어봅시다

▸ 이 챕터를 마치면
  • 파이썬 3.12 이상을 내 컴퓨터에 설치할 수 있다
  • VS Code를 설치하고 파이썬 확장을 추가할 수 있다
  • print("Hello, Python!")을 직접 실행해서 결과를 확인할 수 있다

1.필요한 것 두 가지

파이썬 코딩을 시작하려면 딱 두 가지만 설치하면 됩니다.

1
Python 인터프리터

파이썬 코드를 실제로 실행하는 프로그램입니다. 우리가 쓴 코드를 컴퓨터가 이해할 수 있도록 번역하고 실행합니다. python.org에서 무료로 다운로드합니다.

2
VS Code (코드 편집기)

코드를 작성하는 프로그램입니다. 메모장에도 쓸 수 있지만, VS Code는 자동완성·색상 강조·디버깅 기능이 있어 훨씬 편합니다. code.visualstudio.com에서 무료로 다운로드합니다.

💡 IDE vs 에디터

VS Code는 코드 편집기(Editor)입니다. PyCharm 같은 IDE(통합개발환경)도 있지만, 입문자에게는 가볍고 빠른 VS Code를 권장합니다. 나중에 익숙해지면 다른 도구로 바꿔도 됩니다.

2.Python 설치

OS에 맞는 탭을 선택해서 따라하세요.

🪟 Windows
🍎 macOS
🐧 Linux
  1. python.org/downloads 에 접속합니다.
  2. 노란 버튼 "Download Python 3.x.x" 를 클릭해서 설치 파일을 다운로드합니다.
  3. 다운로드된 .exe 파일을 실행합니다.
  4. 설치 화면 맨 아래 "Add Python to PATH" 체크박스를 반드시 체크합니다. ← 이게 핵심!
  5. "Install Now" 를 클릭하고 설치가 완료될 때까지 기다립니다.
⚠ PATH 체크를 빠뜨리면

나중에 터미널에서 python 명령을 쳤을 때 "명령어를 찾을 수 없습니다" 오류가 납니다. 설치 시 반드시 체크하세요. 빠뜨렸다면 제어판 → 앱 → Python → 수정에서 다시 설정할 수 있습니다.

  1. python.org/downloads 에 접속합니다.
  2. "Download Python 3.x.x" 버튼을 클릭해서 .pkg 파일을 다운로드합니다.
  3. 다운로드된 파일을 더블클릭해서 설치를 진행합니다.
  4. 또는 Homebrew가 설치되어 있다면 터미널에서 아래 명령으로 설치할 수 있습니다.
Terminal 
$ brew install python
  1. 대부분의 Linux 배포판에는 파이썬이 이미 설치되어 있습니다.
  2. 터미널에서 버전을 먼저 확인하세요.
  3. 3.10 미만이거나 설치되지 않은 경우 아래 명령으로 설치합니다.
Terminal — Ubuntu / Debian 
$ sudo apt update
$ sudo apt install python3 python3-pip

설치 확인

설치가 끝나면 터미널(명령 프롬프트)을 열고 아래 명령어를 입력해서 확인합니다.

Terminal 
$ python --version
Python 3.12.3

$ python3 --version   # macOS / Linux는 python3
Python 3.12.3
✅
버전 번호가 출력되면 성공입니다! 3.10 이상이면 이 강의를 따라하는 데 문제 없습니다. "명령어를 찾을 수 없습니다" 오류가 나면 아래 팁 박스를 확인하세요.
⚠ Windows에서 오류가 날 때

python이 안 되면 py를 대신 써보세요. 그래도 안 되면 Python 설치 시 "Add to PATH"를 빠뜨린 것입니다. 제어판 → 앱 → Python → 수정에서 PATH를 추가할 수 있습니다.

3.VS Code 설치 및 설정

code.visualstudio.com 에 접속해서 본인 OS에 맞는 버전을 다운로드하고 설치합니다. 설치 과정은 "다음" 버튼만 계속 누르면 됩니다.

Python 확장 설치

VS Code 자체는 파이썬을 모릅니다. 파이썬 지원을 추가하는 확장(Extension)을 설치해야 합니다.

1
확장 탭 열기

VS Code 왼쪽 사이드바에서 네모 4개 아이콘(Extensions)을 클릭하거나, 단축키 Ctrl+Shift+X (macOS: Cmd+Shift+X)를 누릅니다.

2
Python 검색 후 설치

검색창에 "Python"을 입력합니다. Microsoft가 만든 Python 확장(파란 배지)을 찾아 Install 버튼을 클릭합니다.

3
인터프리터 선택

파일을 열면 우측 하단에 파이썬 버전이 표시됩니다. 클릭해서 방금 설치한 Python 3.12를 선택합니다. 또는 Ctrl+Shift+P → "Python: Select Interpreter"로 선택할 수 있습니다.

💡 추천 추가 확장

당장은 필요 없지만, 나중에 설치하면 편한 확장들입니다.
Pylance — 더 강력한 자동완성과 타입 검사
indent-rainbow — 들여쓰기 단계를 색으로 구분해서 읽기 편함
Korean Language Pack — VS Code 메뉴를 한국어로

4.첫 코드 실행하기

드디어 코드를 직접 실행해볼 차례입니다. 두 가지 방법 모두 알아두면 편리합니다.

방법 A — VS Code에서 파일로 실행

1
새 파일 만들기

VS Code에서 Ctrl+N (macOS: Cmd+N)으로 새 파일을 만들고, Ctrl+S로 hello.py로 저장합니다. .py가 파이썬 파일 확장자입니다.

2
코드 입력

아래 코드를 그대로 입력합니다.

python — hello.py
12
print("Hello, Python!")
print("나의 첫 번째 파이썬 코드")
3
실행

우측 상단의 ▶ 버튼을 클릭하거나, 단축키 Ctrl+F5를 누릅니다. 하단 터미널에 결과가 출력됩니다.

실행 결과
Hello, Python!
나의 첫 번째 파이썬 코드

방법 B — 터미널(REPL)에서 바로 실행

파일 없이 터미널에서 한 줄씩 바로 실행하는 방법입니다. 빠르게 테스트할 때 유용합니다.

Terminal — Python REPL 
$ python          # 파이썬 실행
Python 3.12.3 ...
>>> print("Hello!")
Hello!
>>> 1 + 2
3
>>> exit()          # 종료
📌 REPL이란?

Read-Eval-Print Loop의 약자입니다. 코드를 입력하면(Read) 즉시 실행하고(Eval) 결과를 보여주는(Print) 것을 반복(Loop)합니다. 작은 코드를 빠르게 테스트할 때 유용합니다. >>>가 입력 대기 표시입니다.

⌨ 직접 해보기

설치가 완료됐다면 아래 순서대로 직접 해보세요.

01. 터미널에서 python --version을 입력해서 버전이 출력되는지 확인합니다.
02. VS Code를 열고 hello.py 파일을 만들어서 print("Hello, Python!")을 입력하고 실행합니다.
03. 터미널에서 python을 입력해 REPL을 열고, 1 + 2를 입력해서 3이 출력되는지 확인합니다.
도전. hello.py에 내 이름과 나이를 출력하는 코드를 추가해보세요. print()는 다음 챕터에서 자세히 배우지만, 지금 한 번 써보는 것도 좋습니다!

Q1.

이름이 "Alice", 나이가 25인 딕셔너리 person을 만들고, person["name"]의 값을 출력하세요.

출력:Alice
HINT

딕셔너리는 {"키": 값} 형태로 만들어요. 값을 꺼낼 때는 변수["키"]를 사용합니다.

ANSWER
# 딕셔너리 생성 person = {"name": "Alice", "age": 25} print(person["name"]) # 키로 값 조회
Q2.

아래 딕셔너리에서 "banana"의 가격을 출력하세요.

prices = {"apple": 1200, "banana": 800, "cherry": 3000}
출력:800
HINT

딕셔너리["키"] 형태로 값에 접근하세요.

ANSWER
prices = {"apple": 1200, "banana": 800, "cherry": 3000} print(prices["banana"]) # 키 "banana"에 해당하는 값 800 출력
Q3.

아래 딕셔너리에 "city": "Seoul" 항목을 추가하고, 딕셔너리 전체를 출력하세요.

info = {"name": "Bob", "age": 30}
출력:{'name': 'Bob', 'age': 30, 'city': 'Seoul'}
HINT

존재하지 않는 키에 값을 대입하면 자동으로 추가됩니다: 딕셔너리["새키"] = 값

ANSWER
info = {"name": "Bob", "age": 30} info["city"] = "Seoul" # 새 키-값 쌍 추가 print(info)
Q4.

아래 딕셔너리에서 "age" 값을 28로 수정하고, 수정된 age를 출력하세요.

user = {"name": "Charlie", "age": 25}
출력:28
HINT

이미 존재하는 키에 다시 값을 대입하면 값이 수정됩니다: 딕셔너리["기존키"] = 새값

ANSWER
user = {"name": "Charlie", "age": 25} user["age"] = 28 # 기존 키의 값 수정 print(user["age"])
Q5.

아래 딕셔너리에서 "score" 항목을 삭제하고, 삭제 후 딕셔너리를 출력하세요.

data = {"name": "Dana", "score": 95, "level": "A"}
출력:{'name': 'Dana', 'level': 'A'}
HINT

del 딕셔너리["키"] 또는 딕셔너리.pop("키")를 사용하세요.

ANSWER
data = {"name": "Dana", "score": 95, "level": "A"} del data["score"] # "score" 키-값 쌍 삭제 print(data)
Q6.

아래 딕셔너리에서 "email" 키가 존재하는지 확인하는 코드를 작성하고 결과를 출력하세요.

member = {"id": 1001, "name": "Eve"}
출력:False
HINT

"키" in 딕셔너리 를 사용하면 키의 존재 여부를 True/False로 알 수 있어요.

ANSWER
member = {"id": 1001, "name": "Eve"} print("email" in member) # "email" 키 없음 → False 출력
Q7.

아래 딕셔너리의 전체 키(key) 목록을 출력하세요.

car = {"brand": "Toyota", "model": "Camry", "year": 2022}
출력:dict_keys(['brand', 'model', 'year'])
HINT

딕셔너리.keys() 메서드를 사용하세요.

ANSWER
car = {"brand": "Toyota", "model": "Camry", "year": 2022} print(car.keys()) # dict_keys 객체 출력
Q8.

아래 딕셔너리의 전체 값(value) 목록을 출력하세요.

scores = {"math": 90, "english": 85, "science": 92}
출력:dict_values([90, 85, 92])
HINT

딕셔너리.values() 메서드를 사용하세요.

ANSWER
scores = {"math": 90, "english": 85, "science": 92} print(scores.values()) # dict_values 객체 출력
Q9.

get() 메서드를 사용해서 아래 딕셔너리에서 "phone" 키의 값을 조회하되, 키가 없으면 "없음"을 반환해 출력하세요.

contact = {"name": "Frank", "email": "frank@test.com"}
출력:없음
HINT

딕셔너리.get("키", 기본값) — 키가 없을 때 반환할 기본값을 두 번째 인자로 넣어주세요.

ANSWER
contact = {"name": "Frank", "email": "frank@test.com"} print(contact.get("phone", "없음")) # "phone" 키 없음 → "없음" 반환

Q10.

아래 딕셔너리의 모든 키를 for문으로 순서대로 출력하세요.

fruit_color = {"apple": "red", "banana": "yellow", "grape": "purple"}
출력:apple banana grape
HINT

딕셔너리를 for로 순회하면 기본적으로 키가 반환됩니다: for key in 딕셔너리:

ANSWER
fruit_color = {"apple": "red", "banana": "yellow", "grape": "purple"} for key in fruit_color: print(key) # 딕셔너리 순회 시 기본은 키
Q11.

.items()를 사용해 아래 딕셔너리의 키와 값을 쌍으로 출력하세요.

menu = {"coffee": 4500, "juice": 5000, "tea": 4000}
출력:coffee 4500 juice 5000 tea 4000
HINT

for k, v in 딕셔너리.items(): 형태를 사용하면 키와 값을 동시에 받아올 수 있어요.

ANSWER
menu = {"coffee": 4500, "juice": 5000, "tea": 4000} for k, v in menu.items(): print(k, v) # 키와 값을 동시에 언패킹
Q12.

아래 딕셔너리에서 pop()으로 "banana"를 꺼낸 뒤, 꺼낸 값을 출력하세요.

basket = {"apple": 3, "banana": 5, "mango": 2}
출력:5
HINT

딕셔너리.pop("키")는 키를 삭제하면서 해당 값을 반환합니다.

ANSWER
basket = {"apple": 3, "banana": 5, "mango": 2} removed = basket.pop("banana") # 삭제 + 반환 print(removed)
Q13.

update()를 사용해 아래 딕셔너리에 {"city": "Busan", "job": "engineer"}를 병합하고, 결과를 출력하세요.

profile = {"name": "Grace", "age": 27}
출력:{'name': 'Grace', 'age': 27, 'city': 'Busan', 'job': 'engineer'}
HINT

딕셔너리.update(다른딕셔너리) — 인자로 넘긴 딕셔너리의 내용이 원본에 추가됩니다.

ANSWER
profile = {"name": "Grace", "age": 27} profile.update({"city": "Busan", "job": "engineer"}) # 두 항목 추가 print(profile)
Q14.

아래 딕셔너리에서 점수가 80 이상인 과목의 이름만 출력하세요.

grades = {"국어": 75, "영어": 88, "수학": 92, "과학": 70}
출력:영어 수학
HINT

for k, v in 딕셔너리.items():로 순회하면서 if v >= 80 조건을 걸어보세요.

ANSWER
grades = {"국어": 75, "영어": 88, "수학": 92, "과학": 70} for subject, score in grades.items(): if score >= 80: print(subject) # 조건 충족 시 과목명 출력

아래 딕셔너리의 모든 값을 합산해서 출력하세요.

sales = {"1월": 120, "2월": 95, "3월": 140, "4월": 110}
출력:465
HINT

sum(딕셔너리.values())를 사용하거나, for문으로 직접 누적 합산해도 됩니다.

sales = {"1월": 120, "2월": 95, "3월": 140, "4월": 110} print(sum(sales.values())) # values()의 합계
Q16.

아래 딕셔너리를 복사본 copy_d를 만들고, copy_d["x"]를 99로 바꾼 뒤 원본 d["x"]를 출력하세요. (원본은 바뀌지 않아야 합니다)

d = {"x": 10, "y": 20}
출력:10
HINT

딕셔너리.copy() 메서드로 얕은 복사본을 만들 수 있어요. copy_d = d는 같은 객체를 가리키므로 안 됩니다!

ANSWER
d = {"x": 10, "y": 20} copy_d = d.copy() # 독립된 복사본 생성 copy_d["x"] = 99 # 복사본만 수정 print(d["x"]) # 원본 d는 그대로 10
Q17.

아래 코드의 출력 결과를 예상하고, 실제로 작성해 확인하세요.

word_count = {} words = ["apple", "banana", "apple", "cherry", "banana", "apple"] for w in words: word_count[w] = word_count.get(w, 0) + 1 print(word_count)
출력:{'apple': 3, 'banana': 2, 'cherry': 1}
HINT

get(w, 0)은 키 w가 없으면 0을 반환해요. 그래서 처음 등장하는 단어는 0 + 1 = 1로 시작하고, 이미 있는 단어는 기존 값에 1을 더하게 됩니다.

ANSWER
# 단어 빈도 카운터 — 딕셔너리의 대표적인 활용 패턴 word_count = {} words = ["apple", "banana", "apple", "cherry", "banana", "apple"] for w in words: word_count[w] = word_count.get(w, 0) + 1 # 없으면 0+1, 있으면 기존값+1 print(word_count)
Q18.

딕셔너리 컴프리헨션으로 1~5까지의 숫자를 키, 그 제곱을 값으로 갖는 딕셔너리를 만들어 출력하세요.

출력:{1: 1, 2: 4, 3: 9, 4: 16, 5: 25}
HINT

딕셔너리 컴프리헨션 구조: {키표현식: 값표현식 for 변수 in 범위} — 리스트 컴프리헨션과 비슷하게 { } 안에 작성해요.

ANSWER
# 딕셔너리 컴프리헨션: {키: 값 for 변수 in 범위} squares = {i: i**2 for i in range(1, 6)} print(squares)

Q19.

아래 중첩 딕셔너리에서 "Alice"의 "math" 점수를 출력하세요.

students = { "Alice": {"math": 90, "english": 85}, "Bob": {"math": 78, "english": 92} }

중첩 딕셔너리는 대괄호를 두 번 사용합니다: 딕셔너리["외부키"]["내부키"]

Q20.

아래 중첩 딕셔너리에서 모든 학생의 이름과 수학 점수를 출력하세요.

students = { "Alice": {"math": 90, "english": 85}, "Bob": {"math": 78, "english": 92}, "Carol": {"math": 95, "english": 88} }
출력:Alice 90 Bob 78 Carol 95
HINT

for name, info in students.items():로 순회하고, info["math"]로 점수에 접근하세요.

ANSWER
students = { "Alice": {"math": 90, "english": 85}, "Bob": {"math": 78, "english": 92}, "Carol": {"math": 95, "english": 88} } for name, info in students.items(): print(name, info["math"]) # 이름 + 수학 점수
Q21.

아래 딕셔너리에서 값이 가장 큰 키를 출력하세요.

stock = {"삼성": 75000, "카카오": 52000, "네이버": 188000, "LG": 95000}
출력:네이버

max(딕셔너리, key=딕셔너리.get)을 사용하면 값 기준으로 가장 큰 키를 찾을 수 있어요.

stock = {"삼성": 75000, "카카오": 52000, "네이버": 188000, "LG": 95000} print(max(stock, key=stock.get)) # 값 기준 최대 키 반환
Q22.

아래 코드에서 잘못된 부분을 찾아 고쳐 출력 결과가 서울이 되도록 수정하세요.

# 오류가 있는 코드 city_map = {"KR": "서울", "JP": "도쿄", "US": "뉴욕"} print(city_map["kr"])

딕셔너리의 키는 대소문자를 구분합니다! 키가 "KR"인데 "kr"로 접근하면 KeyError가 발생해요.

# "kr" → "KR" 로 대소문자 수정 city_map = {"KR": "서울", "JP": "도쿄", "US": "뉴욕"} print(city_map["KR"]) # 키는 대소문자 구분!
Q23.

아래 리스트를 딕셔너리로 변환하세요. 각 과일을 키, 해당 인덱스(0부터)를 값으로 만들어 출력하세요.

fruits = ["apple", "banana", "cherry"]

Q24.

두 리스트를 zip()으로 묶어 딕셔너리를 만들고 출력하세요.

keys = ["name", "age", "city"] values = ["Hana", 29, "Incheon"]
출력:{'name': 'Hana', 'age': 29, 'city': 'Incheon'}

dict(zip(keys, values)) — zip()은 두 이터러블을 쌍으로 묶고, dict()로 딕셔너리로 변환합니다.

keys = ["name", "age", "city"] values = ["Hana", 29, "Incheon"] result = dict(zip(keys, values)) # zip으로 쌍을 만들고 dict로 변환 print(result)
Q25.

아래 딕셔너리에서 값이 0인 항목을 모두 제거하고, 결과를 출력하세요.

inventory = {"사과": 5, "바나나": 0, "딸기": 3, "포도": 0, "수박": 2}
출력:{'사과': 5, '딸기': 3, '수박': 2}
HINT

딕셔너리 컴프리헨션으로 조건에 맞는 항목만 남길 수 있어요: {k: v for k, v in d.items() if 조건}

ANSWER
inventory = {"사과": 5, "바나나": 0, "딸기": 3, "포도": 0, "수박": 2} # 값이 0이 아닌 항목만 필터링 result = {k: v for k, v in inventory.items() if v != 0} print(result)
Q26.

아래 딕셔너리의 키와 값을 뒤바꾼 새 딕셔너리를 만들어 출력하세요.

eng_to_kor = {"apple": "사과", "banana": "바나나", "cherry": "체리"}
출력:{'사과': 'apple', '바나나': 'banana', '체리': 'cherry'}
HINT

컴프리헨션에서 키와 값의 위치를 바꾸면 됩니다: {v: k for k, v in d.items()}

ANSWER
eng_to_kor = {"apple": "사과", "banana": "바나나", "cherry": "체리"} # 키와 값 위치 교환 (값 → 키, 키 → 값) kor_to_eng = {v: k for k, v in eng_to_kor.items()} print(kor_to_eng)
Q27.

아래 문자열에서 각 문자가 몇 번 등장하는지 딕셔너리로 만들어 출력하세요. (공백 제외)

text = "hello world"
출력:{'h': 1, 'e': 1, 'l': 3, 'o': 2, 'w': 1, 'r': 1, 'd': 1}
HINT

문자열도 for로 순회할 수 있어요. 공백은 if c != " "로 건너뛰고, get(c, 0) + 1 패턴으로 카운팅하세요.

ANSWER
text = "hello world" char_count = {} for c in text: if c != " ": # 공백 건너뜀 char_count[c] = char_count.get(c, 0) + 1 print(char_count)
Q28.

아래 학생 목록(딕셔너리의 리스트)에서 점수가 90 이상인 학생의 이름만 리스트로 만들어 출력하세요.

students = [ {"name": "Anna", "score": 88}, {"name": "Brian", "score": 95}, {"name": "Cindy", "score": 72}, {"name": "David", "score": 91} ]

Q30.

🏆 종합 도전 문제 — 아래 조건에 맞는 간단한 성적 분석 프로그램을 작성하세요.

  • 딕셔너리 scores에는 5명의 학생과 점수가 저장되어 있습니다
  • 평균 점수를 계산해 평균: XX.X 형태로 출력하세요
  • 최고 점수를 받은 학생 이름을 최고: 이름 형태로 출력하세요
scores = {"민준": 88, "서연": 95, "지호": 72, "하은": 81, "도윤": 90}
출력:평균: 85.2 최고: 서연
HINT

평균: sum(scores.values()) / len(scores), 최고점자: max(scores, key=scores.get)
소수 1자리 출력: round(avg, 1) 또는 f"{avg:.1f}"

ANSWER
scores = {"민준": 88, "서연": 95, "지호": 72, "하은": 81, "도윤": 90} # 평균 계산: 전체 합 ÷ 인원 수 avg = sum(scores.values()) / len(scores) print(f"평균: {avg:.1f}") # 최고점자: 값 기준으로 최대 키 best = max(scores, key=scores.get) print(f"최고: {best}")

🎮 미니 프로젝트 ①

나만의 자기소개 카드 생성기

지금까지 배운 print, 변수, 연산자, 문자열, f-string을 모두 써서 터미널에 자기소개 카드를 출력하는 프로그램을 만듭니다. 처음으로 "내가 만든 프로그램"을 완성하는 순간입니다.

▸ 이 프로젝트를 마치면
  • 변수에 개인 정보를 저장하고 의미 있는 이름을 붙일 수 있다
  • f-string 서식 지정자로 숫자를 원하는 형태로 표시할 수 있다
  • 문자열 메서드와 연산자로 텍스트를 가공할 수 있다
  • print()로 정돈된 레이아웃의 출력을 직접 설계할 수 있다
  • 지금까지 배운 개념들이 어떻게 하나의 프로그램으로 합쳐지는지 느낄 수 있다

1.사용할 개념들

이 프로젝트는 지금까지 배운 것만 사용합니다. 새로운 문법은 하나도 없습니다.

print() 변수 / 데이터 타입 산술 연산자 문자열 * 반복 문자열 메서드 f-string 서식 지정자 :.1f :, :.1% int()
💡 미니 프로젝트 공략법

처음부터 정답을 보지 마세요. 기능 명세 읽기 → 혼자 시도 → 막히면 힌트 → 최후의 수단으로 정답 순서로 진행하면 실력이 훨씬 빠르게 늡니다. 막히는 것은 당연한 과정입니다.

3.기능 명세서

코딩을 시작하기 전에 이 표를 먼저 읽고 전체 구조를 머릿속에 그려보세요.

구분기능사용 개념
변수 설정이름, 나이, 직업, 관심사, 목표, 한 마디 등 변수 선언변수, str, int, float
상단/하단 테두리╔══╗ 형태 박스 출력문자열 * 반복, print()
기본 정보 출력이름·나이·직업·관심사·목표를 정렬된 형태로 출력f-string, 정렬 서식 :<N
구분선── 수평 구분선문자열 * 반복
통계 정보 출력공부 시작일, 목표 시간, 남은 챕터 수, 진행률 출력f-string, 산술 연산, :.1%
텍스트 진행 바[████░░░] 형태 진행 바 생성문자열 * 반복, int()
한 마디 출력나만의 한 마디를 따옴표 안에 출력f-string, 문자열

4.단계별 구현 가이드

한 번에 다 만들려 하지 말고, 각 단계를 완성할 때마다 실행해 결과를 확인하며 나아가세요.

01
변수 준비하기
파일 맨 위에 이름, 나이, 직업, 관심사, 목표, 공부 시작 날짜, 하루 목표 시간, 전체 챕터 수, 현재 챕터 번호, 한 마디를 변수로 선언하세요. 이 변수들만 바꾸면 카드 내용 전체가 자동으로 달라집니다.
02
상단 박스 테두리 출력하기
╔, ═, ╗ 문자를 조합해 박스 상단을 출력하세요. "═" * 34처럼 문자를 반복하면 깔끔하게 만들 수 있습니다.
03
기본 정보 출력하기
f-string으로 이름·나이·직업·관심사·목표를 출력하세요. 항목 이름에 :<7 서식을 쓰면 콜론이 세로로 일직선 정렬됩니다.
04
통계 정보 계산해서 출력하기
진행률(현재 챕터 / 전체 챕터)과 남은 일수(전체 챕터 - 현재 챕터)를 f-string 안에서 직접 계산해 출력하세요. 진행률에는 :.1% 서식을 쓰세요.
05
텍스트 진행 바 만들기
█와 ░를 반복해 [████░░░░░░░░] 형태의 진행 바를 만드세요. 전체 길이를 16칸으로 고정하고, 채워진 칸 수는 int(진행률 * 16)으로 계산합니다.
06
한 마디와 하단 테두리 출력하기
나만의 한 마디를 따옴표로 감싸 출력하고, 상단과 같은 방식으로 하단 박스를 닫아 마무리합니다.

5.막혔을 때 — 단계별 힌트

충분히 혼자 시도해본 뒤에 힌트를 열어보세요.

name     = "김파이썬"      # 이름 (문자열)
age      = 24             # 나이 (정수)
job      = "예비 개발자"
interest = "파이썬, 데이터, 게임 개발"
goal     = "브론즈 과정 완주!"

start_date   = "2026-03-04"
daily_hours  = 1    # 하루 목표 공부 시간
total_chaps  = 48   # 전체 챕터 수
current_chap = 12   # 현재 완료 챕터

quote = "코딩은 어렵지 않다, 익숙하지 않을 뿐!"
# "═" 문자를 34번 반복해 테두리를 만듭니다
border = "═" * 34

print(f"╔{border}╗")
print(f"║  ★  나의 파이썬 자기소개 카드  ★  ║")
print(f"╚{border}╝")
# {항목:<7} 처럼 쓰면 왼쪽 정렬 7칸 너비로 출력됩니다.
# 이렇게 하면 모든 줄의 : 위치가 맞아떨어집니다.

print(f"  {'이름':<7}: {name}")
print(f"  {'나이':<7}: {age}살")
print(f"  {'직업':<7}: {job}")
print(f"  {'관심사':<7}: {interest}")
print(f"  {'목표':<7}: {goal}")
# 진행률 = 현재 챕터 / 전체 챕터
progress  = current_chap / total_chaps
remaining = total_chaps - current_chap

print(f"  {'공부 시작일':<12}: {start_date}")
print(f"  {'목표 공부 시간':<12}: 하루 {daily_hours}시간")
print(f"  {'목표 달성까지':<12}: 약 {remaining}일 (챕터 수 기준)")
print(f"  {'오늘의 진행률':<12}: {progress:.1%}  [진행바 자리]")
# 전체 16칸, 채워진 칸 수는 진행률에 비례
bar_len = 16
filled  = int(progress * bar_len)   # 예: 0.25 * 16 = 4
bar     = "█" * filled + "░" * (bar_len - filled)

# 사용 예시
print(f"  {'오늘의 진행률':<12}: {progress:.1%}  [{bar}]")

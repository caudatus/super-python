input() — 사용자와 대화하기

지금까지는 코드를 실행하면 항상 같은 결과가 나왔습니다. 이제 사용자가 직접 값을 입력하면 그에 따라 다르게 반응하는 진짜 프로그램을 만들 수 있습니다.

LEARNING OBJECTIVES
  • input()으로 사용자 입력을 받는 방법을 이해한다
  • input()이 항상 문자열을 반환한다는 것을 안다
  • int(), float(), str()로 타입을 변환할 수 있다
  • 입력값을 변수에 저장하고 활용할 수 있다
  • prompt 메시지를 잘 쓰는 방법을 익힌다

1. input()이 뭔가요?

지금까지 만든 프로그램은 어떤 값을 쓸지 코드 안에 다 정해져 있었습니다. 예를 들어 name = "지수"처럼요. 이렇게 하면 지수가 아닌 다른 사람이 쓸 때도 항상 "지수"라고 나옵니다.

input()은 프로그램이 실행되는 도중에 사용자에게 "잠깐, 뭔가 입력해줘!"라고 요청하는 함수입니다. 마치 자판기 화면에 "음료를 선택하세요"라고 뜨는 것처럼, 파이썬이 실행을 멈추고 사용자가 뭔가를 타이핑할 때까지 기다립니다.

이름을 입력하세요: 김민준

기본 문법은 아주 간단합니다:

문법
변수 = input("화면에 보여줄 안내 메시지")

가장 기본적인 예제를 실행해보겠습니다:

예제 1 — 이름 입력받기
name = input("이름을 입력하세요: ")
print(f"안녕하세요, {name}님!")
이름을 입력하세요: 김민준
안녕하세요, 김민준님!

코드가 딱 두 줄인데, 이 프로그램은 누가 실행하느냐에 따라 다른 결과를 출력합니다. 이것이 input()의 핵심입니다.

💡 prompt 메시지 끝에 공백을 넣으세요

input("이름: ")처럼 메시지 끝에 띄어쓰기를 하나 넣으면, 사용자가 입력할 때 커서가 메시지 바로 옆에 붙지 않고 한 칸 떨어져서 더 읽기 편합니다. 작은 차이지만 UX가 훨씬 좋아집니다.

2. input()은 항상 문자열을 반환한다

여기서 가장 중요한 규칙 하나를 반드시 기억해야 합니다:

⚠️ 핵심 규칙

input()은 사용자가 무엇을 입력하든, 항상 문자열(str)로 반환합니다.
숫자 42를 입력해도 파이썬은 이것을 숫자가 아닌 문자열 "42"로 받아들입니다.

이게 왜 중요한지 실제 예제로 확인해봅시다:

예제 2 — 문자열로 받아오는 문제
age = input("나이를 입력하세요: ")
print(type(age))   # 타입 확인
print(age + 1)     # ← 이 줄에서 오류 발생!
이름을 입력하세요: 20
<class 'str'>
TypeError: can only concatenate str (not "int") to str

분명히 20을 입력했는데 str(문자열) 타입이라고 나옵니다. 그래서 숫자 1을 더하려고 하면 "문자열에는 문자열만 붙일 수 있어요"라는 오류가 발생합니다.

이 문제를 해결하는 방법이 타입 변환(Type Conversion)입니다.

3. 타입 변환 — int(), float(), str()

input()이 문자열로 준 값을 원하는 타입으로 바꿔주는 함수들이 있습니다:

함수변환 결과예시결과
int()정수 (소수점 없음)int("42")42
float()실수 (소수점 있음)float("3.14")3.14
str()다시 문자열로str(42)"42"

정수 입력받기

숫자를 입력받아 계산에 쓰려면 int()로 감싸주세요:

예제 3 — 나이 계산
age = int(input("나이를 입력하세요: "))   # 문자열 → 정수로 변환
next_year = age + 1
print(f"내년에는 {next_year}살이 됩니다.")
나이를 입력하세요: 25
내년에는 26살이 됩니다.

int(input(...)) — 이렇게 두 함수를 겹쳐 쓰는 것이 파이썬에서 가장 흔한 패턴입니다. 안쪽에서 바깥쪽 순서로 실행됩니다: 먼저 input()이 값을 받고, 그 결과를 int()가 변환합니다.

사용자 입력
"25"
→
input()
str 반환
→
int()
int 변환
→
변수에 저장
age = 25

실수 입력받기

소수점이 있는 숫자를 입력받을 때는 float()를 사용합니다:

예제 4 — 키와 BMI
height = float(input("키를 입력하세요 (cm): "))
weight = float(input("몸무게를 입력하세요 (kg): "))
h_m = height / 100              # cm → m
bmi = weight / (h_m * h_m)
print(f"BMI: {bmi:.1f}")
키를 입력하세요 (cm): 172
몸무게를 입력하세요 (kg): 68.5
BMI: 23.2
📌 int vs float — 언제 뭘 쓸까?

사람 수, 나이, 점수(정수)처럼 소수점이 절대 없는 값 → int()
키, 몸무게, 온도, 가격처럼 소수점이 있을 수 있는 값 → float()

4. 여러 개 입력받기

input()을 여러 번 써서 여러 값을 받을 수 있습니다. 각 input()은 Enter를 누를 때까지 대기합니다:

예제 5 — 여러 정보 입력받기
name = input("이름: ")
age  = int(input("나이: "))
city = input("사는 곳: ")

print(f"안녕하세요! {city}에 사는 {age}살 {name}입니다.")
이름: 박서연
나이: 22
사는 곳: 부산
안녕하세요! 부산에 사는 22살 박서연입니다.

input()과 리스트 함께 쓰기

입력받은 값들을 리스트에 담아두면 나중에 활용하기 편합니다:

예제 6 — 점수 3개 입력받아 평균 계산
scores = []                        # 빈 리스트 준비

s1 = int(input("국어 점수: "))
s2 = int(input("영어 점수: "))
s3 = int(input("수학 점수: "))

scores.append(s1)
scores.append(s2)
scores.append(s3)

avg = sum(scores) / len(scores)
print(f"평균: {avg:.1f}점  |  최고: {max(scores)}점  |  최저: {min(scores)}점")
국어 점수: 88
영어 점수: 92
수학 점수: 79
평균: 86.3점 | 최고: 92점 | 최저: 79점

5. 변환 실패 — 흔한 실수

숫자를 기대하고 int(input(...))를 썼는데, 사용자가 문자를 입력하면 어떻게 될까요?

예제 7 — 변환 실패 상황
age = int(input("나이를 입력하세요: "))   # 사용자가 "스물다섯" 입력
나이를 입력하세요: 스물다섯
ValueError: invalid literal for int() with base 10: '스물다섯'

ValueError가 발생합니다. "스물다섯"이라는 문자열을 정수로 바꿀 수 없기 때문입니다. 이런 상황을 안전하게 처리하는 방법은 나중에 예외처리(try/except) 챕터에서 배웁니다. 지금은 사용자가 올바른 값을 입력한다고 가정하고 진행합니다.

⚠️ 자주 하는 실수들

① int()로 소수점 문자열 변환 시도
int("3.14") → ValueError 발생. 소수점 문자열은 먼저 float()로, 그다음 필요하면 int()로 변환하세요.

② 변환 없이 숫자 계산 시도
input() 결과를 그냥 + 1하면 TypeError. 반드시 int()나 float()로 감싸야 합니다.

③ 공백 문자열 주의
아무것도 입력하지 않고 Enter를 누르면 ""(빈 문자열)이 옵니다. int("")는 오류입니다.

6. 한 줄에 여러 값 받기 — split()

간단한 프로그램에서는 값을 하나씩 받는 대신, 한 줄에 공백으로 구분해서 여러 값을 한꺼번에 받기도 합니다. 이때 문자열의 .split() 메서드를 씁니다:

예제 8 — split()으로 여러 값 한 번에 받기
# "10 20 30" 형태로 입력받기
line = input("세 수를 공백으로 구분해 입력하세요: ")
parts = line.split()         # ["10", "20", "30"] — 아직 문자열
a = int(parts[0])
b = int(parts[1])
c = int(parts[2])
print(f"합계: {a + b + c}")
세 수를 공백으로 구분해 입력하세요: 10 20 30
합계: 60

.split()은 공백(스페이스, 탭)을 기준으로 문자열을 잘라서 리스트로 만들어 줍니다. 아직 배우지 않은 내용이라 처음엔 낯설 수 있지만 패턴만 기억해 두면 됩니다.

💡 split()을 더 짧게 쓰는 방법

파이썬에는 위 코드를 한 줄로 줄이는 방법도 있습니다. 아직 배우지 않은 내용이라 지금 당장 외울 필요는 없지만, 나중에 보게 될 코드들입니다:

a, b, c = input("세 수: ").split() — 공백 구분 문자열을 세 변수에 한 번에 대입
a, b, c = map(int, input("세 수: ").split()) — 변환까지 한 번에

7. 직접 해보기 — 간단한 계산기

간단한 계산기 만들기

두 수를 입력받아 사칙연산 결과를 모두 출력하는 프로그램입니다:

따라하기 — 사칙연산 계산기
# 두 수 입력받기
a = float(input("첫 번째 수: "))
b = float(input("두 번째 수: "))

# 계산 결과 출력
print(f"{a} + {b} = {a + b}")
print(f"{a} - {b} = {a - b}")
print(f"{a} × {b} = {a * b}")
print(f"{a} ÷ {b} = {a / b:.2f}")
첫 번째 수: 12
두 번째 수: 4
12.0 + 4.0 = 16.0
12.0 - 4.0 = 8.0
12.0 × 4.0 = 48.0
12.0 ÷ 4.0 = 3.00

이렇게 바꿔보세요:

  • 정수로 깔끔하게 출력하려면 float 대신 int로 바꿔보세요
  • a % b로 나머지도 추가해보세요
  • prompt 메시지를 더 친절하게 바꿔보세요

8. 자주 발생하는 오류 패턴

❌ 틀린 코드변환 없이 덧셈
num = input("숫자: ")   # "10" 입력
print(num + 5)       # TypeError!
✅ 올바른 코드
num = int(input("숫자: "))  # int()로 변환
print(num + 5)            # 정상 동작
❌ 틀린 코드소수점 문자열을 int()로
h = int(input("키(cm): "))   # 172.5 입력 → ValueError!
✅ 올바른 코드
h = float(input("키(cm): "))  # float()로 소수점 허용

x = input("메시지: ")

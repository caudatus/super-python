1.프로젝트 소개

MINI PROJECT ②
나만의 성적표 생성기
이름과 과목별 점수를 직접 입력받아 합계·평균·최고점·최저점을 계산하고, 등급까지 판별하는 성적표를 출력합니다. input()과 리스트가 처음으로 만나 진짜로 "동작하는" 프로그램이 됩니다.
input() int() 리스트 append() sum / max / min f-string 문자열 서식

2.실행 결과 미리보기

여러분이 완성할 프로그램을 실행하면 아래와 같이 동작합니다. 밑줄 친 부분이 사용자가 직접 입력한 값입니다.

터미널 
=============================
     📋  성적표 생성기 v1.0

학생 이름을 입력하세요: 김민준

과학 점수 (0~100): 85

4.단계별 구현 가이드

한 번에 전체를 만들려 하지 말고, 한 단계씩 완성하고 실행해보면서 나아가세요. 각 단계가 제대로 출력되는 것을 확인한 뒤 다음 단계로 진행합니다.

STEP 1 헤더 출력과 이름·과목 수 입력받기 print · input · int

프로그램이 시작될 때 제목 배너를 출력하고, 학생 이름과 과목 수를 입력받습니다.

과목 수는 int(input(...))로 변환하고 n 변수에 저장하세요.

STEP 1 구현
print("=============================")
print("     📋  성적표 생성기 v1.0")
print("=============================")
name = input("학생 이름을 입력하세요: ")
n    = int(input("과목 수를 입력하세요 (3~5): "))
print()  # 빈 줄

▶ 실행하면 배너와 입력 프롬프트가 나타나야 합니다.

STEP 2 과목 이름과 점수를 리스트에 수집하기 리스트 · append · 반복 입력

빈 리스트 subjects와 scores를 만들고, 과목 수(n)만큼 입력을 반복해서 받습니다.

지금은 for문을 배우지 않았으니, n을 직접 사용해 필요한 횟수만큼 입력 코드를 작성하세요. (최소 3개)

STEP 2 구현 — 과목 수를 3으로 고정해서 먼저 시도해보세요
subjects = []   # 과목 이름 리스트
scores   = []   # 점수 리스트

# 과목 1
subj = input("과목 1 이름: ")
score = int(input(f"{subj} 점수 (0~100): "))
subjects.append(subj)
scores.append(score)

# 과목 2, 3도 같은 패턴으로 반복
# ...

각 과목을 입력받은 뒤 print(subjects, scores)로 리스트가 올바르게 채워지는지 확인하세요.

STEP 3 합계·평균·최고·최저 계산하기 sum · max · min · 인덱싱

리스트가 완성되면 통계를 계산합니다. 최고점과 최저점의 과목 이름을 찾으려면 인덱스가 필요합니다.

STEP 3 구현
total     = sum(scores)
avg       = total / len(scores)
best      = max(scores)
worst     = min(scores)

# 최고점 과목명: 점수 리스트에서 최고점의 위치를 찾아 subjects에서 꺼내기
best_subj  = subjects[scores.index(best)]
worst_subj = subjects[scores.index(worst)]

.index(값)는 리스트에서 해당 값의 위치(인덱스)를 반환합니다. scores.index(max(scores))는 "점수 리스트에서 최고점이 몇 번째 위치?"를 알려줍니다.

STEP 4 등급 판별하기 평균 기반 조건 분기

평균 점수를 기준으로 등급 문자(A/B/C/D/F)를 결정합니다. 조건문(if/elif/else)은 다음 챕터에서 배우지만, 이번 프로젝트를 위해 간단히 사용법만 미리 씁니다.

STEP 4 구현 — 조건문 미리 사용해보기
if avg >= 90:
    grade = "A"
elif avg >= 80:
    grade = "B"
elif avg >= 70:
    grade = "C"
elif avg >= 60:
    grade = "D"
else:
    grade = "F"

지금은 이 패턴을 그대로 사용하면 됩니다. 어떻게 동작하는지는 18번 챕터에서 자세히 배웁니다.

STEP 5 성적표 출력하기 f-string · 문자열 서식 정렬

계산된 모든 값을 정리해서 예쁜 성적표로 출력합니다. 과목 이름과 점수를 f-string의 정렬 서식으로 가지런히 맞춰보세요.

STEP 5 구현 — 출력 형식
print()
print("=============================")
print(f"   📊  {name} 님의 성적표")
print("=============================")

# 각 과목 출력 — 과목명을 8칸 너비로 왼쪽 정렬
print(f" {subjects[0]:{8}}: {scores[0]:>4}점")
# 나머지 과목도 같은 방식으로 ...

# 구분선 + 통계
print("-----------------------------")
print(f" {'합계':{8}}: {total:>4}점")
print(f" {'평균':{8}}: {avg:>6.1f}점")
print(f" {'최고점':{8}}: {best:>4}점  ({best_subj})")
print(f" {'최저점':{8}}: {worst:>4}점  ({worst_subj})")
print(f" {'등급':{8}}: {grade}")
print("=============================")

{과목명:<8}은 과목명을 8칸 너비에 왼쪽 정렬, {점수:>4}는 4칸 너비에 오른쪽 정렬입니다. 이렇게 하면 어떤 이름이 들어와도 세로가 일렬로 맞춰집니다.

5.막혔을 때 — 단계별 힌트

충분히 혼자 시도해본 뒤에 힌트를 열어보세요.

💡 힌트 1 — 빈 리스트에 값을 추가하는 방법 ▼

.append(값)를 쓰면 리스트 끝에 값이 추가됩니다.

names = []
names.append("국어")
names.append("영어")
print(names)  # ['국어', '영어']

과목 이름과 점수를 각각 별도 리스트에 append()로 추가하세요.

💡 힌트 2 — f-string 안에서 input() 사용하기 ▼

과목명을 먼저 입력받은 뒤, 그 이름을 prompt 메시지에 포함시키면 더 친절한 UX가 됩니다.

subj = input("과목 이름: ")
score = int(input(f"{subj} 점수: "))
# → "국어 점수: " 처럼 과목명이 들어간 프롬프트가 나옵니다
💡 힌트 3 — 최고점·최저점의 과목명 찾기 ▼

두 리스트(subjects, scores)는 같은 인덱스끼리 짝을 이룹니다. 점수 리스트에서 최고점의 위치를 찾으면 같은 위치의 과목 이름을 꺼낼 수 있습니다.

best       = max(scores)
best_idx   = scores.index(best)  # 최고점의 위치
best_subj  = subjects[best_idx]   # 같은 위치의 과목명
💡 힌트 4 — 과목명과 점수를 가지런히 정렬 출력하기 ▼

f-string의 {변수:<너비}로 왼쪽 정렬, {변수:>너비}로 오른쪽 정렬이 가능합니다.

subj  = "국어"
score = 88
print(f" {subj:<8}: {score:>4}점")
# " 국어      :   88점"  (과목명 8칸, 점수 4칸)

한글 과목명은 2바이트라 완벽한 정렬이 어렵습니다. 지금은 근사치로 맞추면 충분합니다.

💡 힌트 5 — 과목 수에 따라 출력 개수를 유동적으로 만들기 ▼

과목 수가 3~5개로 바뀌면 출력 줄 수도 달라야 합니다. 인덱스를 직접 쓰는 대신, 리스트를 순회하는 방식을 사용할 수 있습니다. 아직 for를 배우지 않았으니 다음과 같은 패턴으로 모든 경우를 출력해보세요.

# 방법 1: 인덱싱으로 각각 출력 (과목 수 고정 시)
print(f" {subjects[0]:<8}: {scores[0]:>4}점")
print(f" {subjects[1]:<8}: {scores[1]:>4}점")
# ... n개만큼 반복

# 참고: for 루프를 배운 뒤에는 이렇게 할 수 있습니다 (선택 도전)
for i in range(n):
    print(f" {subjects[i]:<8}: {scores[i]:>4}점")

6.전체 정답 코드

17-grade-report.py정답
# ──────────────────────────────────────
# 🎮 미니 프로젝트 ② — 나만의 성적표 생성기
# 사용 개념: input, int, 리스트, append,
#           sum/max/min, f-string, 조건문
# ──────────────────────────────────────

# 1. 시작 배너
print("=============================")
print("     📋  성적표 생성기 v1.0")
print("=============================")

# 2. 기본 정보 입력
name = input("학생 이름을 입력하세요: ")
n    = int(input("과목 수를 입력하세요 (3~5): "))
print()

# 3. 과목 이름·점수 수집 (n개)
subjects = []
scores   = []

for i in range(1, n + 1):
    subj  = input(f"과목 {i} 이름: ")
    score = int(input(f"{subj} 점수 (0~100): "))
    subjects.append(subj)
    scores.append(score)

# 4. 통계 계산
total     = sum(scores)
avg       = total / len(scores)
best      = max(scores)
worst     = min(scores)
best_subj  = subjects[scores.index(best)]
worst_subj = subjects[scores.index(worst)]

# 5. 등급 판별
if avg >= 90:
    grade = "A"
elif avg >= 80:
    grade = "B"
elif avg >= 70:
    grade = "C"
elif avg >= 60:
    grade = "D"
else:
    grade = "F"

# 6. 성적표 출력
print()
print("=============================")
print(f"   📊  {name} 님의 성적표")
print("=============================")

for i in range(n):
    print(f" {subjects[i]:<8}: {scores[i]:>4}점")

print("-----------------------------")
print(f" {'합계':<8}: {total:>4}점")
print(f" {'평균':<8}: {avg:>6.1f}점")
print(f" {'최고점':<8}: {best:>4}점  ({best_subj})")
print(f" {'최저점':<8}: {worst:>4}점  ({worst_subj})")
print(f" {'등급':<8}: {grade}")
print("=============================")

📌 정답 코드에서 for 루프를 사용한 이유: 과목 수 n이 3~5개로 가변적이기 때문입니다. 아직 for 루프를 배우지 않았다면 subjects[0], subjects[1]... 방식으로 출력해도 충분합니다. for 루프는 21번 챕터에서 배웁니다.

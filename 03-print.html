print()란?

파이썬과 처음으로 대화하는 방법 — 화면에 무언가를 출력하기

▸ 이 챕터를 마치면
  • print()로 텍스트, 숫자, 계산 결과를 화면에 출력할 수 있다
  • 여러 값을 한 번에 출력하고, sep으로 구분자를 바꿀 수 있다
  • end로 줄바꿈을 제어할 수 있다
  • print()에서 자주 발생하는 오류를 읽고 고칠 수 있다

1.print()가 뭐죠?

print()는 화면에 무언가를 출력하는 함수입니다. 파이썬에서 가장 먼저, 가장 자주 쓰는 도구입니다.

비유하자면 파이썬에게 "이걸 나한테 보여줘!"라고 말하는 방법입니다. 파이썬이 계산하거나 처리한 결과를 눈으로 확인하려면 반드시 print()가 필요합니다.

💡 함수란?

함수는 특정 작업을 수행하는 코드 묶음에 이름을 붙인 것입니다. 지금은 "괄호 안에 넣으면 출력된다"는 것만 기억하세요. 함수는 챕터 29에서 깊이 배웁니다.

2.print()의 구조

print()는 이렇게 생겼습니다. 각 부분이 무슨 역할을 하는지 살펴봅시다.

print("Hello, Python!")
함수 이름
print
괄호
( )
인자 (출력할 것)
"Hello, Python!"

괄호 안에 출력하고 싶은 것을 넣습니다. 텍스트(문자열)는 반드시 따옴표로 감싸야 합니다. 큰따옴표(")와 작은따옴표(') 모두 사용할 수 있습니다.

3.기본 사용법

텍스트 출력

python
1234
print("Hello, Python!")
print("안녕하세요")
print('작은따옴표도 됩니다')
print()                 # 아무것도 없으면 빈 줄 출력
실행 결과
Hello, Python!
안녕하세요
작은따옴표도 됩니다

숫자와 계산식 출력

숫자는 따옴표 없이 그대로 씁니다. 계산식을 넣으면 결과를 출력합니다.

python
12345
print(42)
print(3.14)
print(100 + 200)    # 더하기
print(10 * 5)       # 곱하기
print(7 / 2)        # 나누기
실행 결과
42
3.14
300
50
3.5

4.여러 값 한 번에 출력하기

쉼표(,)로 여러 값을 나열하면 한 줄에 모두 출력됩니다. 기본적으로 값 사이에 공백 한 칸이 자동으로 들어갑니다.

python
123
print("이름:", "파이썬")
print("1 더하기 2는", 1 + 2, "입니다")
print("A", "B", "C", "D")
실행 결과
이름: 파이썬
1 더하기 2는 3 입니다
A B C D

5.sep — 구분자 바꾸기

값 사이의 공백을 다른 문자로 바꾸고 싶다면 sep 옵션을 씁니다. sep은 separator(구분자)의 약자입니다.

python
123456
# 날짜를 하이픈으로 구분
print("2025", "03", "03", sep="-")

# 전화번호를 하이픈으로 구분
print("010", "1234", "5678", sep="-")

# 구분자를 없애면 붙여서 출력
print("슈", "퍼", "파", "이", "썬", sep="")
실행 결과
2025-03-03
010-1234-5678
슈퍼파이썬

6.end — 줄바꿈 제어하기

print()는 출력 후 자동으로 줄바꿈을 합니다. 내부적으로 end="\n"이 기본값으로 설정되어 있기 때문입니다. \n은 줄바꿈을 의미하는 특수 문자입니다. end 옵션을 바꾸면 줄바꿈 대신 원하는 문자를 넣을 수 있습니다.

python
123456789
# 기본: 자동 줄바꿈
print("첫 번째")
print("두 번째")

# end="" — 줄바꿈 없이 이어서 출력
print("Hello", end="")
print(", World!")

# end=" " — 줄바꿈 대신 공백 한 칸
print(1, end=" ")
print(2, end=" ")
print(3)
실행 결과
첫 번째
두 번째
Hello, World!
1 2 3

7.sep과 end 함께 쓰기

sep과 end는 함께 쓸 수 있습니다. 파라미터 표로 한눈에 정리해봅시다.

파라미터 기본값 역할 예시
sep " " (공백) 값과 값 사이에 들어가는 문자 sep="-" → 하이픈으로 구분
end "\n" (줄바꿈) 출력이 끝난 후 붙는 문자 end=" " → 줄바꿈 대신 공백
python
123
print("A", "B", "C", sep="-", end="!\n")
print("10", "20", "30", sep=", ", end=" → 합계: ")
print(10 + 20 + 30)
실행 결과
A-B-C!
10, 20, 30 → 합계: 60

8.자주 하는 실수와 오류

print()에서 가장 흔히 발생하는 오류 3가지를 미리 알아두면 당황하지 않습니다.

① 따옴표를 빠뜨린 경우

python — 잘못된 코드
1
print(Hello)   # 오류! Hello를 변수로 인식
오류 메시지
NameError: name 'Hello' is not defined
✅ 수정 방법

텍스트는 반드시 따옴표로 감싸세요. print("Hello")

② 따옴표 종류가 다른 경우

python — 잘못된 코드
1
print("안녕하세요')  # 오류! 큰따옴표로 시작해서 작은따옴표로 끝남
오류 메시지
SyntaxError: EOL while scanning string literal
✅ 수정 방법

시작과 끝의 따옴표를 같은 종류로 맞추세요. print("안녕하세요") 또는 print('안녕하세요')

③ 괄호를 빠뜨린 경우

python — 잘못된 코드
1
print "Hello"   # 오류! 파이썬 2 문법, 파이썬 3에서는 괄호 필수
오류 메시지
SyntaxError: Missing parentheses in call to 'print'
✅ 수정 방법

파이썬 3에서는 반드시 괄호가 필요합니다. print("Hello")

▸ 핵심 요약
기본 출력 print("텍스트") — 따옴표 안의 내용을 화면에 출력 숫자 출력 print(42) / print(1 + 2) — 숫자·계산식은 따옴표 불필요 여러 값 print(a, b, c) — 쉼표로 구분, 사이에 공백 자동 삽입 sep sep="-" — 값 사이 구분자 지정 (기본값: 공백 " ") end end="" — 출력 후 붙는 문자 지정 (기본값: 줄바꿈 "\n") 빈 줄 print() — 아무것도 넣지 않으면 빈 줄 하나 출력

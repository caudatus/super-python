03기능 명세서

프로그램을 만들기 전에 어떤 함수가 필요한지 먼저 설계합니다. 함수 하나가 기능 하나를 담당하는 원칙을 지킵니다.

함수 이름기능사용 개념
get_input() 과목 수와 이름·점수를 입력받아 리스트 두 개로 반환 input(), for, int()
get_total(scores) 점수 리스트를 받아 합계를 반환 for문, +=
get_average(scores) 점수 리스트를 받아 평균을 반환 get_total(), len()
get_max_score(scores) 점수 리스트를 받아 최고점을 반환 for문, if 비교
get_min_score(scores) 점수 리스트를 받아 최저점을 반환 for문, if 비교
get_grade(average) 평균 점수를 받아 학점 문자를 반환 if / elif / else
print_report(...) 과목명·점수·통계를 보기 좋게 출력 for문, f-string, 최고/최저점 표시
💡 함수를 먼저 설계하는 습관

코드를 바로 작성하는 것보다 "어떤 함수가 필요할까?"를 먼저 생각하면 실수가 줄어듭니다. 실제 개발자들이 코딩 전에 설계를 먼저 하는 이유입니다. 이번 프로젝트에서 그 습관을 연습합니다.

04단계별 구현 가이드

STEP 1 입력 받기 — get_input() input · for · int()

먼저 과목 수를 입력받고, 그 수만큼 반복하며 과목 이름과 점수를 리스트에 저장합니다. 두 리스트를 함께 반환합니다.

python
def get_input():
    """과목 수, 이름, 점수를 입력받아 두 리스트를 반환합니다."""
    n = int(input("과목 수를 입력하세요: "))
    subjects = []   # 과목명 리스트
    scores   = []   # 점수 리스트

    for i in range(n):
        name  = input(f"과목 {i+1} 이름: ")
        score = int(input(f"{name} 점수: "))
        subjects.append(name)
        scores.append(score)

    return subjects, scores   # 두 리스트를 함께 반환
💡 두 값을 동시에 return

return subjects, scores처럼 쉼표로 구분하면 두 값을 한 번에 반환할 수 있습니다. 호출할 때는 subjects, scores = get_input()처럼 받습니다. 이를 다중 반환(multiple return)이라 합니다.

STEP 2 통계 함수 4개 작성 for · if · return

합계, 평균, 최고점, 최저점을 각각 별도 함수로 만듭니다. 각 함수는 점수 리스트를 받아 숫자를 하나 반환합니다.

python
def get_total(scores):
    """합계 반환"""
    total = 0
    for s in scores:
        total += s
    return total

def get_average(scores):
    """평균 반환 — get_total()을 재활용합니다"""
    return get_total(scores) / len(scores)

def get_max_score(scores):
    """최고점 반환"""
    highest = scores[0]   # 첫 번째 값을 기준으로 시작
    for s in scores:
        if s > highest:
            highest = s
    return highest

def get_min_score(scores):
    """최저점 반환"""
    lowest = scores[0]
    for s in scores:
        if s < lowest:
            lowest = s
    return lowest
💡 scores[0]부터 시작하는 이유

최고점을 찾을 때 처음엔 "기준값"이 필요합니다. 0이나 -999처럼 임의의 값보다, 리스트 첫 번째 요소(scores[0])를 기준으로 시작하면 어떤 범위의 점수에도 올바르게 동작합니다.

STEP 3 학점 계산 — get_grade(average) if · elif · else · return

평균 점수를 받아 학점 문자열을 반환합니다. return을 만나면 즉시 함수가 종료되므로 elif를 깔끔하게 사용할 수 있습니다.

python
def get_grade(average):
    """평균을 받아 학점(A~F)을 반환합니다."""
    if average >= 90:
        return "A"
    elif average >= 80:
        return "B"
    elif average >= 70:
        return "C"
    elif average >= 60:
        return "D"
    else:
        return "F"
STEP 4 보고서 출력 — print_report() for · f-string · 최고/최저 표시

모든 통계를 받아 보기 좋게 출력합니다. 각 과목 뒤에 최고점·최저점 표시를 붙이는 것이 포인트입니다.

python
def print_report(subjects, scores, total, average, grade):
    """성적 보고서를 출력합니다."""
    highest = get_max_score(scores)
    lowest  = get_min_score(scores)

    print("\n━━━━━━━━━━━━━━━━━━━━━━━━━━━━")
    print("        📊 성적 분석 결과")
    print("━━━━━━━━━━━━━━━━━━━━━━━━━━━━")

    for i in range(len(subjects)):
        tag = ""
        if scores[i] == highest:
            tag = "  ← 최고점 🏆"
        elif scores[i] == lowest:
            tag = "  ← 최저점 📉"
        print(f"  {subjects[i]:8s}: {scores[i]:3d}점{tag}")

    print("----------------------------")
    print(f"  합 계 : {total}점")
    print(f"  평 균 : {average:.1f}점")
    print(f"  학 점 : {grade}")
    print("━━━━━━━━━━━━━━━━━━━━━━━━━━━━")
💡 포맷 지정자 — 정렬과 소수점

f"{subjects[i]:8s}"는 문자열을 8칸에 왼쪽 정렬합니다. f"{scores[i]:3d}"는 정수를 3칸에 오른쪽 정렬합니다. f"{average:.1f}"는 소수점 1자리까지 출력합니다. 이렇게 하면 숫자가 줄줄이 정렬됩니다.

STEP 5 메인 코드 — 함수 조합하기 함수 호출 · 다중 반환

지금까지 만든 함수들을 올바른 순서로 호출합니다. 이 부분이 프로그램의 "뼈대"입니다.

python — 메인 코드
# 1. 입력 받기
subjects, scores = get_input()

# 2. 통계 계산
total   = get_total(scores)
average = get_average(scores)
grade   = get_grade(average)

# 3. 결과 출력
print_report(subjects, scores, total, average, grade)

메인 코드는 단 세 단계로 읽힙니다. 입력 → 계산 → 출력. 각 단계가 함수 호출 한 줄로 표현되어 있어 코드 전체 흐름이 한눈에 들어옵니다. 이것이 함수를 쓰는 진짜 이유입니다.

05막혔을 때 — 단계별 힌트

힌트 1 — 두 리스트를 어떻게 동시에 return 하나요? ▼

return a, b처럼 쉼표로 구분하면 됩니다. 파이썬은 이를 튜플(tuple)로 묶어서 반환합니다.

python
def get_two():
    return ["국어", "수학"], [88, 95]

names, scores = get_two()   # 각각 받기
힌트 2 — 최고점을 어떻게 찾나요? ▼

리스트의 첫 번째 값을 "현재까지 최고"라고 가정하고 시작합니다. 반복하면서 더 큰 값이 나오면 갱신합니다.

python
scores = [88, 72, 95, 81]
highest = scores[0]   # 88 — 첫 번째 값으로 시작
for s in scores:
    if s > highest:
        highest = s   # 72는 통과, 95에서 갱신, 81은 통과
print(highest)   # 95
힌트 3 — f-string에서 소수점을 어떻게 맞추나요? ▼

{변수:.Nf} 형식으로 소수점 자릿수를 지정합니다.

python
avg = 85.2
print(f"평균: {avg:.1f}")   # 평균: 85.2
print(f"평균: {avg:.2f}")   # 평균: 85.20
print(f"평균: {avg:.0f}")   # 평균: 85 (반올림)
힌트 4 — 전체 코드 구조 (의사 코드) ▼

코드를 작성하기 전에 구조를 글로 정리하면 막히는 곳이 줄어듭니다.

pseudocode

06전체 정답 코드

먼저 스스로 도전해보세요. 막히면 위의 힌트를 활용하고, 완성했다면 아래 정답과 비교해봅니다.

def get_input():
    """과목 수, 이름, 점수를 입력받아 반환합니다."""
    n = int(input("과목 수를 입력하세요: "))
    subjects = []
    scores   = []
    for i in range(n):
        name  = input(f"과목 {i+1} 이름: ")
        score = int(input(f"{name} 점수: "))
        subjects.append(name)
        scores.append(score)
    return subjects, scores


def get_total(scores):
    """합계를 반환합니다."""
    total = 0
    for s in scores:
        total += s
    return total


def get_average(scores):
    """평균을 반환합니다."""
    return get_total(scores) / len(scores)


def get_max_score(scores):
    """최고점을 반환합니다."""
    highest = scores[0]
    for s in scores:
        if s > highest:
            highest = s
    return highest


def get_min_score(scores):
    """최저점을 반환합니다."""
    lowest = scores[0]
    for s in scores:
        if s < lowest:
            lowest = s
    return lowest


def get_grade(average):
    """평균으로 학점을 반환합니다."""
    if average >= 90:
        return "A"
    elif average >= 80:
        return "B"
    elif average >= 70:
        return "C"
    elif average >= 60:
        return "D"
    else:
        return "F"


def print_report(subjects, scores, total, average, grade):

    highest = get_max_score(scores)
    lowest  = get_min_score(scores)

    print("        📊 성적 분석 결과")
    print("━━━━━━━━━━━━━━━━━━━━━━━━━━━━")

    for i in range(len(subjects)):
        tag = ""
        if scores[i] == highest:
            tag = "  ← 최고점 🏆"
        elif scores[i] == lowest:
            tag = "  ← 최저점 📉"
        print(f"  {subjects[i]:8s}: {scores[i]:3d}점{tag}")

    print("----------------------------")
    print(f"  합 계 : {total}점")
    print(f"  평 균 : {average:.1f}점")
    print(f"  학 점 : {grade}")
    print("━━━━━━━━━━━━━━━━━━━━━━━━━━━━")

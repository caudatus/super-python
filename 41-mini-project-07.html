🎮 미니 프로젝트 ⑧
도서관 대출 관리 시스템

딕셔너리·리스트·셋·튜플 — 네 가지 자료구조를 모두 투입해 실제로 동작하는 도서관 시스템을 만들어봅니다. 어떤 자료구조가 어떤 역할에 가장 잘 맞는지 직접 느껴보세요.

► 이 프로젝트를 마치면
  • 딕셔너리로 도서 정보(ISBN → 제목·저자·재고)를 구조화할 수 있다
  • 셋으로 대출 중인 도서를 중복 없이 추적할 수 있다
  • 리스트로 대출 이력을 순서대로 기록할 수 있다
  • 튜플로 변경되면 안 되는 반납 기한 정보를 안전하게 보관할 수 있다
  • 네 가지 자료구조가 협력하는 실전 패턴을 이해할 수 있다

1.사용할 개념들

이 프로젝트는 지금까지 배운 것만 사용합니다. 자료구조 4총사가 각자 맡은 역할을 보세요.

딕셔너리 — 도서 정보 저장 셋 — 대출 중인 도서 추적 리스트 — 대출 이력 기록 튜플 — 반납 기한 (불변 데이터) for / if f-string 함수
💡 미니 프로젝트 공략법

정답을 바로 보지 마세요. 기능 명세 읽기 → 혼자 시도 → 막히면 힌트 → 최후의 수단으로 정답 코드 순서로 진행하면 실력이 가장 빠르게 늡니다.

3.기능 명세서

📋 데이터 구조 설계

① books (딕셔너리) — {"ISBN": {"title": ..., "author": ..., "stock": ..., "genre": ...}}
② borrowed_set (셋) — 현재 대출 중인 ISBN 목록 (중복 불가)
③ history (리스트) — (ISBN, "대출"/"반납", 날짜) 튜플을 순서대로 기록
④ 반납 기한 (튜플) — (연, 월, 일) 형태로 반환, 변경 불가

📋 구현할 함수 목록

① show_books(books) — 전체 도서 목록 출력
② get_due_date() — 오늘부터 14일 후 반납 기한을 튜플로 반환
③ borrow_book(books, borrowed_set, history, isbn) — 대출 처리
④ return_book(books, borrowed_set, history, isbn) — 반납 처리
⑤ show_status(borrowed_set, history) — 현재 대출 현황 출력
⑥ books_by_genre(books) — 장르별로 묶어 출력

STEP 3 대출 처리 함수 셋 리스트 ►

borrow_book(books, borrowed_set, history, isbn) 함수를 만드세요.

조건: ① ISBN이 books에 없으면 "없는 도서" 메시지 ② 이미 borrowed_set에 있으면 "이미 대출 중" 메시지 ③ 재고(stock)가 0이면 "재고 없음" 메시지 ④ 모두 통과하면 — borrowed_set에 add(), stock 1 감소, history에 (isbn, "대출", 반납기한) 튜플 추가

def borrow_book(books, borrowed_set, history, isbn):
    if isbn not in books:
        print("  없는 도서입니다")
        return
    if isbn in borrowed_set:          # 셋으로 O(1) 검색
        print("  이미 대출 중")
        return
    if books[isbn]["stock"] == 0:
        print("  재고 없음")
        return
    # 대출 처리
    borrowed_set.add(isbn)            # 셋에 추가
    books[isbn]["stock"] -= 1
    year, month, day = get_due_date() # 튜플 언패킹
    due = f"{year}-{month:02d}-{day:02d}"
    history.append((isbn, "대출", due))  # 리스트에 튜플 추가
def borrow_book(books, borrowed_set, history, isbn):
    if isbn not in books:
        print("  ❌ 없는 도서입니다")
        return
    title = books[isbn]["title"]
    if isbn in borrowed_set:          # 셋 — O(1) 검색
        print(f"  ⚠️  '{title}' 는 이미 대출 중입니다")
        return
    if books[isbn]["stock"] == 0:
        print(f"  ❌ '{title}' 재고가 없습니다")
        return
    borrowed_set.add(isbn)             # 셋에 추가
    books[isbn]["stock"] -= 1
    year, month, day = get_due_date()  # 튜플 언패킹
    due = f"{year}-{month:02d}-{day:02d}"
    history.append((isbn, "대출", due))  # 리스트에 튜플 추가
    print(f"  ✅ '{title}' 대출 완료 — 반납 기한: {due}")
STEP 4 반납 처리 함수 셋 리스트 ►

return_book(books, borrowed_set, history, isbn) 함수를 만드세요.

조건: ① borrowed_set에 없으면 "대출 기록 없음" 메시지 ② 있으면 — borrowed_set에서 discard()로 제거, stock 1 증가, history에 (isbn, "반납", "—") 추가

def return_book(books, borrowed_set, history, isbn):
    if isbn not in borrowed_set:
        print("  대출 기록이 없습니다")
        return
    borrowed_set.discard(isbn)   # 셋에서 안전하게 삭제
    books[isbn]["stock"] += 1
    history.append((isbn, "반납", "—"))
    print(f"  '{books[isbn]['title']}' 반납 완료")

STEP 5 현황 출력 + 장르별 분류 딕셔너리 셋 ►

show_status(borrowed_set, history) — 현재 대출 중 권수와 전체 이력 건수를 출력하세요.

books_by_genre(books) — 장르를 키로, 도서 제목 리스트를 값으로 하는 딕셔너리를 동적으로 만들어 출력하세요. 빈 딕셔너리에서 시작해 순회하면서 채워가는 패턴이 핵심입니다.

def books_by_genre(books):
    genre_dict = {}            # 빈 딕셔너리 시작
    for isbn, info in books.items():
        genre = info["genre"]
        if genre not in genre_dict:
            genre_dict[genre] = []     # 처음 보는 장르면 빈 리스트 생성
        genre_dict[genre].append(info["title"])
    for genre, titles in genre_dict.items():
        print(f"  {genre}: {', '.join(titles)}")
def show_status(borrowed_set, history):
    print("\n[현재 상태]")
    print(f"  대출 중인 도서 수: {len(borrowed_set)}권")
    print(f"  전체 대출 이력:    {len(history)}건")
